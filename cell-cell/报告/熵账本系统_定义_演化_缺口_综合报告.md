## 一、熵账本系统定义

### 1.1 核心概念

熵账本是 nexus_v1 的**被动审计与观测系统**，定义来源：

| 来源 | 定义 | 日期 |
|------|------|------|
| [RULES.md](nexus_v1/RULES.md) 原则 2 | 每次参数修改后检查信号深度+能量守恒+权重演化+无爆炸/崩塌 | 05-22 |
| [LOCAL_entropy.md](cell-cell/docs/atlas/LOCAL_entropy.md) | 三层审计架构 | 06-08 |
| [GLOBAL_ARCH.md](cell-cell/docs/atlas/GLOBAL_ARCH.md) §5 | 验证系统：NoetherProbe + EntropyLedger + RegressionTest | 06-08 |
| [A7 审计](cell-cell/archive/undated/analysis_A7_entropy_audit.A7实施方案审计-熵账本系统架构决策.md) | 三个独立熵系统分散在两个文件 | undated |
| [盲区审计](cell-cell/archive/undated/analysis_differentiation_entropy_blindspot.母本分化-熵账本-盲区审计.md) | 纯被动观测器，体感层不可见 | undated |

### 1.2 三层审计架构

```
Layer 1: NoetherProbe (每100步)
  ├── 能量守恒: E_in = E_out + E_stored + E_leak
  ├── 电荷 KCL: ΣI_in = ΣI_out (per Capacitor)
  ├── Landauer: Q/bit ≥ kT ln2
  └── 权重稳定: |mean(dw)| < 0.01/step

Layer 2: EntropyLedger (每1000步)
  ├── 能量收支追踪 + 热耗散
  ├── 层活动 / 层能量 / 层热耗散
  ├── ISI 熵 (每个发放神经元的脉冲时间编码)
  ├── 层间传递熵 (上游→下游 Pearson 相关系数)
  └── 能效 (spk/heat)

Layer 3: TOPRXin Ledger (每1000步)
  ├── T = 拓扑快照
  ├── O = 振荡器状态
  ├── P = 功率分布
  ├── R = 运动势 (ν, P_ν)
  └── Xin = 预测误差分布
```

### 1.3 熵的四种度量

| 度量 | 符号 | 定义 | 代码 |
|------|------|------|------|
| **结构熵** | H_struct | 权重 Shannon 熵 (50-bin)，突触分布多样性 | [`weight_entropy.py`](nexus_v1/ledger/weight_entropy.py) |
| **流程熵** | H_flow | 跨 Bundle Xin 分布熵，信号路由多样性 | [`noether_probe.py`](nexus_v1/ledger/noether_probe.py) L338 |
| **ISI 熵** | — | 脉冲间期 Shannon 熵，时间编码信息量 | [`energy_ledger.py`](nexus_v1/ledger/energy_ledger.py) L188 |
| **层间传递** | — | 上游→下游层激活 Pearson 相关系数 | [`energy_ledger.py`](nexus_v1/ledger/energy_ledger.py) L226 |

  ┌──────────────────────────────────────────────────────────────────┐

### 5.4 缺失的"诊断→处方"转换层

[盲区审计](cell-cell/archive/undated/analysis_differentiation_entropy_blindspot.母本分化-熵账本-盲区审计.md) 指出：当前所有熵组件都是**纯被动观测器**（READ-ONLY observer）。

```
当前能做到的:
  ✅ 报告 L6_Mot avg_activity=0.0008
  ✅ 报告 Col 全部 activation=0
  ✅ 报告 K_ema=47327 发散

当前做不到的:
  ❌ "Col 层 5000 步持续零活动 → 疑似被侧向抑制压死 → 建议降低 LateralInhibition.strength"
  ❌ "K_ema 在 15k-25k 步增长 9× → 影子层自由能发散 → 建议启动影子 Col 分化"
  ❌ "DA 热耗散占总耗散 89% → DA 可能遮蔽其他层的学习信号 → 建议降低 DA gm"
  ❌ "权重熵 delta=0 持续 50000 步 → STDP 未触发 → 检查 pre/post trace 是否衰减过快"
```

**`GuidedConstructionAuditor` 类（structural.py L626）的存在表明这个需求已被预见，但它从未被实例化。**
