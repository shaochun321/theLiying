## 1. 项目总览

**Cell-CC** 是一个以**半导体物理组件（Capacitor/MOSFET/Memristor/PowerRail）为计算基元的神经环路仿真器**，模拟从机械力感知到运动输出的完整信号处理链路。

### 核心四原语

| 组件 | 物理类比 | 生物对应 | 功能 |
|------|----------|----------|------|
| **Capacitor** | 电容器 Q=CV | 膜电容 C_m≈10pF | 所有"记忆"的物理载体，RC衰减 |
| **MOSFET** | 场效应管 V>V_th导通 | 离子通道（K/Ca/MET） | 阈值判断原语，亚阈值指数区+超阈值线性区 |
| **Memristor** | 忆阻器 w∈[0,1] | 突触权重 | 塑性电阻，STDP修改w |
| **PowerRail** | 电源轨 + 内阻 | 线粒体 ATP | IR降=天然增益限制器 |

### 项目规模

- **85** 个 Python 文件 (~12,000 行代码)
- **45** 个神经元用同一 `Neuron` 类
- **34** 个突触束用同一 `SynapticBundle` 类
- **16** 个集成测试
- **~100+** 个 Obsidian 知识库文档

### nexus_v1 ↔ governance 关系

两个系统**平等并行**，非从属关系：

- **nexus_v1** = "有机体" — 运行物理 + 学习
- **governance** = "审计者" — 监控 + 裁定 + 建模

governance 的五大能力：
1. **Fuse** — 物理定律违反 → 电路熔断
2. **Adjudicator** — 公理合规检查（J1/J2运行时）
3. **Validator** — 参数物理合理性
4. **Modeler** — 独立数学模拟预测
5. **MathCandidate** — 公式生命周期管理

### 3.2 11条核心原则（RULES.md）

| # | 原则 | 核心内容 |
|---|------|----------|
| 1 | **真实对象约束** | 每个半导体组件对应已知生物/物理对象，参数从测量值推导 |
| 2 | **熵账本审计** | 每次修改后检查信号深度、能量守恒、权重演化、无爆炸/崩塌 |
| 3 | **建模先行** | 定位→文献→数学模型→推导参数→实施→熵审计，禁止"试参数" |
| 4 | **归一化方案** | V_norm=(V_bio-E_K)/(E_Ca-E_K)，t_norm=t_bio/dt，g_norm=g_bio/g_ref |
| 5 | **文献溯源** | 每个参数组标注 `# REF:/# BIO:/# NORM:` |
| 6 | **降级追踪** | 异常记录到 `degradation_registry.md`，DEG-XXX编号 |
| 7 | **组件类型分类** | TYPE:BIO/SEMI/MATH/HYBRID/INFRA |
| 8 | **修复记录** | 每次修复记录到 `fix_registry.md`，FIX-XXX编号 |
| 9 | **完备性优先** | 建模→实现→完备性审查→注册→测试调参 |
| 10 | **层接口契约** | 每层定义I/O合格范围，契约破坏=降级 |
| 11 | **母本分化** | 契约隔离+灵敏度矩阵+自动化回归，三防线防级联爆炸 |

### 3.3 信号链（6层前庭转导）

```
力 → Met(L1) → HC(L2) → Aff(L3) → Enc(L4) → Col(L5) → Motor(L6) → Body
    阈值门    3通道放大  脉冲发放   模式识别   多源整合   运动输出
              Ca²⁺释放   regular    T/O/P/R    侧向抑制   肌肉收缩
                        irregular   /Xin循环
```

**6个处理轴**: yaw, pitch, roll (半规管) + oto_x, oto_y, oto_z (耳石)

### 3.4 MetaNeuron（统一神经元）

单个 `Neuron` 类支持多种模式：
- `mode="simple"` — 1 MOSFET，连续输出
- `mode="hair_cell"` — 3 MOSFETs (MET+K+Ca)，Ca²⁺ 门控
- `mode="afferent"` — 1 MOSFET，AdEx脉冲发放
- `mode="multi"` — 3 MOSFETs + 脉冲

补偿组件（可选的半导体反馈回路）：
- **A. VoltageRegulator** — 活动依赖代谢恢复
- **B. DecouplingCapacitor** — 突触输入平滑
- **C. BiasCurrentSource** — 基强自发驱动
- **D. AutomaticGainControl** — 稳态增益适应
- **E. FatigueCapacitor** — 脉冲率依赖阈值适应
- **F. Mitosis** — 持续过载→神经元分裂
- **G. Apoptosis** — 持续能量不足→程序化死亡
- **H. CalciumRateIntegrator** — 脉冲率→连续信号
- **I. DivisiveNormalizationReceptor** — 输入范围适应
- **J. D2Autoreceptor** — DA特异性负反馈

### 3.5 SynapticBundle + STDP

连接源神经元到目标神经元，每个 (src, tgt) 对有一个 Memristor：
- 学习规则: STDP / BCM / frozen
- B层慢速环流反馈
- 静默突触 + 再激活
- C4驻波检测
- 基于成熟阶段的塑性率

### 3.6 VariantCircuit — 闭环扩展

继承 HebbianCircuit，叠加12种变体组件：
1. ResonantOscillator — ISI同步
2. MagnetofluidDamper — 自适应阻抗
3. ExtracellularMatrix — 离子缓冲+PNN
4. VascularCooling — NVC热移除
5. NDRElement — 不应期门控
6. InhibitorySynapse — 侧向抑制
7. LiquidMetalRouter — 动态拓扑
8. Neuromodulator — DA三因素学习
9. Motor→Column反馈 — 运动副本
10. World+Body — 3D环境+热源
11. ThermalMembrane — 体表热感知
12. MuscleSystem — 运动输出→物理力

### 3.7 影子层（ShadowSandbox）

结构复制主系统的 Enc→Col→Mot 子图，接收 Xin 张力作为输入。
- 收缩 = Xin驱动STDP + 链接重排
- 能量受限：E_remodel成本 + 权重饱和 + PNN冻结
- **只读观察者**（不写回主系统）
- 计算收缩、聚类、自由能、运动势

## 4. governance — 熵治理系统

### 4.1 Fuse（熔断器）

5项物理检查：

| 规则 | 检查内容 | 阈值 |
|------|----------|------|
| F1 | 能量守恒（第一定律） | dissipation + stored ≈ input |
| F2 | 熵单调性（第二定律） | dS/dt ≥ 0 over window (100步) |
| F3 | 权重边界 | w ∈ [w_min, w_max] |
| F4 | 膜电压发散 | \|V\| < 50 |
| F5 | 能量非负性 | E ≥ 0 |

行为：
- `enabled=True`: 违反 → `FuseTrippedError`（停止模拟）
- `enabled=False`: 违反 → 记录日志，继续（调试模式）

### 4.2 Adjudicator（裁定者）

| 规则 | 检查内容 | 时机 |
|------|----------|------|
| J1 | 主观层访问客观变量 | 静态分析（代码审查） |
| J2 | 只读观察者修改电路状态 | 运行时 |
| J3 | 学习规则绕过 plasticity_gate | 代码审查 |
| J4 | 新参数无物理来源 | 代码审查 |
| J5 | 公式与 G-001 v2.0 不一致 | 代码审查 |

### 4.3 Validator（验证器）

| 规则 | 检查内容 |
|------|----------|
| V1 | 维度一致性 |
| V2 | 边界合理性 |
| V3 | 数值稳定性（τ/dt > 5） |
| V4 | 对称性来源（物理 vs 人工） |
| V5 | 基线活动影响 |

### 4.4 Modeler（建模器）

独立数学模拟，预修改预测：
- **M1**: 参数扫描
- **M2**: 稳定性分析（线性化特征值）
- **M3**: 增益链追踪
- **M4**: 基线预测 `(bc, R, C, V_th) → V_ss`
- **M5**: 耦合增益验证

### 4.5 MathCandidate（数理候选登记）

公式生命周期：PROPOSED → TESTING → ADOPTED/REJECTED/WITHDRAWN

### 4.6 GovernanceLedger（治理账本）

扩展版熵账本：
- 运行总计（热耗散、spike数、步数）
- 逐层能量/热/活动追踪
- 基线活动 EMA
- 增益链 `L1→L2→L3→L4→L5→L6`
- ds²/κ 历史

### 6.2 核心概念体系

#### 四大电子原语 (P.01-P.04)
- **Capacitor** — 所有"记忆"的物理载体，Q=CV
- **MOSFET** — 阈值判断原语，亚阈值指数区 + 超阈值线性区
- **Memristor** — 塑性电阻 w∈[0,1]
- **PowerRail** — 带内阻恒压源，IR降 = 天然增益限制器

#### 五大元结构
- **MetaUnit** — 完整处理器 = Cap+MOSFET+PowerRail+STDP+钙稳态+成熟度
- **MetaPipe** — 信息传输管道 = Memristor阵列+T/O/P/R/Xin五阶段
- **MetaBlock** — 供电节点，远端自然低能→功能分化
- **MetaFruit** — "便利贴"，创建时拍Xin快照，三向命运
- **MetaGate** — 学习调控器，0(freeze)~1(allow)

#### T/O/P/R/Xin 环路
- **T** (Transfer) — 信号传输（忆阻器权重矩阵）
- **O** (Observe) — 观测当前激活
- **P** (Predict) — 将R的历史重投影到当前R上
- **R** (React) — 实际到达的信号
- **Xin** (心/信息张力) — |P-R|的累积绝对值，唯一驱动物理结构性生长的力

#### 时间-空间-结构三位一体
- **ds²** — 时空度量，ds²=Σ(g_ij·da_i·da_j)
- **ν (nu)** — 运动势，dρ/dt
- **κ (kappa)** — 收缩/膨胀，跨轴相关/自方差
- **核心命题**: 结构来自于时间，空间来自于环流
