### 1.1 Plan: VitalOscillator 实现方案

**文件**: [[01_工程执行/implementation_plan.VitalOscillator实现方案体征搏动器]]

**核心定义**:

> 体征搏动器 = 能量耗散结构的自持振荡，由 EnergyStore 驱动，输出耦合到 Motor 层产生基底随机游走。
> 解决 EXP-015 瓶颈 #2（无基底运动），同时为瓶颈 #1（Shadow 饱和/DA 崩溃）提供时变信号。

**关键设计**:

| 维度 | 内容 |
|------|------|
| **频率** | 2 Hz (心跳频段, 2000 步/周期 dt=0.001) |
| **VdP 参数** | μ=2.0 (弛豫振荡→近方波) |
| **基底振幅** | 0.005 (<< 运动信号 ~0.01) |
| **能量消耗** | ~0.0002/步，≈ basal_drain |
| **死亡开关** | fill < 0.1 → A_vital → 0 |
| **能量-振幅耦合** | A_vital(t) = A_base · fill_fraction(t) |
| **信号路径** | 三路输出: Motor 膜注入 + 内感受 Enc + MotionState 诊断 |

**三路输出设计**:
```
EnergyStore.fill_fraction
    ↓ 振幅调制
ResonantOscillator(f=2Hz, μ=2.0) → output(t)
    ├── → Motor 膜电位注入 (基底驱动)
    ├── → 内感受 Enc 输入 (体征信号, Phase 2)
    └── → MotionState.vital_pulse (诊断)
```

**Noether 能量会计**: 搏动消耗通过 `EnergyStore.withdraw()` 结算 → 自动记账。

**未决问题** (Plan 中的三个 Open Questions):
1. 频率 2 Hz 是否合适？
2. 内感受轴（路径 2）是否在 Phase 1 一起实现？
3. 轴间相位差的物理来源是否需要 CoupledOscillatorArray？

### 1.4 分析评判: 体征模块思考

**文件**: [[archive/undated/analysis_vital_oscillator两份分析评判-体征模块思考]]

**核心论点**:

1. **否决高斯噪声**: 两份分析文件都提出"给 Motor 注入高斯白噪声"——但这是**语义补丁**。所有信号必须有物理来源。
2. **搏动 = 更好答案**: 体征搏动有物理来源（内部时钟 = RC 时间常数），由能量代谢驱动，不是 `random()` 调用。
3. **三路耦合**: 搏动→Motor(基底游走) + 搏动→Enc(内感受) + E_fill 低→进食激活
4. **统一内外**: 外部世界(热源→SkinPatch→Fourier→物理信号) 与 体征(EnergyStore→VitalOscillator→RC振荡→物理信号) **都是能量耗散→物理元件→Noether 会计**。唯一的区别是信号来源(外 vs 内)。

**文件 5 的关键判断** (物理引擎架构师视角):
- DA ≠ reward, DA = surprise/urgency ✅
- 枪毙 ν_th 人工映射 → 违反"拒绝语义标签"铁律 ✅
- Nociceptor = 天然硬件微分器 (τ=4ms, I_c=C·dV/dt) ✅

**但存在物理缺口**: Nociceptor 仅在 damage_integral > 0 时激活(damage_threshold=3.0, T_skin≈1.98) → **从未激活**。需要非伤害性 AC 通道(Thermoreceptor 本身的 dT/dt 分量)。

## 三、与相关概念的交织

### 与 C3' (环流耦合) 的关系

定义三个环流通道:
1. **体征环流 (~100 步)**: ThermalMembrane → ECM → Enc → Col → 稳态参考
2. **运动环流 (~500 步)**: Motor → Body → Oto → Aff → Enc → Col → Motor
3. **摄食环流 (~5000 步)**: thermal error → Col → Motor → Body → 位移 → 热源

VitalOscillator 在体征环流中是**底层振动源** — 最低频的物理振荡，所有其他环流叠加在此之上。

### 与损伤机制的关系

四个反馈环路，均涉及搏动或体征:
- **A 环**: damage → VitalOscillator 振幅衰减 (组织损伤→心脏抑制→搏动减弱)
- **B 环**: damage → EnergyStore 额外代谢修复成本 (组织修复耗能→能量枯竭→饥饿)
- **C 环**: Nociceptor → 方向性 motor 反射 (硬连线, 非学习)
- **D 环 (可选)**: damage → ECM 热 breach (屏障失败→环境热穿透深部组织)

### 与 "类生体层" (孤岛 B2) 的关系

类生体层 = 作为**底层输入源**的 VitalOscillator — 与热源/运动源/声源并列。这是 VitalOscillator 的一个文档化角色——但"底层信源"的定性在 VitalOscillator 的 Plan/Walkthrough 中仅模糊提及。完整定义只在对话日志和孤岛文档中存在。
