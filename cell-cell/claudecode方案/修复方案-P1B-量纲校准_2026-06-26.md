## D-03：Column v_peak 重新推导

### 问题复述

Phase 4-P0 将 Column `v_peak` 从 0.25 下调至 0.10，原因是 coupler 压缩导致 Column 无法放电（EMA=0）。但：

1. `v_peak=0.10` 在 RULES §归一化方案下对应 `V_bio = 0.10 × 130 - 80 = -67mV`，低于钾离子反转电位 E_K=-80mV——**无生物意义**
2. `v_peak=0.10` 与 `v_threshold=0.08` 之间只有 0.02V 窗口，放电极易被微小噪声触发，CRI 积累不稳定
3. 降低 v_peak 是"让测试通过"的参数试错，违反 §原则3（建模先行）

### 生物推导

**Column 神经元的类比对象**：皮层 Layer 2/3 锥体细胞（cortical pyramidal cell），整合来自 Thalamus（L4）的输入并投射到 L5/6（Motor 层）。

```
BIO: L2/3 cortical pyramidal cell
REF: Liu et al. 2014, J Neurophysiol 112:2478-2489
     AP threshold V_th ≈ -50 to -55 mV (mean -52 mV)
REF: Markram et al. 1997, J Physiol 500:409-440
     Resting V_rest ≈ -70 mV
```

归一化（§原则4）：
```
E_K = -80 mV, E_Ca = +50 mV, range = 130 mV

V_rest_norm = (-70 - (-80)) / 130 = 10/130 = 0.077
V_th_norm   = (-52 - (-80)) / 130 = 28/130 = 0.215
V_peak_AP_norm = (+40 - (-80)) / 130 = 120/130 = 0.923  ← 动作电位峰值

对于模拟神经元（仿真中 spike 不到真实 AP 峰值，reset 在 v_reset 处）：
v_peak 应代表 "足够触发 spike 的最低膜电位"，即 V_th_norm = 0.215
```

### 方案一：恢复至 v_peak=0.20（最小改动，有生物基础）

```
v_peak=0.20 对应 V_bio = 0.20 × 130 - 80 = -54mV
接近 V_th_bio=-52mV（Liu 2014），是 AP 阈值的合理近似
```

要让 Column 在 v_peak=0.20 时能放电，需要确保 enc→col 提供足够电流：

```
当前配置：enc→col synapse_gain=3.0, initial_weight=0.15
enc 激活率（连续 EMA）≈ 0.7（来自回归测试）
每步电流 I_col = act × synapse_gain × weight = 0.7 × 3.0 × 0.15 = 0.315
V_ss_col = I_col × R_leak = 0.315 × 5.0 = 1.575V >> v_peak=0.20 ✓

⚠️ 但这是"满载"情形（act=0.7）。静息时（act=0.02，encoding bias only）：
I_col = 0.02 × 3.0 × 0.15 = 0.009
V_ss_col = 0.009 × 5.0 = 0.045V < v_peak=0.20 → Column 静息时不放电 ✓
```

结论：v_peak=0.20 在 enc 驱动时可放电，静息时不自激，**生物合理且无需其他参数变动**。

## D-04：DA 总体量纲校准

### 问题复述

P1-A 方案将 Hunger 信号从直接 inject 改为通过 HungerDriveNeuron + bundle。但 RPE 信号仍直接 inject：

```python
_rpe_da = self.da_gate.step(fill_fraction, dt)
if _rpe_da > 0:
    for nid, neuron in self.da_neurons.items():
        neuron._membrane.inject(_rpe_da, dt)
```

`_rpe_da` 的量纲：
- `DADifferentialGate(η_da=7.5, clip_max=5.0)`
- `_rpe = max(0, η_da × Δfill/dt)`，clip 到 5.0
- 单位：`Δfill/dt` 是"填充率变化速率"（/s），乘以 7.5 后约 0~5 之间
- `inject(5.0, 0.001)` → dQ = 0.005，dV = 0.005/C_da

需确认 C_da 并校验 dV 量级。

### DA 神经元电容查找

```bash
grep -n "da_neuron\|neuron_id.*da\|capacitance" cell-cc-other/nexus_v1/circuit/variant_adapter.py | grep -A2 "da"
```

目标：找到 DA 神经元的 `capacitance` 值。假设与 `Encoding` 神经元类似（`C=0.15`）：

```
RPE peak (clip_max=5.0): inject(5.0, 0.001) → dV = 5.0 × 0.001 / 0.15 = 0.033V/step
τ_DA = R × C = 5.0 × 0.15 = 0.75 → 每步 leak = V × exp(-0.001/0.75) ≈ V × 0.99867
稳态：dV = V × 0.00133 → V_ss_rpe = 0.033 / 0.00133 ≈ 25V（远超 v_peak）
```

这说明 RPE 信号在 DA 神经元膜上也是**超量注入**，会立即驱动到饱和放电。

### RPE DA 信号的结构化修复（P1-B 范围内的完整修复）

目标：RPE 信号也应通过 bundle 传递到 DA 神经元，但 RPE 信号来自 `DADifferentialGate`（非 Neuron），需要一个中继神经元：

```
DADifferentialGate.step() → float (_rpe_da)
    ↓ step()
RPE_Relay_Neuron (TYPE:HYBRID — VTA burst detector)
    ↓ SynapticBundle: rpe_to_da
DA Neurons
```

**但**：这属于完整的 DA 电路结构化重建，超出 P1-B 的范围。

### P1-B 的最小修复：RPE 量级校准

在 RPE 继续直接注入的前提下，将注入量纲调整到合理范围：

**目标**：RPE 峰值（_rpe_da=5.0）使 DA 膜电压增量 ≤ v_peak 的 50% 每步。

以 C_da=0.15，v_peak=0.35 为例：
```
目标：dV ≤ 0.35 × 0.5 = 0.175V/step
实际：inject(rpe, 0.001) → dV = rpe × 0.001 / 0.15 = rpe × 0.00667
要满足：rpe × 0.00667 ≤ 0.175 → rpe ≤ 26
当前 clip_max=5.0 → dV_max = 5.0 × 0.00667 = 0.033V/step（实际上不大）
```

等等——重新计算，dV/step = 0.033V 看起来合理（而不是之前我估计的"太大"）。

问题不在于每步的 dV，而在于稳态（如上面计算：V_ss=25V）。但 DA 神经元有放电复位机制（spiking=True, v_reset），一旦达到 v_peak 就 reset。所以稳态不是无界的。

**真正的问题**：DA 神经元如果每步被 inject 0.033V，而每步 leak 约 0.001V（dt/τ=0.00133），则膜电压每步净增 0.032V，在 `v_peak/0.032 ≈ 11 步`内就从 0 充到 v_peak 并放电。放电后 reset 到 v_reset，然后再次被充电。这意味着 DA 在 RPE 活跃期以最大速率连续放电（每 11 步一次 spike，约 90Hz）——确实是"饱和"状态。

这与 Phase 4 实验观察一致：前 20k 步 DA=1.0（满激活）。

### 量纲校准方案

调整 `DADifferentialGate` 的 `clip_max` 或引入 DA 注入比例因子：

```python
# circuit/variant_adapter.py，约第 894-901 行

# 目标：RPE 高峰时 DA 神经元达到约 50% 激活（不饱和）
# DA 神经元稳态：50% act 对应 V_steady ≈ v_peak × 0.5 ÷ MOSFET_curve
# 简化：目标 dV/step ≈ v_peak × 0.1 = 0.035V/step
# → inject_current = dV × C / dt = 0.035 × 0.15 / 0.001 = 5.25A

# 等等，这需要知道 MOSFET 曲线。简单量纲方法：
# inject(I_da, dt)，稳态 V_ss = I_da × R_leak（当没有 spike 复位时）
# 目标 V_ss < v_peak → I_da < v_peak / R_leak = 0.35 / 5.0 = 0.07
# 当前 _rpe_da 最大值 = 5.0，需要一个 scale factor
DA_INJECT_SCALE = 0.07 / 5.0  # = 0.014
                                # V_ss(RPE peak) = 5.0×0.014×5.0 = 0.35 = v_peak ✓
                                # → DA 在满 RPE 时稳态达到 v_peak（边界，会放电）
                                # 实际 V_ss < v_peak（有 spike 复位），合理

_rpe_da = self.da_gate.step(self.energy_store.fill_fraction, dt)
if _rpe_da > 0:
    _scaled_rpe = _rpe_da * DA_INJECT_SCALE
    # BIO: Schultz 1997 — VTA burst rate ≤ 20 Hz on unexpected reward
    # NORM: DA_INJECT_SCALE=0.014, clip_max=5.0 → V_ss(peak)=0.35V ≤ v_peak
    # REF: 皮层除颤方案 §DA calibration; EXP-013 (if available)
    for nid, neuron in self.da_neurons.items():
        neuron._membrane.inject(_scaled_rpe, dt)
```

常量 `DA_INJECT_SCALE = 0.014` 标注为 `# ESTIMATED: requires EXP-da-calibration`。

### 合并效果：P1-A + P1-B 后的 DA 驱动

| 来源 | 机制 | 峰值 dV/step | 持续性 |
|------|------|------------|--------|
| RPE (Schultz) | inject × scale | 0.033×0.014/0.014? 重算 | 瞬时（Δfill 快速归零） |
| Hunger (LH) | HungerNeuron → bundle | ~0.001V/step | 持续（fill<0.5 期间） |

两者在 DA 膜上叠加（inject + bundle 电流），互相独立，DA 神经元的膜动力学作为天然低通滤波器。
