## FIX-P0-01: Deviation Motor Gain dt 缩放（D-01）

### 问题

`DEVIATION_MOTOR_GAIN=1.0` 在 EXP-012（`dt=1.0`）下校准。在 Phase 4（`dt=0.001`）中，`inject(motor_drive, dt)` 每步增加的膜电荷 = `motor_drive × dt`，比 dt=1.0 时小 **1000×**，导致 deviation reflex 的每步充电量仅约 0.0003V/step，无法在合理时间窗口内驱动 Motor 放电。

### 量纲推导

`inject(I, dt)` 的稳态：
```
V_ss = I × τ / C = I × R  (对 RC 电路成立，τ=R×C 消掉)
```
`V_ss` 与 `dt` 无关——**稳态电压相同**。但到达稳态的时间常数 τ = R × C = 5.0 × 0.3 = 1.5 仿真单位 = **1500 步**（dt=0.001）。

换句话说：deviation reflex 在 dt=1.0 下 ~2 步内生效，在 dt=0.001 下需要 **1500 步（1.5s）**才能到达稳态电压。对于 Δf=25Hz 量级的快速运动，这意味着 reflex 事实上是滞后的。

对于 Phase 4 来说，这不是最关键的问题（deviation reflex 在当前架构中是语义硬编码，见 V-03），但量纲应记录正确。

### 修复

**文件**: `circuit/variant_adapter.py`，约第 909 行

```python
# BEFORE（EXP-012 在 dt=1.0 下校准，注释量纲错误）:
# Phase 1 calibration (EXP-012): 0.05 produced 0.0000045V/step
# ("whispering in a hurricane"). 1.0 → 0.09V/step at deviation=0.19
DEVIATION_MOTOR_GAIN = 1.0
```

```python
# AFTER（注释修正，标注 UNGROUNDED，说明 dt 依赖关系）:
# BIO: hypothalamic thermoregulatory drive → locomotor CPG (Satinoff 1978)
# CALIBRATION: EXP-012 at dt=1.0; τ_response = R×C/GAIN ≈ 16.7s at dt=0.001
# The same inject() value produces the same V_ss regardless of dt, but
# the response time is 1500× slower at dt=0.001 vs dt=1.0.
# UNGROUNDED: gain value needs re-calibration for dt=0.001 environment.
# At dt=0.001: to match EXP-012 behavior, raise to ~1000.0 OR accept slow response.
DEVIATION_MOTOR_GAIN = 1000.0   # FIX-P0-01: dt=0.001 re-calibration
```

**注意**：若 V-03（结构重建）先完成，则此行连同整个 deviation reflex 块一起删除，FIX-P0-01 不再需要。所以这是一个临时修复，优先级低于 V-03。

若暂时保留 deviation reflex，更改 `DEVIATION_MOTOR_GAIN = 1000.0` 可在 Phase 5 实验中立即测试其有效性。

### 修复

**文件**: `components/langevin_noise.py`，第 62-69 行

```python
# BEFORE:
    # OU correlation time (s) — finite bandwidth, avoids white noise pathology.
    # BIO: hair cell membrane time constant τ_hair ≈ 0.1-1.0 s
    # Chosen as 0.5 s to sit within the hair cell integration window.
    tau: float = 0.5
```

```python
# AFTER:
    # OU correlation time (s) — finite bandwidth, avoids white noise pathology.
    # Phase4: EXTENDED from 0.5→3.0 to produce macroscopic body displacement.
    # BIO (original, tau=0.5): hair cell membrane time constant τ_hair ≈ 0.1-1.0 s
    #   REF: Eatock & Songer 2011, Table 1
    # UNGROUNDED (tau=3.0): 3.0s exceeds τ_hair range; the intended timescale is
    #   locomotion-level body diffusion, NOT membrane dynamics.
    #   Closest BIO reference: klinokinesis run time τ_run ~ 1-10 s (Berg & Brown 1972,
    #   PNAS 69:1678). τ=3.0 sits within this range but the component is misnamed
    #   (hair cell noise vs body locomotion diffusion).
    # TODO: Consider splitting into separate ThermalNoise (τ=0.5) + LocomotionDiffusion (τ=3.0)
    tau: float = 3.0  # Phase4: macroscopic displacement; BIO target is locomotion timescale
```

## FIX-P0-04: grad_dot_v 写入 MotionState（I-02）

### 问题

`grad_dot_v = Σ(grad_T × vel)` 已经计算，但仅存储了 `mean_dT` 到 `motion_state.thermal_potential`。`grad_dot_v`（"是否朝热源方向移动"）是诊断热趋性行为的关键量，应同时记录。

### 修复

**文件**: `circuit/variant_adapter.py`，约第 1005-1009 行

```python
# BEFORE:
        mean_dT = sum(abs(patch_temps[pid][1])
                      for pid in self.somatosensory.patch_ids
                      ) / len(self.somatosensory.patch_ids)
        self.motion_state.thermal_potential = mean_dT
```

```python
# AFTER:
        mean_dT = sum(abs(patch_temps[pid][1])
                      for pid in self.somatosensory.patch_ids
                      ) / len(self.somatosensory.patch_ids)
        self.motion_state.thermal_potential = mean_dT
        # FIX-P0-04: Record thermal alignment (observability only — never use for control)
        # grad_dot_v > 0 means body is moving toward higher temperature (toward heat source)
        self.motion_state.thermal_alignment = grad_dot_v
```

同时确认 `MotionState` 类中存在 `thermal_alignment` 字段（或添加）：
