## V-03：Deviation Reflex 结构化（P0 降级后的深度修复）

### 问题

```python
# 当前（语义硬编码）：
if deviation > 0.1:
    motor_drive = (deviation - 0.1) * DEVIATION_MOTOR_GAIN
    for key, mot in self.motor_neurons.items():
        mot._membrane.inject(motor_drive, dt)
```

`deviation` 是 CirculationProportion 输出的归一化偏差量。"偏差 > 0.1 → 驱动所有电机" 是把行为目标写进代码。结构正确的路径是让偏差信号经由物理电路驱动电机，STDP 决定哪个轴的响应最有效。

### 三问回答

**Q1 生物对应物**：
- Circulation deviation ≈ 血流分配偏差（某区域供血不足）
- 对应生物结构：下丘脑体温调节中心 → 脊髓运动前神经元（locomotor CPG）
- REF: Satinoff 1978 (已在代码中引用); Romanovsky 2007, Am J Physiol 292:R37

**Q2 物理结构**：
```
CirculationProportion.step() → deviation (float)
    ↓
DeviationRelayNeuron [TYPE:HYBRID — hypothalamic thermoregulatory integrator]
    ↓ SynapticBundle: deviation_to_motor
Motor Neurons (move_x, move_y, move_z)
```

**Q3 参数依据**：
- 设计 DeviationRelayNeuron 的膜参数使其在 deviation=0.19 时激活约 50%
- bundle weight 从 STDP 学习（motor 方向与 deviation 方向相关时 LTP）

### 实现规划

**步骤 1**：创建 `DeviationRelayNeuron`

```python
# variant_adapter.py __init__ 中
self._deviation_relay = Neuron(NeuronConfig(
    neuron_id="deviation_relay",
    # BIO: hypothalamic thermoregulatory neurons, integrative, slow
    # τ = C × R = 1.0 × 5.0 = 5.0s (thermoregulatory integration timescale)
    # REF: Romanovsky 2007 — hypothalamic Tset integration ~minutes, relay ~5-10s
    capacitance=1.0,
    r_leak=5.0,
    spiking=False,   # graded output (hypothalamic relay)
    channels=[ChannelConfig("deviation_ch", v_threshold=0.0, gm=0.5, ...)],
    ...
))
```

**步骤 2**：每步驱动 DeviationRelayNeuron

```python
# 替换 if deviation > 0.1 块：
deviation = circ['deviation']
# BIO: hypothalamic thermoregulatory input ∝ Tcore - Tset
# NORM: deviation [0,1] maps to normalized temperature error
self._deviation_relay.step(deviation, dt)
# 传播通过 bundle（而非直接 inject）
```

**步骤 3**：创建 deviation_to_motor bundle，STDP 学习

```python
# 三问 Q3：initial_weight 和 stdp_lr
# 目标：deviation=0.1 时 Motor 刚好开始被驱动
# deviation_relay act ≈ sigmoid(0.1 × 5.0) ≈ 0.38（粗略）
# 需要：act × synapse_gain × weight × R_motor ≈ 0.1 × v_peak_motor
# 0.38 × 1.0 × weight × 5.0 ≈ 0.02 → weight ≈ 0.01（初始弱，STDP 强化）
cfg_dev_motor = BundleConfig(
    bundle_id="deviation_to_motor",
    learning_rule="stdp",
    initial_weight=0.01,    # ESTIMATED: weak initial connection
    weight_max=0.5,
    stdp_lr=0.01,           # ESTIMATED: moderate plasticity
    synapse_gain=1.0,
)
self._bundle_deviation_to_motor = SynapticBundle(
    cfg_dev_motor, [self._deviation_relay], motor_list)
```

**步骤 4**：删除 `if deviation > 0.1` 整个代码块

**步骤 5**：更新 census（deviation_relay + bundle）

### 风险

- 去掉 deviation reflex 后，CirculationProportion 的热趋性驱动力暂时消失，直到 STDP 学习 deviation→motor 关联
- 建议：Phase 5 实验完成后再实施（不影响当前实验方向）

## V-04：Xin Relay Bundle 化（量纲修正）

### 问题

```python
self._xin_relay.step(self._xin_integrator.voltage * 0.5, dt)
```

`_xin_integrator.voltage`（V 单位）被当作电流（A 单位）输入 Neuron.step()。

### 三问回答

**Q1 BIO**：
- Xin 预测残差（|ξ|）→ 调制 VTA DA 活性
- BIO: 前额叶皮层预测误差信号 → VTA（Corlett et al. 2011, Neuropsychopharmacol 36:52）
- Xin relay 类比：ACC (anterior cingulate cortex) 预测误差神经元

**Q2 物理结构**：
```
Xin Integrator (Capacitor，tracking |dξ/dt|)
    ↓ [SynapticBundle: xin_int_to_relay]
Xin Relay Neuron
    ↓ [Bundle: xin_to_da（已有）]
DA Neurons
```

**Q3 参数**：
- xin_integrator 电压范围：0~1（Zener 箝位）
- bundle synapse_gain：使 xin_v=0.5 时 relay 激活 ~50%
  `I_relay = xin_v × synapse_gain × weight = 0.5 × 2.0 × 0.5 = 0.5A`
  `dV = 0.5 × 0.001 / C_relay` — 需从 xin_relay 配置中读取 C

### 实现规划

删除 `self._xin_relay.step(voltage × 0.5, dt)` 这行，改为：

```python
# 在 _init_da_circuit() 中新增：
cfg_xin_int_relay = BundleConfig(
    bundle_id="xin_int_to_relay",
    learning_rule="none",           # fixed: xin积分→relay是物理硬连接
    initial_weight=0.5,             # ESTIMATED: calibrate in EXP-xin-relay
    synapse_gain=2.0,
    stdp_lr=0.0,
)
self._bundle_xin_int_to_relay = SynapticBundle(
    cfg_xin_int_relay,
    [self._xin_integrator],    # Capacitor 作为 source — 需验证 bundle 支持此用法
    [self._xin_relay]
)
```

**关键问题**：`SynapticBundle` 的 source 必须是 `Neuron` 对象（有 `.activation` 属性）。`Capacitor` 没有这个属性。

**解决方案**：创建 `XinIntegratorNeuron`，它的行为就是把电容电压作为激活输出：

```python
class XinIntegratorNeuron:
    """TYPE:INFRA — Wraps a Capacitor as a Neuron-like source for SynapticBundle."""
    def __init__(self, capacitor):
        self._cap = capacitor
    @property
    def activation(self):
        return self._cap.voltage
    @property
    def _activation_ema(self):
        return self._cap.voltage
    # ... 其他 Neuron 接口属性

self._xin_int_wrapper = XinIntegratorNeuron(self._xin_integrator)
```

这是一个 INFRA 类型的适配器，使 Capacitor 能作为 SynapticBundle 的 source。

### 复杂度评估

**高**：需要引入 `XinIntegratorNeuron` 适配器类，或修改 `SynapticBundle` 接受 duck-typed source。建议单独作为一个 PR，完整测试后合并。

## V-02：VitalOscillator 中继化

### 问题

`self.motor_neurons[mkey]._membrane.inject(vital_outputs[i], dt)` — VdP 振荡器直接打 Motor 膜。

### 方案概要

**需要**：`VitalRelayNeuron` × 3（对应 x/y/z 轴），接受 VdP 振荡器输出作为输入，通过 `SynapticBundle(vital_to_motor)` 连接 Motor。

**Q1 BIO**：心脏节律 → 脑干网状脊髓束 → 脊髓运动前神经元（spinal CPG）
- REF: Pfaff 2006 — brainstem reticulospinal projections to motor pools

**Q2 结构**：
```
VdP Oscillator output
    → VitalRelayNeuron (brainstem reticulospinal neuron)
    → SynapticBundle(vital_to_motor)
    → Motor Neurons
```

**实现挑战**：VdP 输出是交流信号（±），会在 x/y/z 三轴间轻微去相关产生旋转运动。直接注入时保留了此特性。通过 bundle 后，全波整流（activation≥0）会改变行为。可能需要允许负值 activation 的特殊配置。

**优先级**：低（当前直接注入功能正确，KCL 合规，行为符合设计）。改造主要是原则合规，不改变功能。

## V-05：Lateral Inhibition Bundle 化

### 问题

`col._membrane.inject(inhibitions[idx], dt)` — Python WTA 结果直接注入 Column 膜。

### 方案概要

**正确结构**：
```
Column_A 激活 → [InhibitorySynapse: col_A_to_B_inhib] → Column_B（抑制）
```

每个 Column 对之间一条 `InhibitorySynapse`（系统已有此原语，在 SomatosensoryChain 的 `lateral_gain` 中使用）。对于 N 个 Column 轴，需要 N×(N-1) 条抑制连接（7 个轴 → 42 条）。

**参数**：
```
initial_weight = 0.05  # 弱初始抑制（避免多轴运动全被压到 0）
weight_max = 0.2       # 上限：防止过强 WTA 导致单轴垄断
lateral_gain 参数已在 Phase 4 调为 0.3（对 SomatosensoryChain）
```

**挑战**：42 条连接的初始化和管理较复杂；且当前 WTA 强度（导致所有轴归零）是已知问题，改为 bundle 后参数需重新校准。

**优先级**：低（当前 WTA 本身就是问题，无论是 Python 实现还是 bundle 实现都需要先校准强度）。


3. 典型模式（应有）：
   ```python
   self.K_ema = self.K_ema * (1 - alpha) + K_new * alpha  # EMA，有自然衰减
   ```
