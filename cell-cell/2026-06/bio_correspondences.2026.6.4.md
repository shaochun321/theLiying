# 生物学对应与可借鉴的理论/项目

## 概念 → 生物学映射

### 1. 垫支 → 记忆固化 + 髓鞘化

| 我们的概念 | 生物学对应 | 关键论文 |
|-----------|----------|---------|
| Fruit: growing → ripe | **突触标记与捕获** (Synaptic Tagging & Capture) | Frey & Morris 1997 |
| Maturation: spine → area | **关键期关闭** (Critical Period Closure) | Hubel & Wiesel 1970 |
| 垫支 = ripe + area | **髓鞘化** (Myelination) | Fields 2008 |
| 基底冻结 | **围神经元网 (PNN) 包裹** | Pizzorusso 2002 |

**突触标记与捕获**：STDP 产生 early-LTP（"标记"） → 如果持续激活 → 蛋白质合成 → late-LTP（"捕获"） → 永久增强。这就是 growing → ripe 的两步过程。

**髓鞘化**：频繁使用的轴突被胶质细胞包裹髓鞘 → 传导速度增加 10-100 倍 → 连接变得快且稳定。这是**物理意义上的垫支**——给线路加绝缘层。

**我们已经有 PNN**：`ecm.py` 中的 `pnn_maturity` 正在积累。PNN 包裹神经元 → 可塑性降低 → area 状态。代码和生物学在这里是**直接对应的**。

### 2. 影子层 → 海马-皮层转移

| 我们的概念 | 生物学对应 | 关键理论 |
|-----------|----------|---------|
| 影子层接收主层 Xin | **海马接收皮层信号** | 海马是"fast learner" |
| 影子层跨轴 STDP | **海马内的快速关联学习** | 一次曝光即可学习 |
| 影子层 ripe → 反传播主层 | **睡眠中的海马重播 → 皮层固化** | McClelland 1995 |
| 主层获得新连接 | **皮层慢速学习** | 需要多次重播才能稳定 |

**互补学习系统理论** (Complementary Learning Systems, McClelland et al. 1995)：

```
海马（影子层）：快速学习，一次即可
  → 在离线时（睡眠）重播
  → 稳定的模式传递给皮层

皮层（主层）：慢速学习，需要多次曝光
  → 接收海马重播后逐渐整合
  → 成为长期记忆（垫支）
```

**这正是"侦察兵"模式的生物学原型。** 海马快速试探 → 成功的模式通过重播传给皮层 → 皮层缓慢固化。

**可直接借鉴**：
- 海马重播的时间压缩比 ≈ 5-20×（我们的 SHADOW_K=10 接近）
- 重播只发生在"安静"时段（可以设定影子层只在低 Xin₀ 时反传播）

### 3. 多尺度内化 → 系统固化理论 + 图式理论

| 时窗 | 我们的 | 生物学 |
|------|--------|--------|
| 短（10s） | 一阶驻波 | **工作记忆** (prefrontal cortex, θ-γ coupling) |
| 中（100s） | 二阶驻波 | **情景记忆** (hippocampal indexing) |
| 长（1000s+） | 垫支/基底 | **语义记忆** (neocortical schema) |

**图式理论** (Tse et al. 2007)：一旦皮层建立了"图式"（= 垫支后的基底结构），新的一致信息可以**跳过海马**直接整合到皮层。

在我们的系统中：如果 $\mathcal{H}_{base}$ 足够丰富，新的输入模式如果与 base 一致，可以不经过影子层直接被主层吸收 → 学习加速。

### 4. 时空偏振 → 位置细胞 + 时间细胞

| 我们的 | 生物学 | 发现者 |
|--------|--------|--------|
| Aff_reg（空间偏振） | **位置细胞** (Place Cells) | O'Keefe 1971 |
| Aff_irr（时间偏振） | **时间细胞** (Time Cells) | MacDonald 2011 |
| 跨轴 bundle（时空耦合） | **网格细胞** (Grid Cells) | Hafting/Moser 2005 |
| 影子层 ds²（时空间隔） | **θ 相位进动** (Phase Precession) | O'Keefe & Recce 1993 |

**网格细胞是关键借鉴对象**：

```
网格细胞 = 内嗅皮层中的六角格子
  → 提供空间的"度量衡"（不依赖感觉输入的内在坐标系）
  → 通过路径积分（速度积分）自主更新位置估计

我们的 SpatialNavigator（当前 stub）应该模仿网格细胞：
  → 积分 otolith 信号 → 估计位置
  → 用六角格子离散化空间 → 产生空间测度
```

**θ 相位进动**：位置细胞的放电时间在 θ 节律内前移 → 编码"距离到目标"。这是**时间编码空间**——正是时空偏振耦合的生物学实例。

### 5. T/O/P/R/Xin → 预测编码 + 自由能原理

| 我们的 | Friston 体系 | Rao & Ballard |
|--------|-------------|---------------|
| P | 先验预测 $\mu$ | 自上而下预测 |
| R | 感觉输入 | 自下而上输入 |
| Xin | **预测误差** $\epsilon = \mu - s$ | 残差信号 |
| sprout (Xin > θ) | **模型复杂度增加** | 增加层/神经元 |
| 垫支 | **精度加权** (precision weighting) | 确定性高的预测不再更新 |

**自由能原理** (Friston 2010)：

$$F = -\ln p(s|\mu) + D_{KL}[q(\theta)||p(\theta)]$$

系统最小化自由能 = 最小化预测误差 + 保持模型简单。

**我们的系统已经在做这件事**，但没有显式的自由能目标函数。影子层的 K = Σξ² 就是自由能的近似。

**关键区别**：Friston 的体系是连续的（微分方程），我们的是离散的（spike + STDP）。但数学结构同构。

### 6. 驻波 → 神经振荡 + 共振

| 我们的 | 生物学 |
|--------|--------|
| 一阶驻波（单信号） | **γ 振荡** (30-100 Hz)：局部回路共振 |
| 二阶驻波（跨信号） | **β 振荡** (15-30 Hz)：长距离同步 |
| 垫支（基底固化） | **α 振荡** (8-12 Hz)：皮层静息态 |
| 回荡（Xin 振荡） | **θ 振荡** (4-8 Hz)：海马信息整合 |

**通信通过相干** (Communication Through Coherence, Fries 2005)：

两个脑区只有在振荡相位对齐时才有效通信。这给了我们 CPG 相位耦合的生物学支持——不同轴的 motor 在 CPG 相位对齐时协调运动。

## 可借鉴的项目/理论

### ① Nengo / 语义指针架构 (Chris Eliasmith, 滑铁卢)

**借鉴什么**：向量符号架构 (VSA)——用高维向量表示概念，向量叠加/绑定实现"与/或"逻辑。

**为什么相关**：Motor 分化可以用 VSA——每个 motor 不是标量（0/1），而是一个高维向量。不同 motor 编码不同的运动"概念"。

**借鉴难度**：中等。需要把 neuron 的标量激活改为向量。

### ② 海马重播机制 (Buzsáki / Foster)

**借鉴什么**：尖波涟漪 (Sharp-Wave Ripple, SWR) 期间的压缩重播。清醒时的经历在睡眠时以 5-20× 速度重播。

**为什么相关**：影子层的反传播可以模仿 SWR——在"低活动"时段，影子层把成熟模式以压缩形式"注入"主层。

**借鉴难度**：低。只需在 Xin₀ < 阈值时触发影子层→主层的权重传递。

### ③ 网格细胞路径积分 (Moser / Burak & Fiete)

**借鉴什么**：连续吸引子网络 (Continuous Attractor Network)——一群神经元形成六角格子，通过速度输入积分位置。

**为什么相关**：SpatialNavigator 的实现。不需要外部坐标——网络自身的吸引子动力学就能维持位置估计。

**借鉴难度**：高。需要专门设计吸引子网络拓扑。但可以先用简化版（直接积分 otolith）。

### ④ 预测编码层次模型 (Rao & Ballard 1999)

**借鉴什么**：每层只传递预测误差给上层（不传原始信号）。上层只发送预测给下层。

**为什么相关**：我们的 T/O/P/R/Xin 递归。可以让 Enc→Col 只传 Xin（不传原始激活），Col 向下发送 P。

**借鉴难度**：中等。需要修改 bundle 的传播方式——目前传的是信号，不是误差。

### ⑤ 互补学习系统 (McClelland 1995 / Kumaran 2016)

**借鉴什么**：海马（快速学习）+ 皮层（慢速固化）的双系统架构。

**为什么相关**：直接对应影子层（快速）+ 主层（慢速）。包括"重播时的交错训练"防止灾难性遗忘。

**借鉴难度**：低-中。我们已有双层结构，只需加反传播通道。

### ⑥ 突触标记模型 (Clopath et al. 2008)

**借鉴什么**：三因子学习规则——STDP 只做"标记"，需要第三个信号（多巴胺/新奇）才能固化。

**为什么相关**：Fruit 的 growing → ripe 转变。当前只要 STDP 累积就自动 ripe。加入三因子后，ripe 需要 DA 确认 → 只有"有意义的"学习才被固化。

**借鉴难度**：低。DA 已存在但在 baseline。让 DA 参与 Fruit 状态转移即可。
