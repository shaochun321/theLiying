# 审阅：环流记忆的落地问题

## 一、核心矛盾

一个 `float ρ_normal_ema = 0.7` 不是记忆——它是**作弊**。

真实的记忆必须满足：
1. **存储在结构中**（不是存储在变量中）
2. **可被信号读出**（不是被代码直接引用）
3. **可被经历修改**（不是硬编码常量）
4. **有物理代价**（遗忘、干扰、容量限制）

EMA 作为 float 变量违反了全部四条。问题是：如何用系统已有的物理基底来存储环流的"正常模式"？

---

## 二、文献发现

### 发现 1：吸引子即记忆（Hopfield → 动力学视角）

> "稳态不是一个存储的数值——它是网络动力学的不动点。"

文献表明：稳态设定点**不需要显式存储**。如果回路的权重和拓扑定义了一个吸引子，系统自然会趋向这个吸引子。"正常状态"就是吸引子本身。

**在我们的系统中**：如果影子层的 STDP 在长期稳定输入下将权重调整到使 ρ_homeo ≈ 0.7 成为吸引子，那么这个 0.7 就**隐式编码在权重矩阵中**——不需要单独存储。

### 发现 2：多时钟组（Polychronous Groups, Izhikevich 2006）

> "具有异质突触延迟的网络可以形成时间锁定的非同步发放模式。"

这些"多时钟组"在网络中**循环传播**——这就是环流模式的记忆。每个稳定的环流模式对应一个多时钟组。记忆不是存储在一个变量中，而是存储在**发放时序模式**中。

**在我们的系统中**：三个子系统（体征/运动/进食）的环流各有不同的延迟路径。稳定态 = 三个多时钟组的叠加。失衡 = 某个组被破坏 → 其他组被激活以补偿。

### 发现 3：Turrigiano 的突触缩放（Synaptic Scaling）

> "神经元监测自己的平均放电率，通过全局调整突触强度来维持设定点。"

**机制**：钙依赖传感器 → 监测平均去极化水平 → 补偿性调整 AMPA 受体密度。

**关键启示**：设定点的物理载体是**受体密度**——一种**结构性编码**。不是一个数值，而是"这个神经元表面有多少受体"。

**在我们的系统中**：对应的是 Neuron 的 `config` 参数（阈值、时间常数、通道增益）。如果这些参数随长期活动缓慢调整，它们就成为了设定点的结构载体。

### 发现 4：异位调节（Allostasis, Sterling 2012）

> "设定点本身可以移动——这是异位调节而非稳态调节。"

Hammel 模型 + 环形吸引子网络：活动的"凸起"在环形空间中移动，代表当前的设定点位置。外部输入可以推动凸起 → 设定点改变。

**在我们的系统中**：环流比例的"正常态"不是固定的 0.7/0.2/0.1——它本身可以随经历而移动。例如，如果热源长期消失，系统应该降低 ρ_feed 的"正常值"。

## 三、三种候选方案

### 方案 A：吸引子方案（推荐）

**思路**：不显式存储 ρ_normal。让影子层的权重矩阵定义吸引子。

```
影子层 STDP 在稳定期学习 → 权重矩阵 W_shadow 收敛
W_shadow 的不动点 = ρ_normal（隐式）
当主层偏离 → 影子层 Xin₁ ≠ 0 → DA 释放
当主层回归 → 影子层 Xin₁ → 0 → DA 回落
```

**落地方式**：
- 影子层已经有 STDP（在 shadow_sandbox.py 中）
- 影子层已经计算 K = Σξ²（总自由能）
- C1 已经连通 ν = dK/dt → DA
- **不需要新代码**。影子层的现有权重矩阵就是环流记忆的物理载体。

**数学基底**：Hopfield 能量函数

$$E = -\frac{1}{2} \sum_{ij} W_{ij} s_i s_j + \sum_i \theta_i s_i$$

稳态 = $\nabla E = 0$ 的不动点。环流比例是 $s$ 向量在不动点处的分量比例。

**优点**：零新代码、纯结构计算、与理论一致
**缺点**：不可显式读出 ρ_normal 的数值（它是隐式的）

### 方案 B：结构参数方案

**思路**：将 ρ_normal 编码进 Neuron 的 config 参数。

```
长期稳定态 → 影子层某些 neuron 的阈值/增益缓慢适应
适应后的参数值 = 该子系统"正常"激活水平的编码
偏离正常 → 激活偏离适应后的阈值 → 误差信号
```

**落地方式**：
- 影子层 neuron 的 `v_threshold` 随长期 EMA 缓慢调整
- 调整时间常数 τ ≈ 100000 步（远慢于 STDP）
- 阈值本身就是"受体密度"的类比 → Turrigiano 机制

**数学基底**：BCM 的滑动阈值 θ_m

$$\frac{d\theta}{dt} = \frac{a^2 - \theta}{\tau_\theta} \qquad (\tau_\theta \gg \tau_{STDP})$$

θ 追踪活动的平方均值 → θ 就是"正常活动水平"的结构编码。

**我们已有这个机制**——BCM 的 `theta_m` 在 bundle.py L244 中实现。只是 τ 太短（100），需要大幅增加。

**优点**：有生物学支持（Turrigiano）、可读出
**缺点**：需要区分"哪些参数编码体征/运动/进食"

### 方案 C：混合方案

**思路**：A + B 的层级组合。

```
层级 1（快速，秒级）：权重 W 变化 → 当前环流模式
层级 2（中速，百秒级）：影子层 W_shadow 变化 → "正常"模式的吸引子
层级 3（慢速，千秒级）：θ_m 参数变化 → 设定点的结构固化

层 1 偏离层 2 → DA 释放（C1 已实现）
层 2 偏离层 3 → ？（需要新机制）
层 3 = 垫支后的基底 → 极其稳定
```

**对应你的三层记忆**：

| 层级 | 对应 | 编码 | 时间常数 |
|------|------|------|---------|
| 层 1 | 化学编码（离子浓度） | W, DA, Xin | dt=0.001s |
| 层 2 | 表观遗传（STDP 沉积） | W_shadow | τ≈100s |
| 层 3 | 基因组（结构参数） | θ_m, v_threshold | τ≈1000s |

## 四、建模候选

| 数理框架 | 描述什么 | 与环流记忆的关系 |
|---------|---------|---------------|
| **Hopfield 能量** | 吸引子不动点 | ρ_normal 是能量极小值 |
| **BCM 滑动阈值** | 活动依赖的设定点漂移 | θ_m 编码"正常活动" |
| **自由能原理** | 预测误差最小化 | 环流回归 = 最小化 F |
| **环形吸引子** | 连续吸引子上的凸起移动 | ρ 比例是环形空间中的角度 |
| **多时钟组** | 时序锁定的循环模式 | 环流 = 多时钟组的稳定模态 |

**推荐**：先用方案 A（吸引子，零新代码），观察影子层权重是否自然收敛到稳定模式。如果不收敛 → 追加方案 B（θ_m 慢适应）。

## 五、对整体设计的审阅

### 设计中成立的部分

1. **环流比例替代标量奖励** ✅ — 比"热源→DA"更丰富、更真实
2. **三子系统耦合** ✅ — 体征/运动/进食的比例关系有生物学支持
3. **影子层作为比较器** ✅ — C1 已建立通道
4. **震荡=环流的时间视图** ✅ — ZCR 可以检测主导模态切换
5. **C5 自然溶解到这个框架中** ✅ — Fruit 成熟条件 = DA 低 + ZCR 稳定

### 设计中需要警惕的部分

1. **ρ_homeo = 0.7 从哪里来？** — 如果用方案 A，这个数值由影子层权重自然决定。但如果影子层权重初始随机，"正常态"可能收敛到任意比例。需要初始化约束（= 方案 C 的"基因组"层）。

2. **三个子系统的边界** — 环流路径实际上是交叉的（运动环流经过的 Col 也参与体征环流）。分离三个振幅需要某种解耦机制——或者接受它们天然不可完全分离。

3. **时间尺度分离** — 体征环流 (~0.1s) / 运动环流 (~0.5s) / 进食环流 (~5s) 的时间尺度差 50 倍。影子层的 STDP 时间常数需要足够长，才能同时编码三个尺度的"正常态"。

### 建议的执行路径

```
当前 → 不加新代码，跑长时间实验（100k+），
       观察影子层权重是否在稳定输入下自然收敛

如果收敛 → 方案 A 成立，影子层即环流记忆
如果不收敛 → 追加方案 B（θ_m 慢适应，τ=100000）
最终 → 方案 C（三层记忆的完整层级）
```
