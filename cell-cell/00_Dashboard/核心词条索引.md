## 三、信号流：T/O/P/R/Xin 环路

> 源文件：[[2026-05/analysis_meta_guide.2026.5.20]] · [[2026-05/modeling_hypergraph_math.2026.5.22]] · [[2026-05/analysis_project_architecture_map.2026.5.23]] · [[2026-05/system_narrative.2026.5.31]]

### 3.1 六层前庭链

```
Met(L1) → HC(L2) → Aff(L3) → Enc(L4) → Col(L5) → Motor(L6) → Body → Oto → ...
```

| 层 | 词条 | 核心功能 |
|----|------|----------|
| L1 | **Met** | 阈值门：信号<0.3忽略，≥0.3准入 |
| L2 | **HC** (hair cell) | 3通道放大(met/k/ca MOSFET) + Ca²⁺ 释放 |
| L3 | **Aff** (afferent) | AdEx脉冲发放。分叉：Regular (tonic→DC/重力) 和 Irregular (phasic→AC/加速度) |
| L4 | **Enc** (encoding) | 融合 regular+irregular 流，T/O/P/R/Xin 循环 |
| L5 | **Col** (column) | 多源整合→决策输出，侧抑制(半径3) |
| L6 | **Motor** | 脉冲→肌肉收缩，max 20/轴(P2约束)，过载时有丝分裂克隆 |

### 3.2 五阶段信息处理

| 阶段 | 词条 | 核心含义 |
|------|------|----------|
| **T** | Transfer | 信号传输（忆阻器权重矩阵） |
| **O** | Observe | 观测当前激活 |
| **P** | Predict | **将R的历史重投影到当前R上**：P = T·O·R_current |
| **R** | React | 实际到达的信号 |
| **Xin** | 心/信息张力 | **\|P-R\|的累积绝对值**。唯一驱动物理结构性生长的力 |

> ⚡ **核心理念**：P 不是猜测——是结构(T)将过去R的沉积，投射到现在的R上。Xin = \|P-R\| 度量的是**结构的时效性**。

## 五、环流 (Circulation)

> 源文件：[[archive/undated/analysis_circulation_structural_anatomy.环流的结构解剖 — 三层追踪]] · [[2026-05/analysis_hypergraph_circulation_foundation.2026.5.23]] · [[archive/undated/design_homeostatic_circulation.C3-设计-进食-运动-体征的环流耦合]] · [[2026-06/review_circulation_memory.2026.6.5]]

### 5.1 环流定义

| 词条 | 核心含义 |
|------|----------|
| **环流** | **不依赖专用管道**——是在现有拓扑上的**涌现动力学模式**。路径：Col→Bind→Mot→FBCap→Col |
| **三层分布** | 母结构(纯前馈)、交感层(大环流→Col→Bind→Mot→FBCap→Col)、影子层(微环流→跨轴双向) |
| **CirculationProportionCircuit** | **开环传感器**（不是环流！），测量三类信号比例→偏差检测→MOSFET比较→DA电流 |
| **TemporalCoupler B层** | 环流的**强度调节器**（不是环流本身），调节每个"河床"段的流量 |

### 5.2 三类环流比例

| 环流类型 | 周期 | 路径 |
|----------|------|------|
| **ρ_homeo** (稳态) | ~100步 (0.1s) | Thermal→ECM→Enc→Col |
| **ρ_motor** (运动) | ~500步 (0.5s) | Motor→Body→Oto→Aff→Enc→Col→Motor |
| **ρ_feed** (摄食) | ~5000步 (5s) | Thermal→Col→Motor→Body→位移→Thermal |

**ρ 向量**：(p_c, p_b, r_c, r_b, m_b, x, u) ∈ Δ⁶ 单纯形，L1范数=1

> ⚡ **核心理念**："回报不是标量——回报是环流模式回到平衡的过程。" ρ向量取代标量回报信号。

### 5.3 环流记忆

| 方案 | 原理 | 新代码量 |
|------|------|----------|
| **A: 吸引子** | 影子层权重矩阵隐式定义吸引子 (Hopfield能量函数) | 零 |
| **B: 结构参数** | 编码在神经元配置的BCM滑动阈值中 | 小 |
| **C: 混合** | 三层记忆(快W, 中W_shadow, 慢θ_m) | 中 |
