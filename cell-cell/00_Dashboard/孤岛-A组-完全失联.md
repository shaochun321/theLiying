## A2. 新原点 (New Origin) — 自我涌现

**对话来源**: `02_AI对话日志/Hebbian Circuit Structural Optimization.2026.5.16.1.md` (L3606-3703), `02_AI对话日志/Hebbian Circuit Structural Optimization.2026.5.16.md`

**时间**: 2026-05-16

**内容摘要**:

用户提出了项目最深刻的哲学-数学理念之一：**"新原点"**。

**核心定义**:

新原点不是物理引擎的几何原点 `(0, 0, 0)`，而是一个**概率构造**：

```
x* = argmax P(divergence(Phi) → 1)

其中:
  Phi = 运动-感知循环中的内部状态向量
  divergence = 预期位置 vs 实际位置的不匹配度
  P(divergence → 1) = 系统通过反复运动实践累积收敛的概率密度
```

**"自我"的数学定义**:

1. 系统在初始状态下没有"自我"概念
2. 每次运动-感知循环产生预期与实际之间的 divergence
3. 系统通过 Hebbian 学习发现：某类特定运动模式下的 divergence 趋近于 1（完美匹配）
4. 这些运动模式的"交集点"就是 x*——系统推断出的"自己的身体位置"
5. x* 不是固定的——它随运动实践不断更新

**架构讨论**:

- 方案 A: 新原点作为赫布超图中的独立 origin 层（用户选中此方案）
- 方案 B: 新原点作为特殊神经元嵌入现有层
- 选中理由: "独立层 + 带宽可扩展，不污染已有拓扑"

**当前状态**: 完全丢失。无任何 Dashboard/Plan/Walkthrough/Analysis 提及。

## A6. 信号补偿 / 驻波变压器

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.6.7.1.md`

**时间**: 2026-06-07

**内容摘要**:

**核心机制**: 一种自适应信号匹配结构，在不同时空粗粒度区域之间进行 dt/tau 对齐。

**类比**: 变压器（transformer）——在不同电压等级的电网之间传递功率。项目中的"电压"对应 dt/tau 比率。

**工作原理**:
1. 前端信号有特定的 dt/tau（时间/空间分辨率）
2. 后端处理有另一个 dt/tau
3. 不匹配 → 信号质量下降 → 系统性能受损
4. 自适应结构插入中间 → 实时匹配前后端 → 信号完整传输

**吸引子定义**: 自适应结构的收敛目标 = 前后端时间环流和空间环流的函数。结构本身不编码"正确匹配"——它在使用中被梯度驱动至吸引子。

**震荡→驻波→结构**:
```
震动（噪声）→ 被阈值筛选 → 进入自适应结构
→ 回路增益 > 1 → 震荡
→ 震荡在匹配成功时稳定 → 驻波
→ 驻波在解构器中被分解 → 结构沉积
```

**用户决定**: "先撤回。如果对无用脉冲进行补偿和放大，项目会混沌。" 但未完全否定——"后续再做"。

**当前状态**: 完全无文档化。`矛盾·遗留·生长点` G12 (元学习/自愈参数) 和 G26 (BCM 滑动阈值) 有微弱关联但不涉及信号补偿机制。

## A3. 平行构建分析中的关键架构洞察

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.6.10.md` (L495+), `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.6.10.1.md` (L135+)

**时间**: 2026-06-10

**内容摘要**:

收到平行构建 `p3g_dynamic_ecology` 后，进行了完整的对比分析。从分析中提取的关键架构洞察:

1. **双环流制约关系**: "热源分化"导致逃跑环流耦合与进食环流耦合之间存在**制约关系**——两者不能同时最大化。这意味着系统必须学会"选择"——在饥饿/躲避之间进行决策。这是环流比例模型 (C3') 的另一个维度（C3' 只分了稳态/运动/摄食三类）。

3. **皮层体积 + 刚体属性**: p3g 给神经结构赋予了物理体积和刚体属性。用户认同这个方向但觉得"过早"。

## A5. 信息「回荡」预测

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.6.4.md`

**时间**: 2026-06-04

**内容摘要**:

用户在 6.4 对系统的未来行为做出了一个明确的理论预测:

**回声/回荡 (Reverberation)**: 信息在结构和器件不完全、容量有限的环境下被迫在限定的"路径"中反复变形。波长相干 → 叠加干涉 → **凭空产生的噪音**。这种噪音会被本地的结构器件读取，成为 Xin 中不能用原因来解释的"多余"。

**重要背景**: 在提出此预测时（6.4），用户尚未看到 5.29 的 2M 实验结果。而2M 实验恰好发现了 Oto Xin 在 19-147 之间**永久振荡**（2026-05/analysis_2M_growth.2026.5.29.md）——这正是回荡的一种形式。后来在 2026-05/system_narrative.2026.5.31.md 的术语表中将"回荡"正式定义为"Xin 在被限制的结构中振荡不收敛"，并将 2M 实验作为观察证据。

但用户 6.4 的**原始预测**和"做回档还原点"的建议未单独记录，仅散见于 6.4 对话和 5.29 分析。

**当前状态**: 无 DEG 条目，无追踪，无验证/证伪计划。"回荡"的完整预测和内涵讨论被收容到 `system_narrative` 但缺失前后呼应的独立分析。

## A7. 「振荡作为 Xin」理论探讨

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.5.29.md`

**时间**: 2026-05-29

**内容摘要**:

**核心思想**: 系统的结构生长（sprout）具有振荡性质——不是平滑增长而是有节奏的脉冲。用户提出振荡**本身**可能是 Xin 的一种形式：不是残差（P-R），而是系统性节律体现的信息。

**深层挖掘**（用户原话）:
> "振荡作为 xin 是一种思维的深层挖掘，是在闲暇中不断积累成显性的。时间和空间彼此互为 P/R？xin 是信息所表现轨迹的超预期。"

**问题**: 当前振荡幅度太小 (0.01)，被 otolith 的 Xin 值 (25+) 淹没。这不意味着理念无效——可能意味着当前的测量尺度不对。

**时空反演**: 如果时间和空间可被视为彼此的 P/R（即：一个预测另一个），则"xin = 时空不匹配的轨迹"成为一个新定义范式。

**当前状态**: 未在任何 Dashboard/Plan/Analysis 中记录。`矛盾·遗留·生长点` G03 (四大理论桥) 模糊对应但不涉及此理论。

**时间**: 2026-06-09 ~ 06-13

**内容摘要**:

用户反复、持续地追问同一个纪律性关切:

- 6.9: "项目是否违反显式语义内置？"
- 6.10: 平行构建 p3g 中发现了硬编码的"方向选择 → 逃跑"，用户重申语义内置问题
- 6.13: "死代码这一块是否意味着你有在用数理硬编码？这违反纪律性。"

**核心关切**:

系统不应被注入人类对"什么是热源的物理语义"的理解。所有梯度、方向、偏好必须从物理交互中涌现。任何在代码中显式写"if hot → flee"的模式都是**纪律性违规**。

**与现有机制的关系**:

- C3' 环流耦合: 符合纪律——DA 从 ρ 偏离中涌现，不硬编码"进食就是奖励"
- DirectionSelector (空壳): 不明——如果将来填充，它是否会硬编码方向语义？
- SpinalReflexArc: 灰色地带——硬连线撤回反射是基于损伤信号而非"危险"语义，但仍需谨慎

**当前状态**: 用户的核心纪律性关切未在任何正式文档中被列为设计原则或审计标准。

## A9. 神经元规模 ↔ 运动判别精度 ↔ 回荡的数理关系

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.6.5.md`

**时间**: 2026-06-05

**内容摘要**:

用户提出了跨学科的物理直觉:

> "限定数量 20 是否会限制运动判别精度？系统结构中的拟真元件数量与系统容许的最小回荡是正相关的吧？这也许会为系统带来一些新的数理关系。"

**翻译为技术语言**:

1. MAX_MOTORS_PER_AXIS=20 是否会成为运动分辨率的物理瓶颈？
2. 元件数 N 与最小可分辨信号振幅之间存在怎样的数学关系？
3. "回荡"——信息在有限结构中被反射/折射——是否由 N 决定下界？
4. 如果存在这种关系，项目可能出现 N→behavior 的可预测相变

**潜在的理论方向**:

- 与 T·S·I 约束的连接：N_eff（有效元件数）已出现在 T·S·I 不等式中
- 与 Landauer 界的连接：每个元件有最小可分辨信息量 kT ln2
- 与 Shannon-Nyquist 的连接：N 个元件能分辨的最大频率为 N/2

**当前状态**: 完全未被追踪。L07/L08 讨论 Motor 分化但从未涉及规模-精度-回声的数学关系。
