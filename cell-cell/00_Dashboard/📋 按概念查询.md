# 📋 按概念查询

> 在下方 Dataview 块的 `concept` 值中修改为你要查的词条。

---

## 🔍 搜索单个概念

**修改下面 `"Xin"` 为你要查的词条**:

```dataview
TABLE type, date, tags
FROM ""
WHERE contains(concepts, "Xin")
SORT date DESC
```

---

## 📊 所有概念的覆盖文档数

```dataview
TABLE length(rows) as "文档数"
FROM ""
FLATTEN concepts
GROUP BY concepts
SORT rows.length DESC
```

## 🔬 演化链视图（按日期排列的概念演化）

```dataview
TABLE concepts, type
FROM #MOC OR #分析 OR #设计 OR #建模 OR #决策 OR #审计 OR #诊断
```
