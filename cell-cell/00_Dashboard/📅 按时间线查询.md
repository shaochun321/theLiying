# 📅 按时间线查询

> 按日期查看所有文档，支持按类型筛选。

---

## 📆 全部文档按日期排列

```dataview
TABLE type, tags, concepts
FROM ""
WHERE date
SORT date DESC
```

---

## 📊 日期-类型分布矩阵

```dataview
TABLE rows.file.link as "文档"
FROM ""
WHERE date
GROUP BY date
SORT date DESC
```

---

## 🏗️ 按 Plan → Walkthrough → Analysis 时序查看

### 五月
```dataview
TABLE type, concepts
FROM "2026-05" OR "01_工程执行"
WHERE date AND date >= date(2026-05-16) AND date <= date(2026-05-31)
SORT date ASC
```

### 六月
```dataview
TABLE type, concepts
FROM "2026-06" OR "2026-06-continued" OR "01_工程执行"
WHERE date AND date >= date(2026-06-01)
SORT date ASC
```
