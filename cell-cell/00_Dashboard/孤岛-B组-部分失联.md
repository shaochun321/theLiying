## B1. 半中枢互抑 (Half-center mutual inhibition)

**对话来源**: `02_AI对话日志/Hebbian Circuit Structural Optimization.2026.5.17.1.md` (L1385)

**时间**: 2026-05-17

**内容摘要**:

用户指定 CPG（中央模式发生器）用"半中枢互抑"机制实现——这是生物学中 CPG 的标准实现方式（Brown 1911, 半中枢理论）。

**工作原理**:
- 两个神经元互相抑制：神经元 A 发放 → 抑制神经元 B → A 疲劳 → 抑制释放 → B 发放 → 抑制 A
- 节律从互相抑制中涌现——不需要外部时钟
- 频率由疲劳恢复速率 τ_fatigue 决定

**Dashboard 对应**: `核心词条索引.md` 在 §十三（关键动力学组件）中提到了 VitalOscillator（搏动器），但未提及"半中枢互抑"是实现方式。

**丢失细节**: CPG 的精确生物实现路径。当前文档只说"Van der Pol 振荡器"和"三心搏动器"，但没说明**为何选用**这个方案而非半中枢互抑、两方案之间如何选择。对话中明确选了半中枢互抑——但文档中完全没提。

## B3. 前庭层重建与两套构建逻辑冲突

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.5.22.md`

**时间**: 2026-05-22

**内容摘要**:

用户面临一个架构困惑：项目的早期设计中有"前身模型"的 5 层细胞球结构，而当前代码有"赫布超图"的 MetaUnit/MetaPipe 体系。

**两套构建逻辑的冲突**:

| 维度 | 前身模型（旧设计） | 赫布超图（当前） |
|------|-----------------|---------------|
| 结构 | 5 层细胞球 (S→G→C→M→B) | MetaUnit + MetaPipe + MetaBlock + MetaFruit + MetaGate |
| 数学 | HH/AdEx 模型（非线性微分方程） | RC-MOSFET 简化模型 |
| 非线性 | Ca²⁺ 复杂动力 → 尖峰 | MOSFET 阈值门控 (V>V_th) |
| 概念 | "细胞" = 生物隐喻 | "元件" = 电子隐喻 |

**用户的核心困惑**: 是否两套系统都是合理的"底层表述"？如果前身模型本质上也是 RC-MOSFET 的特化形式，是否应该放弃旧设计，全力接受赫布超图？

**对话中的转折**: 用户意识到"两套系统都用了神经元、突触、赫布的通用形式来做联结。在赫布超图中，我可以通过分化出特殊的神经部件、以复用元器件的形式，参照前身模型实现等价结构。"这意味着问题转化为**前身设计如何在赫布超图框架下被"翻译"**。

**Dashboard 对应**: `矛盾·遗留·生长点` 矛盾 #9 (轴数 4/6/7) 触及了架构不一致性但不涉及这次特定的构建逻辑冲突讨论。

**丢失细节**: 构建逻辑A(前身模型) vs 构建逻辑B(赫布超图)的**兼容性评估**、**翻译策略**、以及用户的选择。

## B4. 四维积体电路 + 驻波垫支

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.6.5.md`

**时间**: 2026-06-05

**内容摘要**:

用户提出了一个大胆的架构问题: **"那是否可以让影子层成为真实的四维积体电路？"** 同时阐述了"运动势垫支"理念。

**四维积体电路 (Shadow 4D VLSI)**:

这不是指物理上的第四空间维度。而是指影子层有四个组织维度:
1. **拓扑维度** (T): 突触连接的权重结构
2. **能量维度** (E): 电压+电流+功耗分布
3. **时间维度** (τ): 信号传输延迟 + 处理时间
4. **信息维度** (I): Xin 张力 + 果实状态

这四个维度在"积体电路"中共同决定影子层的行为——影子的"空间"是这四个轴的张量积。

**驻波垫支**（用户理念）:

"时间测度、空间测度、运动势都可以被系统结构垫支。垫支的实现形式就是结构，而垫支的内容由占据结构中信号轨迹的驻波来完成。"

即: 影子层的结构**本身**就是一个预先铺设好的"轨道"，外部信息通过这些轨道时形成驻波（稳定振荡），驻波节点=信息位。

**Dashboard 对应**: D8（影子 4D 容积）在 `当前方向` 和 `矛盾·遗留` 中被列为待决决策，但"四维积体"的精确维度和"驻波垫支"概念未记录。

## B5. 结构计算的定义与偏振/反传播

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.6.5.md`

**时间**: 2026-06-05

**内容摘要**:

用户追问了一个根本性问题: **"你说的结构计算到底是怎样的形式来实现？"**

**核心关切**: 时间/空间/信息是否可以通过"偏振"方式来"读取"？反传播机制是否存在？

**术语澄清**:

| 概念 | 用户的定义 |
|------|----------|
| **结构计算** | 不以计算机的离散逻辑门形式出现，而以**物理连续介质**中的电子/电路行为实现的计算 |
| **偏振读取** | 不仅读信号的"方向"，还要读信号的"倾向"——这信号是倾向于空间端还是时间端？读的是微分 |
| **反传播** | 影子层的信息能否"反向"传播回主线？不是作为反馈输入，而是**修改主线的权重矩阵** |

**偏振读取的电路实现**:

用户设想的方案是：信号先通过定向耦合器（比相器），根据相位差决定输出给"时间端"还是"空间端"。相干的信号特征决定路径，而非信号值本身。

**Dashboard 对应**: `核心词条索引.md` 无"结构计算"条目。`理念架构图.md` 无偏振/反传播的任何参照。

## B6. 时空环流的独立生成机制

**对话来源**: `02_AI对话日志/Refactoring Neural Hypergraph Physics.2026.5.29.md`, `02_AI对话日志/Evolving Morphosphere Cognition Engine.2026.5.18.md`, `02_AI对话日志/Integrating Metabolic Homeostasis Morphosphere.md`

**时间**: 2026-05-18 ~ 05-29

**内容摘要**:

对话中反复讨论一个未进入正式文档的核心问题: **时间和空间是否有自己独立的生成机制，而不只是通过信号衰减/耗散来表征？**

**核心讨论**:

1. **"时空现在被环流和信号的衰减/耗散表征"** — 5.18 承认当前时空概念是**导出量**而非独立生成

2. **"时间环流和空间环流可以从被衰减和耗散表征的被动地位，变成拥有与生体层协调生成的主动地位"** — 5.29 提出时空环流应是**一类独立的生体信号**

3. **"时空耦合靠与时空环流对齐来结构性实现"** — 时空耦合不靠外部标定，靠结构本身与独立时空环流参数对齐

4. **"时空在熵账本上表现不同的偏导？对其中一方切分，另一方接近常数"** — 量纲分析：时间和空间在熵账本中可能有不同的"响应特征"

**实验命题**: 如果给系统一个纯空间信号（不随时间变化）和一个纯时间信号（在空间中均匀分布），看环流比例、Xin 和熵账本的偏导是否不同？可验证"时空有独立生成机制"的假设。

**Dashboard 对应**: `概念演化链` 有"时空环流"条目但谈论的是 ds²/ν 的演化，不是时空本身的**独立生成**。`矛盾·遗留` G25 (ds² 因果锥判定) 和 L22 (B4 ds²) 相关但偏重数学方面。
