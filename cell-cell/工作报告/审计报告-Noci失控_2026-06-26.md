## 三、物理推导：V_ss = -135V

### 3.1 Neuron.step() 适应电流注入机制

```python
# components/neuron.py，每步对 spiking 神经元执行：
self._membrane.inject(-self._w_adapt, dt)  # 1. 注入负电流
# → dV = -w_adapt × dt / C

if voltage > v_peak:
    membrane.discharge_to(v_reset)          # 2. spike + 复位
    w_adapt += b_adapt                       # 3. 适应积分

w_adapt *= exp(-dt / tau_w)                 # 4. 指数衰减
```

### 3.2 初始热瞬态触发爆发

**条件**：body@[75,20,25] 在 t=0 瞬间放置到热源（[75,25,25], T=5.0）附近。

皮肤 left patch 的热瞬态：T_skin 从 ambient≈0.1 升至 equilibrium≈4.5，时间常数 τ_thermo = C×R = 1.0×5.0 = 5.0s = 5000步。

**升温速率（初始阶段）：**
```
dT/step ≈ (T_eq - T_0) × dt / τ_thermo
         = (4.5 - 0.1) × 0.001 / 5.0
         = 8.8 × 10⁻⁴ per step  （初始斜率，稳态前更大）
```

**Noci 输入电流：**
```
noci_total = |dT| × NOCI_DT_GAIN = 8.8×10⁻⁴ × 200 = 0.176 per step
dV_input   = 0.176 × dt / C = 0.176 × 0.001 / 0.02 = 0.0088V/step
```

**放电频率：**
```
从 v=0 开始：V 每步 +0.0088V → 到达 v_peak=0.01 只需 ~2步
→ Noci 每 2 步 spike 一次（频率 500 Hz！）
```

### 3.3 w_adapt 积累动力学

每步 spike（约每 2 步）：
- `w_adapt += 0.02`
- 同时衰减：`w_adapt *= exp(-0.001/5.0) = 0.9998`

**稳态积累（连续爆发期）：**
```
w_adapt_ss = b_adapt / (1 - exp(-dt/tau_w))  × spike_rate_per_step
           = 0.02 / (1 - 0.9998) × 0.5
           = 0.02 / 0.0002 × 0.5
           = 50.0    （爆发高峰期）
```

爆发持续约 1000-3000 步（热瞬态期），之后 dT/dt 趋近 0，noci 输入降低。

### 3.4 爆发结束后 w_adapt 衰减到实测值

**10000步后的 w_adapt（从爆发峰值 50 衰减）：**
```
w_adapt(10k) = 50 × exp(-10000 × 0.001/5.0)
             = 50 × exp(-2.0)
             = 50 × 0.135
             = 6.77
```

（实际比此低，因爆发期不足以维持峰值 50，约 15-20 是更现实的峰值）

**膜电压稳态（常数 w_adapt 驱动）：**

达到稳态 dV=0，即注入 = 泄漏：
```
-w_adapt × dt/C + V × (exp(-dt/RC) - 1) = 0
V_ss ≈ -w_adapt × R_leak   （当 dt/RC << 1 时成立）
     = -2.7 × 50
     = -135V
```

**实测验证：**
```
noci_front V = -134.33V  ← 与推导 V_ss = -135V 完全吻合
```

## 四、参数设计错误的生物推导溯源

### 4.1 v_peak 问题

| 参数 | 当前值 | 生物依据 | 归一化换算 |
|------|--------|---------|----------|
| v_peak | 0.01 | — | 对应 V_bio = -80 + 0.01×130 = -78.7mV（接近 E_K，无生物意义）|
| 正确值（TRPV1 spike threshold） | 0.25 | C-fiber 动作电位阈值 V_th ≈ -47mV | (-47+80)/130 = 0.254 |
| 正确值（保守取整） | 0.20 | 对应 V_bio = -54mV（L3/L4 伤害感受器） | REF: Julius & Basbaum 2001 |

**当前 v_peak=0.01 比正确值低 20-25 倍。** 这意味着任何微小输入都会触发放电，包括：
- 积分电路的数值噪声
- 热瞬态的最初几步
- 适应电流消退时的微小过冲

### 4.2 b_adapt 问题

**约束条件**：适应电流不应把膜电压推到 v_peak 以下的极端负值，否则神经元无法从爆发恢复。

**安全上界推导：**
```
最大允许 w_adapt = v_peak / R_leak
                 = 0.25 / 50 = 0.005

每次 spike 后 b_adapt 不应超过（以防一次 spike 就超过此上界）：
b_adapt_max ≈ 0.005 × (1 - exp(-dt/tau_w)) ≈ 0.005 × 0.0002 = 0.000001

这太小了。合理设计应是：在稳定放电率（如 50Hz）下，w_adapt_ss < v_peak/R_leak
```

**重新设计 b_adapt（目标：50Hz 放电时 w_adapt_ss = 0.1×v_peak/R_leak = 0.0005）：**
```
spike_rate = 50Hz = 50/s = 0.05/步 @ dt=0.001
tau_w_steps = 0.5/0.001 = 500步
w_adapt_ss = b_adapt × spike_rate × tau_w_steps  （近似）
0.0005 = b_adapt × 0.05 × 500
b_adapt = 0.0005 / 25 = 0.00002
```

**合理取值：b_adapt ≈ 0.001**（比当前 0.02 小 20×）

### 4.3 tau_w 问题

当前 tau_w=5.0（5s）远超 C-fiber 的 AHP 时间常数。

**BIO**: C-fiber 超极化后适应电流 τ ≈ 200-500ms（TRPV1，Julius & Basbaum 2001；Caterina et al. 1997）

```
tau_w = 0.5s = 500步 @ dt=0.001  ← 生物合理范围
```

## 五、参数修正表

| 参数 | 当前值 | 修正值 | 推导依据 |
|------|--------|--------|---------|
| v_peak | 0.01 | **0.25** | TRPV1 spike threshold: V_th≈-47mV → norm=0.254 （Julius 2001）|
| v_reset | 0.001 | **0.02** | 10% × v_peak（正常 reset gap）|
| b_adapt | 0.02 | **0.001** | V_ss_adapt = b_adapt × R × τ/dt × f_spike < v_peak |
| tau_w | 5.0 | **0.5** | C-fiber AHP τ ≈ 200-500ms（Caterina 1997）|

### 修正后验证

**稳定放电率 50Hz @ dt=0.001 下的 w_adapt_ss：**
```
w_adapt_ss = 0.001 × (50×0.001) × (500) = 0.001 × 0.05 × 500 = 0.025
V_ss = -0.025 × 50 = -1.25V   ← 比当前 -135V 好 100×，但仍偏大
```

进一步收紧：b_adapt = 0.0002
```
w_adapt_ss = 0.0002 × 0.05 × 500 = 0.005
V_ss = -0.005 × 50 = -0.25V   ← 在归一化方案范围内
```

**推荐最终值：b_adapt = 0.0002，tau_w = 0.5**

### 修正后的放电行为模拟

以 v_peak=0.25，dT/step=8.8e-4，NOCI_DT_GAIN=200：
```
noci_input/step = 8.8e-4 × 200 × 0.001 / 0.02 = 0.0088V/step
到达 v_peak=0.25 需要 0.25/0.0088 ≈ 28步
放电频率 ≈ 35Hz（生物合理范围：5-100Hz for C-fibers）
```
