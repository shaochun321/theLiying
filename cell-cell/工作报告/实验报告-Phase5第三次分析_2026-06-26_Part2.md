## 七、DR5 严重退步分析

Round 3 align=0.240（Round 2: 0.460），**热源对齐率减半**。

### 原因

1. **c_L = c_R = 0 全程**：y 轴热觉完全静默，没有 y 方向学习
2. **w_right ≈ w_left（0.046 vs 0.044）**：y 轴权重差异仅 0.002，几乎无方向偏置
3. **体运动主要由前庭系统驱动**：没有热觉方向性学习，运动趋向随机
4. **y 轴热梯度微弱**：ΔT(left-right)=+0.43°C，在 gain=10 下 enc→col 依然不足以激活 col_therm_left

Round 2（gain=4）的 align=0.460 主要来自 x 轴学习（w_back 强 LTP），驱动 -x 运动。该 x 方向偶尔与热源方向一致，contribute align>0。Round 3 连 x 方向学习也几乎消失（w_back=0.015），所以 align=0.240（接近随机游走 0.25）。

### 参数变更的意外后果

```
预期（改动 lambda: 0.0015→0.0008）：
  目的：延长 yolk 寿命至 625k 步
  实际：yolk 确实延长（500k 步仍剩 20%），但 fill 提前 80k 步耗尽
  结论：lambda 降低 ≠ fill 延长；二者通过不同机制影响存活

预期（改动 gain: 4.0→10.0）：
  目的：让 col_therm 更容易激活
  实际：gain=10 在 dt=0.001 实验中只使 c_F/c_B 轻微改善（+30%）
  代价：更强电机输出 → 能耗 +32% → fill 更快耗尽
  结论：gain 收益 < 能耗代价；净效果为负
```

### 变更 1：修复 DR3（fill 全程 > 0）

**推导**：

```
fill 消耗速率估算（Round 2）：
  fill @ 10k = 0.504, fill @ 330k = 0.000
  net_drain_rate = 0.504 / 320k = 1.57e-6/步（净值）
  consumption = lambda + 1.57e-6 = 0.0015 + 1.57e-6 ≈ 0.001502/步

要使 fill 不耗尽（到 500k 步）：
  需要 lambda ≥ consumption = 0.001502
  但还需要 yolk 撑到 500k 步：yolk_level ≥ lambda × 500k = 0.001502 × 500k = 751

方案：
  lambda_yolk = 0.002（> 0.001502，fill 净增）
  yolk_level  = 1000（1000 / 0.002 = 500k 步，正好）

预期 fill 行为：
  净增率 = 0.002 - 0.001502 = 4.98e-4/步
  fill @ 500k = 0.5 + 4.98e-4 × 500k = 0.5 + 249 → 1.0（满格）
  → fill 全程 ≥ 0.5，energy_plasticity_scale = 1.0 全程 ✓
```

**操作**（实验脚本）：
```python
LAMBDA_YOLK_P5 = 0.002       # > 消耗速率 → fill 净增
YOLK_LEVEL_P5  = 1000.0      # 1000 / 0.002 = 500k 步寿命

c.yolk_sac.config.lambda_yolk = LAMBDA_YOLK_P5
c.yolk_sac.level = YOLK_LEVEL_P5
```

### 变更 3（可选）：扩大 y 轴热梯度

c_L = c_R = 0 的根本原因：ΔT(left-right) = 0.43°C 对应的 enc_left - enc_right 差异太小。

即使 gain=4.0，y 轴 col 也不会激活（Round 2 也是如此）。

**选项 A**：接受 y 轴静默——只要 x 轴学习（back/front）足够强，DR1/DR4 可以通过（Round 2 证明了这一点）。

**选项 B（更改 BODY_INIT_POS）**：将起点改为 y 轴更不对称的位置，使 ΔT(left-right) 更大。
- 现在 y=20（比 y=25 热源近 5 单位），ΔT≈0.43
- 如果改为 y=50（正中间），ΔT→接近 0（更糟）
- 如果改为 y=10（离热源 15 单位），ΔT≈0.35（更小）
- **注意**：增大 y 梯度需要让左右热源温度差更大，而非改变位置

**选项 C**：将 LATERAL_GAIN 从 0.30 提高，放大左右差异信号。
- 但这违反"不修改母代码"原则（LATERAL_GAIN 在 somatosensory chain 中定义）

**建议**：Round 4 先用选项 A，不改 y 轴。Round 2 的 align=0.460（接近 0.50）表明仅靠 x 轴学习可能已接近 DR5 阈值。延长可塑性窗口到 500k 步有望从 46% → 50%。

## 十二、总结

Round 3 失败的根本原因是**两项参数修改相互叠加产生了负效果**：

1. **lambda_yolk 降低**：减少每步能量供给 → fill 更快耗尽 → 可塑性提前 80k 步冻结
2. **enc→col gain 提高**：增加电机能耗 → fill 更快耗尽；col 激活提升有限（自适应耦合器限制）

Round 4 的核心策略：
- **回退 gain** 到 Round 2 的有效值（4.0）
- **正确修复 DR3**：同时提高 lambda 和 yolk_level，确保 fill 全程 > 0（可塑性满格）
