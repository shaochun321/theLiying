# 审计报告：热觉链路失效深度分析

**报告日期**: 2026-06-26  
**链路**: Soma patches → enc_therm → col_therm → Motor therm bundles  
**测试条件**: 10k steps, dt=0.001, body@[75,20,25], heat source@[75,25,25]  
**结论**: 热觉链路存在三层级联失效，当前状态下**无法产生方向性 STDP 学习**

---

## 一、链路拓扑与信号流

```
热源 [75,25,25] T=5.0
    ↓ world.temperature_at(patch_world_pos)
皮肤温度 patch_temps (T_skin, dT_skin, damage)
    ├── T_skin → Thermoreceptor (thermo_*) → [thermo_to_relay] → Relay (relay_*)
    └── dT_skin → Nociceptor (noci_*) → [noci_to_relay] → Relay (relay_*)
                                              ↓
Relay 适应滤波 get_mechanical_inputs()  → high-pass: raw - adapt_baseline
    ↓ (therm_front/back/left/right 作为 extra_axes)
L4 Encoding: reg_therm_* (tonic) + irr_therm_* (phasic)
    ↓ enc_to_col_therm_*
L5 Column: col_therm_*
    ↓ therm_*_to_move_* bundles
L6 Motor: move_x, move_y, move_z
```

---

## 二、各层实测激活值

### 皮肤温度（ΔT 梯度确认）

```
front:  T = 4.2394
back:   T = 4.2394
left:   T = 4.4817  ← 最近热源（距 5 units）
right:  T = 4.0516
ΔT(left-right) = +0.4301  [OK] ← 方向性热梯度存在
```

热梯度是真实存在的（left 比 right 热 0.43 单位），问题在于**下游神经元无法传递这个差异**。

### Somatosensory 第一层（Thermo / Noci / Relay）

```
soma_thermo: avg = 10.0000   ← 饱和（设计上限 10.0 = 10× v_peak）
soma_relay:  avg =  7.2585   ← 饱和（输出范围应 [0,1]，实测 7.26）
soma_noci:   avg =  0.0000   ← 完全死链路（V < -134V）
```

**关键失效点 A：Noci 全死，Relay 饱和**

- Thermo 饱和（T_skin=4.2～4.5 >> 正常归一化范围），thermo→relay bundle 把过大信号传至 Relay
- Relay 输出 7.26（远超 [0,1]），驱动 enc_therm 超饱和
- Noci（dT/dt 传感器）被爆发 spike 的适应电流打到 -134V，完全停止放电

### L4 编码层（Encoding）

```
reg_therm_left:  act_ema = 1.2293  ← 超出归一化上界 1.0
reg_therm_right: act_ema = 1.1084
reg_therm_front: act_ema = 1.1556
reg_therm_back:  act_ema = 1.2257

irr_therm_left:  act_ema = 0.0000  ← 完全死链路
irr_therm_right: act_ema = 0.0000
irr_therm_front: act_ema = 0.0000
irr_therm_back:  act_ema = 0.0000
```

**关键失效点 B：reg_therm 超饱和、irr_therm 全死**

- `reg_therm_*`：超过 1.0 → 已超出 MOSFET 激活函数的有效范围，输出被钳位，但 EMA 仍记录超出值
- `irr_therm_*`（对应 noci 输出 / phasic 通道）：依赖 noci 放电，noci 已死 → 完全无信号
- 四个 patch 的 `reg_therm` 激活差异：
  ```
  left=1.2293, right=1.1084, front=1.1556, back=1.2257
  最大差异 Δ(left-right) = 0.1209
  ```
  差异存在，但所有值均超饱和 → MOSFET 输出已压缩到同一高值，**差异被消除**

### L5 列皮层（Column）

```
col_therm_left:  act=0.0202, V=0.0460
col_therm_right: act=0.0128, V=0.0917
col_therm_front: act=0.0212, V=0.0332
col_therm_back:  act=0.0186, V=0.0541
```

**关键失效点 C：Column 阈下，激活极弱**

- v_peak=0.10（D-03 量纲问题）
- col 接受的 enc 输入虽然饱和（1.1-1.2），但 enc→col bundle 的耦合经过 TemporalCoupler（参见 hebbian.py ~524行：`enc_reg.step(tonic_val * 5.0, dt)` 调用传入后被 MOSFET 钳位）
- 实际到达 col 膜的电压只有 0.03-0.09V，正好在 v_peak=0.10 附近
- 激活 EMA 在 0.013-0.021 之间（对于 STDP 来说信号极弱）

注意：left(0.0460V) vs right(0.0917V)的电压差虽然存在，但两者均处于阈下弱激活区，**无法产生有意义的 STDP 梯度**。

### Motor therm bundles（最终输出）

```
therm_therm_left_to_move_y:  w_mean=0.50000, w_max=0.50000
therm_therm_right_to_move_y: w_mean=0.50000, w_max=0.50000
therm_therm_front_to_move_x: w_mean=0.49999, w_max=0.49999
therm_therm_back_to_move_x:  w_mean=0.49979, w_max=0.49979

|Δw(left-right)| = 0.00000
```

**最终诊断：热趋性方向学习完全停滞**

所有 therm bundle 权重仍在初始值附近（0.5），零方向分化。

## 三、Xin 张力分布（STDP 驱动力诊断）

```
therm bundle Xin（从 Soma 到 Motor）：
  thermo_to_relay_left:   Xin = 9.22   ← Soma 层有 Xin（thermo 在运作）
  thermo_to_relay_right:  Xin = 8.81
  enc_to_col_therm_left:  Xin = 0.26   ← enc→col 有少量 Xin
  enc_to_col_therm_right: Xin = 0.21
  therm_left_to_move_y:   Xin = 0.010  ← col→motor 几乎无 Xin
  therm_right_to_move_y:  Xin = 0.008
```

STDP 权重更新量 ∝ Xin 张力。在 col→motor 层，Xin 仅 0.008-0.010，远低于正常前庭通路（Xin ≈ 350+）。这解释了为什么 10k 步后权重几乎未变。

**Xin 衰减路径：**
```
Soma 层（Xin≈9.0）
    → 经过超饱和的 enc（信号压缩）
    → enc→col（Xin≈0.25，下降 36×）
    → col 阈下激活（act≈0.017）
    → col→motor（Xin≈0.009，再下降 28×）
总衰减：9.0 → 0.009，约 1000×
```

## 四、失效分层诊断

| 层 | 状态 | 根因 | 对学习的影响 |
|----|------|------|------------|
| Thermo (L1) | 饱和（avg=10.0） | 皮肤温度无归一化（T=4.2，但 neuron 输入未缩放） | 传入 relay 信号过大 |
| Noci (L1) | 死链路（V=-134V） | v_peak=0.01 爆发 + b_adapt 积累 | irr_therm 全死 |
| Relay (L2) | 饱和（avg=7.26） | 接收过大的 thermo 信号 | 输出超出 [0,1]，差异压缩 |
| reg_therm Enc (L3) | 超饱和（avg>1.0） | Relay 输出 × synapse_gain 过大 | 空间差异被 MOSFET 钳位 |
| irr_therm Enc (L3) | 全死（avg=0.0） | noci 死 + 静止身体 dT≈0 | phasic 通道完全无信号 |
| col_therm Col (L4) | 阈下弱激活（avg=0.017） | v_peak=0.10（D-03） + enc 饱和 | 接近但未稳定超过阈值 |
| Motor therm bundles | 无变化（|Δw|=0） | Xin 衰减 1000×，col 信号太弱 | 方向学习完全停滞 |

## 五、必要修复（按依赖顺序）

**修复必须按序进行**：Noci → Relay 增益 → enc 量纲 → col v_peak，否则后续修复的效果被前序问题掩盖。

### Step 1（P0）：Noci 参数重推（见审计报告-Noci失控）
```python
v_peak  = 0.25   # TRPV1 C-fiber 激活阈 (T=43°C → norm≈0.31)
b_adapt = 0.001  # V_ss_adapt = b_adapt × R / (1-exp(-dt/tau_w)) < v_peak
tau_w   = 0.5    # C-fiber 适应 τ ≈ 300-500ms
```
预期效果：noci 停止爆发积累，V 保持 [-0.3, 0.25] 范围，irr_therm 恢复

### Step 2（P0）：Relay 输入增益校准
```
当前：thermo(T=4.2) → relay 输出 7.26
目标：relay 输出应在 [0, 1]

需要查明 thermo→relay 链路总增益 = synapse_gain × weight × MOSFET 增益
thermo_to_relay bundle: synapse_gain=3.0, initial_weight=0.3 → 理论增益 3.0×0.3=0.9×
但 thermo 本身 avg=10.0（thermo 的输出已超量程）

实际问题：thermoreceptor 接受 T_skin=4.2 作为输入电流
   V_ss_thermo = T_skin × R_leak = 4.2 × 5.0 = 21V >> v_peak（未知，需查）
```
需要追踪 thermoreceptor 神经元配置中 v_peak 的设置。

### Step 3（P1-B）：Column v_peak=0.10 → 0.20
归一化方案推导见审计报告 B（D-03 条目），不重复。

### Step 4（验证）：修复后，irr_therm 需在静止热源下有信号
注：irr_therm 依赖 dT/dt。静止身体 dT≈0，即使 noci 修复后也可能保持极低。需要在有身体运动的条件下验证。这不是修复失败，而是**设计正确性**：phasic 通道只在运动时响应，用于检测"靠近/远离"热源。

---

## 六、与热趋性涌现的关系

Phase 4 实验的验收标准 DR1（|Δw|>0.02）在当前架构下**不可能达成**，原因：

1. `irr_therm`（phasic）全死 → 无法检测接近/远离热源的 dT/dt 变化
2. `reg_therm`（tonic）超饱和 → 空间差异被压缩，col 无法区分 left 热 vs right 热
3. `col_therm` 阈下激活 → Xin 接近零 → STDP 无驱动力

即使给足 500k 步，当前参数下热觉权重 |Δw| 也不会超过 0.005。这不是实验时间不足的问题，而是**信号通路从根本上无法传递方向差异**。
