## 六、DR1/DR4 失败分析

Round 3 中 w_back 仅增长到 0.125（Round 2: 0.495）。为什么 gain=10 反而导致更弱的 LTP？

### 原因 1：fill 更早冻结

可塑性窗口对比：
- Round 2：fill > 0 从 0 到 330k 步（330k 步有效学习时间）
- Round 3：fill > 0 从 0 到 250k 步（250k 步有效学习时间，少 80k 步）

当 fill=0 时，`energy_plasticity_scale=0`，所有权重变化被乘以 0 → 冻结。

### 原因 2：gain=10 增加能耗，进一步压缩学习窗口

```
能耗分析（近似）：
  Round 2（gain=4）：fill 从 0.5 → 0 的速率 ≈ 1.57e-6/步
  Round 3（gain=10）：fill 从 0.5 → 0 的速率 ≈ 2.08e-6/步 （更快！）

推断：gain=10 → col 激活更强 → 电机驱动更强 → 能耗更高
→ fill 速率加快 32%，学习窗口缩短 80k 步
```

### 原因 3：自适应耦合器（coupler_adapt）抑制了额外增益

从回归测试（dt=1.0）数据：
- T3.2：therm_col activation = 0.5015（远高于 Round 2 的≈0.17）

但在实际实验（dt=0.001）中，c_B ≈ 0.003~0.028——比预期低 10-100 倍。

**原因**：`coupler_adapt_vth=0.2（目标激活率 20%）, coupler_adapt_gm=2.0（反馈增益 2.0）`。

当 col 激活超过 20% duty cycle，自适应耦合器增加泄漏电阻，抑制输出。这在 dt=1.0 下（每步=1秒）无明显效果，但在 dt=0.001 下（每步=1ms），coupler 累积慢、抑制弱，但仍会在持续热刺激时介入。

**实质**：gain 从 4→10 并未使 col 激活提高 2.5×，因为自适应耦合器通过负反馈限制了增益。
