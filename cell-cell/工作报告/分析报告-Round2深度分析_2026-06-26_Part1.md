## 二、倒车策略（逆向趋热）分析

### 2.1 几何重建

Round 2 基础配置：
- 初始位置：[75, 20, 25]，面朝 +x（默认方向）
- 最近热源：src[4] = [75, 25, 25]，T=5.0，初始距离=5.0
- 皮肤贴片定义（推断）：
  - front patch：body + [+3, 0, 0] = [78, 20, 25]
  - back patch：body + [−3, 0, 0] = [72, 20, 25]
  - left patch：body + [0, +3, 0] = [75, 23, 25]（+y 方向）
  - right patch：body + [0, −3, 0] = [75, 17, 25]（−y 方向）

**初始热场验证**：
```
front: dist([78,20,25], [75,25,25]) = √(9+25+0) = 5.83
back:  dist([72,20,25], [75,25,25]) = √(9+25+0) = 5.83  ← 对称！
left:  dist([75,23,25], [75,25,25]) = √(0+4+0)  = 2.0
right: dist([75,17,25], [75,25,25]) = √(0+64+0) = 8.0

实验日志：front=4.239, back=4.239, left=4.482, right=4.052 ✓
```
初始状态：front=back（对称），left > right（src[4] 在 +y 方向）。

### 2.2 100k 步时的热场不对称

身体位置 [84.7, 29.3, 32.2]，此时距初始热源 src[4]=[75,25,25] 的距离：

```
front at [87.7, 29.3, 32.2]: dist = √(161.3+18.5+51.8) = √231.6 ≈ 15.2
back  at [81.7, 29.3, 32.2]: dist = √(44.9+18.5+51.8)  = √115.2 ≈ 10.7  ← 更近！

结论：T_back > T_front（body 在 x=75 热源右侧，back 朝向热源）
```

**触发倒车的物理条件**：当 body x > 热源 x（body 位于热源右侧），back patch 更靠近热源 → T_back > T_front → therm_back 激活 → move_x 驱动（gain=−10）→ -x 运动 → body 向热源靠近。

这就是**倒车趋热机制**的物理基础。

### 2.3 倒车策略的自洽性验证

| 阶段 | 体位置 x | T_back vs T_front | therm_back 状态 | 驱动 |
|------|---------|------------------|----------------|------|
| 0~100k | 75→84.7 | T_back > T_front（右侧热源） | 激活 | -x 运动 |
| 100k~150k | 84.7→≈78 | T_back > T_front（仍在右侧）| 强激活 | -x 运动（加速） |
| 150k（diverge_step）| ≈78 | 权重差超过0.02阈值 | 已学到 | DR4 通过 |
| 150k~200k | ≈78→2.4 | 越过 x=75，T_front 有时更大 | 偶发 | -x 运动惯性 |
| 200k~300k | 2.4→20.1 | 体在 x=2~20，多热源交叠 | 间歇 | 部分 LTP |
| 330k+ | fill→0 | — | 冻结 | 无新学习 |

therm_back 束在 0~200k 步的学习阶段正处于 **T_back > T_front** 的物理条件下，即体在 x=75 热源右侧时学得最快。

### 2.4 therm_front 为何衰减

**反向逻辑**：
- therm_front 激活条件：T_front > T_back（body 在热源左侧，front 朝向热源）
- 学习期间（0~330k），body 大多时候在热源右侧或远离热源
- therm_front 激活频率低 → LTP 弱 → 衰减（decay_rate=0.5 持续负向压力）

**实验证据**：
```
w_front 轨迹：0.102 → 0.075 (100k) → 0.051 (200k) → 0.025 (300k) → 0.022 (500k)
（纯衰减，无明显增长）

w_back 轨迹：0.102 → 0.076 (100k) → 0.335 (200k) → 0.495 (300k) → 0.483 (500k)
（先小幅衰减，100k后急速增长至 weight_max）
```

### 2.5 倒车策略的生物类比

这种"感知背侧温度，向热源倒退"的策略在生物学中有对应：

> **BIO**: 果蝇幼虫（Drosophila larva）通过背侧/腹侧温度梯度实现趋热或避热行为（Klein et al., 2015, Cell）。幼虫并非总是"朝向热源前进"，而是通过调整曲率使背侧热感受器接近目标温度。

**结论**：therm_back 强于 therm_front 是**物理涌现的合理策略**，不是错误。该策略对应的行为是"倒车向热源"——当热源在身体正后方时，倒退是唯一能立即接近热源的运动方向（无转向机制）。

## 三、DR5=46% 的真实含义

### 3.1 alignment 指标定义

```python
def thermal_alignment(pos_now, pos_prev, src):
    v = pos_now - pos_prev          # 速度向量
    g = src.position - pos_now      # 指向热源向量
    return dot(v_normalized, g_normalized)  # 余弦相似度
```

alignment > 0：运动方向与热源方向夹角 < 90°（趋热）
alignment < 0：运动方向与热源方向夹角 > 90°（避热）

### 3.2 倒车策略下的 alignment 预期

| 情形 | 速度方向 | src 方向 | alignment |
|------|---------|---------|-----------|
| body 在 x=80，倒退向 x=75 热源 | (-1, 0, 0) | (-1, 0, 0) | **+1.0** |
| body 在 x=60（两组热源之间），向 -x 运动，但最近热源在 +y | (-1, 0, 0) | (0, +1, 0) | **0.0** |
| body 在 x=5，向 +x 运动（靠近 x=25 热源），但学到的是倒退 | (+1, 0, 0) | (+1, 0, 0) | **+1.0** |
| body 在 x=5，向 -x 运动远离 x=25 热源 | (-1, 0, 0) | (+1, 0, 0) | **−1.0** |

### 3.3 为什么 align=46% 不是"学习失败"

从轨迹分析：

**Round 2 三阶段行为**：

| 阶段 | 体位置（x范围） | 主要运动 | 最近热源 | alignment 偏向 |
|------|--------------|---------|---------|--------------|
| 0~150k | x: 75→85（局部探索）| 随机 + 少量倒退 | src[4]=[75,25,25] | 约 0 |
| 150k~200k | x: 85→2（大幅 -x）| **倒退加速** | src[4] 渐远，src[0] 渐近 | 正（倒退时 align>0） |
| 200k~330k | x: 2→20（在 x=25 附近游走）| 混合 | src[0]/src[1] | 约 0（多方向游走） |
| 330k+ | 随机游走（fill=0，无新学习） | 无方向性 | 变化 | 约 0 |

**计算**：50万步中，约 15 万步（100k~200k）有明显倒退，alignment≈+0.6；其余 35 万步 alignment 接近随机（约 0.5）。
估算整体 align_pos_frac ≈ 0.15×0.8 + 0.35×0.5 = 0.12 + 0.175 = 0.295... 

实测 0.460 比预期高，说明倒退策略的持续时间比我估计的更长——实际上从 step 70k~90k 就开始出现显著正 align（DA = 0.753→0.761 上升）。

### 3.4 DR5 阈值 50% 是否合适

DR5 = 50% 实际上等价于"有策略的随机游走"（完全随机 = 50%）。align=46% 意味着**略微偏负**，可能是以下之一：

**A. 学习到了"倒车"但基准是正向运动**：倒退（-x）在热源在+x右侧时是对的，但热源在-x左侧时就反了。多热源环境导致 50% 左右波动。

**B. 330k后的随机游走拉低了整体**：fill=0 后无方向性运动持续 170k 步，相当于 34% 的时间内 alignment≈随机（0.5）。仅从 0~330k 的 alignment 计算：前 330k 可能 align>0.5，而后 170k 随机游走将整体拉到 0.46。

**C. DR5 阈值定义问题**："50% 时间朝向热源"对于没有转向能力的生物来说过于严格。更好的指标可能是**累积位移与热源方向的相关性**。

**结论**：align=46% ≈ 50% 偏差仅 4pp，不能视为"学习失败"，更准确的描述是"弱方向性学习 + 能量耗尽后随机游走"。
