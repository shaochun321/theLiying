# nexus_v1 构建日志 — 补偿机制缺失分析

> **日期**: 2026-05-22
> **阶段**: 信号全通 6/6 后的稳定性分析
> **前置**: 信号深度从 1/6 提升到 6/6（见 walkthrough.md）

---

## 现象

信号深度在 6/6 和 4/6 之间振荡：

```
t=2000: [>>>>>>] 6/6
t=2500: [>>>>>>] 6/6
t=3500: [>>>>..] 4/6  ← Column 崩塌
t=4000: [>>>>>>] 6/6
t=4500: [>>>>..] 4/6  ← 再次崩塌
```

## 诊断数据

### Column 电压轨迹

```
t=2800  col_v=+0.890  chain: →Col→Mot   ← 全通
t=2900  col_v=+0.001  chain: ×          ← 崩塌！
t=3000  col_v=+5.911  chain: →Col→Mot   ← 爆发
t=3100  col_v=+4.577  chain: →Col→Mot   ← 衰减
t=3200  col_v=+0.003  chain: ×          ← 再次崩塌
```

模式：col_v 在 0 ↔ ~6 之间跳变，不是平稳信号。

### 能量耗竭

```
所有神经元 energy = 0.001（初始 1.0，几乎归零）
metabolic_recovery = 0.001/step
heat_output(col) = 1.7 ~ 95/step （耗散/恢复 ≈ 1000~95000:1）
```

## 根因分析

### 1. 代谢补偿不足

```
每步恢复:  +0.001
每步耗散:  -95.0 (Column I²R)
缺口:      ×95000
```

生物对标：线粒体 ATP 产出速率与活动匹配。

### 2. 无稳态可塑性

无 BCM 规则、无突触缩放、无内在兴奋性调节。
结果：正反馈 → 爆发 → 耗尽 → 沉默 → 缓慢恢复 → 爆发...

### 3. 无自发活动

前庭传入无张力放电（生物 ~70 Hz 自发）。
下游在脉冲间隔完全寂静。

## 结论

信号深度波动不是参数问题，是缺少闭环补偿机制：
```
输入能量 → 电活动 → I²R 耗散
    ↑                      ↓
  [缺失] ← [缺失] ← 代谢底物
```
