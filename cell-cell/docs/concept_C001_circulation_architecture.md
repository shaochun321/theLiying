# 环流架构与赫布超图 — 概念文档 C-001

> 日期: 2026-05-23
> 状态: CONFIRMED（本轮对话确认的架构认知）
> 关联: modeling_016, modeling_017

## 核心认知

### C-001.1 赫布超图是底座

赫布超图 = 节点（Neuron）+ 超边（SynapticBundle + Memristor）

所有功能区域都是超图的子图：
- 交界层 = enc_, col_, mot_ 节点 + 连接超边
- 影子层 = s_enc_, s_col_, s_mot_ 节点 + 连接超边
- 绑定层 = bind_ 节点 + 连接超边
- ECM/PNN = 超图的介质属性（不是节点）

### C-001.2 环流 = 结构的动态行为

环流不是独立系统，不会"出现"或"消失"。
环流 = 信号在超图中沿超边流动 = step() 中 propagate() 的自然结果。
只要结构存在，环流就持续运行。

CirculationMeter (原 CirculationDetector, 本轮改名) 只是测量仪表，
不是环流的来源。

**最新测量 (2026-05-24, 20k步):**
- 活跃路径: 60 条
- P 路径: yaw→bind_yaw_pitch→move_x (flow=0.000396)
- P 频率: 0.39
- μ_total: 0.0205
- flow 方差: 7.7e-6 (非常稳定)
- MIN_FLOW: 1e-5

### C-001.3 环流角色分化

不同环流路径需要不同物理特性的承载结构：

| 角色 | 生物对应 | 分化参数 |
|------|---------|---------|
| 前馈(feedforward) | 有髓鞘投射轴突 | 无延迟, 高增益 |
| 反馈(feedback) | 丘脑中继 | 有延迟, 有门控 |
| 跨轴(cross_axis) | 联络纤维 | 有延迟, 低增益 |
| 影子(shadow) | 默认网络内部 | 高延迟, 低增益 |

不需要新元件——同一种 SynapticBundle 用不同参数。

### C-001.4 小环流耦合与合并

- 耦合 = 共享节点（如 yaw 和 oto_x 共享 mot_x）
- 合并 = 绑定层将多轴信号融合注入共享节点
- 主环流 = 多条小环流通过绑定层合并后的整体运动环路
- 对应：基底节-丘脑-运动皮层回路的整合功能

### C-001.5 环流持续性

真实神经系统中三种并行持续循环：

| 循环 | 频率 | 项目对应 |
|------|------|---------|
| 丘脑-皮层 | 40Hz | enc→col→mot→feedback (每步) |
| 基底节回路 | 2-5Hz | 绑定层合并环流 (每步) |
| 默认模式网络 | 0.01Hz | 影子层 (每10步) |
