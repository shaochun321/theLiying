# 四条假设的文献判别

---

## 假设 1：信号补偿机制 — ✅ 完全正确

> "神经细胞对信号有补偿机制，信号会在链路中衰减，要么在链路中补偿，要么在神经细胞内动力补偿，而且神经脉冲的制式应该很简单"

### 文献验证

这精确描述了 **saltatory conduction（跳跃式传导）** 和 **all-or-none principle（全或无原理）**。

**制式极其简单**：动作电位只有一种"形状" — 固定幅度（~100mV），固定持续时间（~1ms）。信息不在脉冲形状中编码，而在**脉冲的时间间隔和模式**中编码。这是1952年 Hodgkin-Huxley 模型确立的基本事实。

**链路中补偿**：髓鞘（myelin）包裹轴突，在 **Ranvier 节（Nodes of Ranvier）** 处暴露的离子通道**再生**信号到原始幅度。信号在节间段被动衰减，在每个节点被主动再生。

```
[节点1: 再生] ──被动衰减──> [节点2: 再生] ──被动衰减──> [节点3: 再生]
  Na⁺/K⁺通道打开           电缆方程传导            Na⁺/K⁺通道打开
  恢复到 ~100mV           衰减到 ~30mV             恢复到 ~100mV
```

**细胞内动力补偿**：突触后电位（PSP）是**分级的**（不是全或无），在树突中被动衰减。但 **树突棘（dendritic spike）** 提供了非线性放大：当局部输入超过阈值时，NMDA 受体的电压门控解除 Mg²⁺ 阻塞，产生再生性钙内流，在树突分支上重新放大信号。

### 与我们系统的对应

| 真实机制 | 我们的代理 | 代码位置 |
|----------|-----------|---------|
| Ranvier 节再生 | `bundle_inertia` 恢复 (myelination proxy) | maintain() L1349-1361 |
| 全或无脉冲 | `threshold` 门控 — |act| > threshold 才传导 | propagate() |
| 树突棘非线性 | 尚缺 — 这是下一步应该添加的 | — |
| 突触后电位衰减 | `last_pre/post_activation *= 0.9` | maintain() L1370-1374 |

> [!IMPORTANT]
> **你的假设完全正确**，且我们系统中已经有了链路补偿（bundle_inertia 恢复 = 髓鞘化代理）和简单制式（threshold 门控 = 全或无代理）。**但缺少树突棘的非线性再生** — 这是一个真实的结构性缺口。

## 假设 2：Column 构造信息链路的"势阱" — ✅ 有成熟理论支持

> "Column 之类的机制会构造链路最佳路线的'法则'（势陷？），就像引力一样"

### 文献验证

这精确对应了 **attractor dynamics（吸引子动力学）** 中的 **computational energy landscape** 理论。

**核心文献**：
- Hopfield (1982): 证明递归神经网络可以用能量函数描述，网络状态向能量最低点（吸引子）演化
- Amit, Brunel, Fusi (1997): 证明皮层微柱的递归连接产生吸引子状态，维持工作记忆
- Rolls & Deco (2010): 系统性论述 attractor network 如何在皮层柱中实现决策

**势阱的物理含义**：

```
             能量景观 (Energy Landscape)
     ╱╲                    ╱╲
    ╱  ╲    ╱╲            ╱  ╲
   ╱    ╲  ╱  ╲    ╱╲   ╱    ╲
  ╱      ╲╱    ╲  ╱  ╲ ╱      ╲
 ╱   吸引子A    ╲╱ B  ╲╱   C    ╲
              鞍点           鞍点
```

- **Column = 势阱**：Column 的递归连接+侧向抑制创建了一个**局部能量最低点**。信号流入 Column 的 receptive field 后被"吸引"到这个稳定状态
- **"引力弯曲"**：你的比喻惊人准确。在广义相对论中，质量弯曲时空，物体沿测地线（最短路径）运动。在 attractor network 中，**Column 的递归增益弯曲了信息流的"地形"**，信号沿"能量梯度最陡"的路径流动

### 与我们系统的对应

| 势阱机制 | 我们的代理 | 效果 |
|----------|-----------|------|
| 递归增益 | Column 的 `stdp_ltp_boost = 2.0` | 信号更容易进入 Column |
| 侧向抑制 | `lateral_suppression_radius = 3` | 排斥邻近 Spine，"挖深"势阱 |
| PRP 扩散 | `prp_emission` | 势阱的"引力范围" — 远程捕获果实 |
| 稳定性 | `inertia ≥ 2.0`, `threshold_adapt_rate × 0.5` | 势阱不容易被扰动 |

> [!TIP]
> **你的直觉（势阱/引力弯曲）有完整的理论支撑**。我们系统中 Column 的四种特权（LTP boost, lateral suppression, PRP, inertia）共同创建了一个 attractor basin。信号被这个 basin "弯曲"，自然趋向 Column 所代表的编码状态。

## 假设 3：基础 P/R 编码复杂 P/R — ✅ 这是 sparse coding 的核心思想

> "大脑并非有大量的 p/r，而是有基础的 p/r，并将其'编码'成复杂的 p/r，这样才能允许不断的修剪和沉默"

### 文献验证

这精确对应了 **Olshausen & Field (1996)** 的 **sparse coding** 理论 — 神经科学中最重要的编码理论之一。

**核心发现**：V1 的简单细胞可以被理解为一组 **基函数（basis functions）**：

$$I(x, y) = \sum_{i} a_i \cdot \phi_i(x, y)$$

- $\phi_i$ = 基函数（基础 P/R — 朝向、空间频率、位置）
- $a_i$ = 激活系数（组合权重）
- 任何复杂图像 = 少量基函数的线性叠加

**关键洞察**：
1. **基础 P/R 数量有限**（~几百种朝向/频率/位置组合）
2. **任意时刻只有少数活跃**（sparse = 大部分 $a_i = 0$）
3. **复杂表征 = 组合**（面孔 = 特定朝向组 + 特定频率组 + 特定位置组的同时激活）

**为什么允许修剪和沉默**：

> 因为信息不在单个连接中，而在**模式**中。

- 修剪一条连接 = 改变一个 $a_i$ 的通道 → 其他 $a_j$ 可以补偿（overcomplete representation 的冗余性）
- 沉默一个神经元 = 一个 $\phi_i$ 暂时不可用 → 剩余 $\phi_j$ 仍然覆盖信号空间

### 与我们系统的对应

| Sparse coding 概念 | 我们的实现 |
|--------------------|-----------|
| 基函数 $\phi_i$ | 7 个 z_t 维度（transition, drift, gamma_desync...） |
| 系数 $a_i$ | z_t profile 值（每个刺激只激活 4-5 个维度） |
| 组合编码 | `cos(movie, scenes) = 0.34` — 部分基函数共享 |
| Overcomplete | cx_ 结晶 — 创建超出 7 维的冗余基函数 |
| 修剪容忍 | 突触收缩不破坏系统 — 因为信息在模式中 |

> [!IMPORTANT]
> **你的假设与 Olshausen-Field 1996 完全一致**。我们系统的 7 个 z_t 维度就是基函数，cx_ 结晶创建组合基函数。这解释了为什么突触收缩和神经元休眠不会破坏系统 — 信息是分布式的。

## 假设 4：P/R 之间的距离和位置 — ✅ 有文献支持 + 需要验证

> "不同 p/r 之间的'距离'和'位置'是怎样的？也许在修剪和迁移的作用下，'关系'相近的不同 p/r 会靠近"

### 文献验证

这对应了两个成熟领域：

**A. 自组织映射（Self-Organizing Map, Kohonen 1982）**
- 皮层的拓扑映射（tonotopic, retinotopic, somatotopic）证明：**功能相似的神经元在物理空间中聚集**
- 机制：Hebbian 学习 + 侧向抑制 → 相似输入的响应区域自然靠近
- 这就是你说的"关系相近的 P/R 会靠近"的精确物理实现

**B. 表征漂移（Representational Drift, Druckmann & Chklovskii 2012; Gonzalez et al. 2019）**
- 最近的发现：即使行为稳定，单个神经元的调谐特性在日/周尺度上持续变化
- 但是**群体流形的几何结构保持不变** — 表征之间的**相对距离**是守恒的
- 这意味着：P/R 可以在神经元空间中"漂移"（迁移），但它们之间的距离关系是拓扑不变量

**C. 反流形拼凑**
- 你之前讨论的"反流形拼凑"对应了流形对齐（Manifold Alignment）：
  - 不同时间/状态下的表征流形可以通过线性变换对齐
  - 相似的表征在对齐后的流形中**聚类**
  - 修剪 = 降维（去除冗余方向），迁移 = 流形中的点重排

### 与我们系统的对应

```
当前 P/R "距离" 度量：
  cos(movie, scenes)    = 0.34   → 较近（共享 drift, xin_residual 等基函数）
  cos(movie, gratings)  = 0.53   → 中等
  cos(scenes, gratings) = 0.04   → 很远（几乎正交）

cx_ 结晶 = "拼凑"操作：
  cx_chu_dri = churn × drift → scenes 和 movie 的共享基被结晶为结构节点
  这个结晶节点拉近了 scenes 和 movie 在表征空间中的 "距离"
```

> [!WARNING]
> **这个假设需要外部数理验证**。具体来说：
> 1. 需要在熵账本中追踪 P/R 之间的 representational similarity matrix 随 tick 的变化
> 2. 验证修剪/结晶是否真的改变了 P/R 之间的拓扑关系
> 3. 验证是否存在 "关系相近的 P/R 靠近" 的自组织效应

## 总结：对项目的结构性意义

| 假设 | 判定 | 现有代理 | 缺口 |
|------|:----:|---------|------|
| ① 信号补偿/简单制式 | ✅ 正确 | inertia 恢复 + threshold 门控 | **缺树突棘非线性再生** |
