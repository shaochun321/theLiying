## §1 三个守恒律

### 守恒律 ①：Motor 放电占比恒定 ≈ 84.3%

```
200k   400k   600k   800k   1M     1.2M   1.4M   1.6M   1.8M   2M
85.9%  85.1%  85.1%  85.1%  84.7%  84.6%  84.4%  84.4%  84.4%  84.3%
```

Motor 从 65→286 个神经元（×4.4），但放电率**始终恒定在 84-85%**。

**数学表达**：

$$\frac{S_{motor}}{S_{total}} = \text{const} \approx 0.843 \pm 0.005$$

**物理含义**：系统的"运动带宽"是一个**拓扑不变量**。无论 motor 有多少个神经元，它们的总放电占比不变。这说明新增 motor 不是"增加新信号"，而是"分摊同一个信号"——每个 motor 的个体放电密度在下降。

验证：
```
Spike density (spikes/neuron):
  Motor:      1365  ← 最低（286个神经元分摊同一个驱动）
  Otolith:    3175  ← 高（9个神经元承载大量信号）
  Vestibular: 2398
  Thermal:    7470  ← 最高（3个神经元，持续放电）
```

### 守恒律 ②：感觉通路宽度恒定

```
Step      Motor   Oto   Vest   Therm
200k        65     9      9      3
2M         286     9      9      3
```

Motor 从 65 增长到 286，但 **Otolith、Vestibular、Thermal 的神经元数量从未变化**。

这不是巧合——是因为这三个感觉通路的 Xin 不够大到触发 sprout（在这些通路上），或者 sprout 立刻被 prune（竞争失败）。结构生长**只发生在 motor 输出端**。

### 守恒律 ③：Oto 和 Vest 的 Xin 趋向平衡

```
Step      Oto |Xin|   Vest |Xin|   Oto/Vest
200k      122.75       12.59        9.75
400k      244.17       12.58       19.41   ← body 运动最剧烈
600k      120.16       12.61        9.53   ← body 接近边界
800k       66.27       12.51        5.30   ← body 卡住
1M         12.86       11.61        1.11   ← 接近平衡！
1.2M       10.04       12.01        0.84
1.4M       10.13       11.62        0.87
1.6M       10.32       12.05        0.86
1.8M       10.54       12.02        0.88
2M         10.70       11.97        0.89   ← 收敛
```

**Body 停止后，Oto Xin 从 244 降到 ~10，与 Vest Xin ~12 达到近似平衡。**

$$\lim_{t \to \infty} \frac{|\xi_{oto}|}{|\xi_{vest}|} \approx 0.89$$

## §2 相变：Body 撞墙

```
200k: [58, 56, 58]   speed=0.117  ← 加速期
400k: [72, 71, 74]   speed=0.152  ← 加速期
600k: [90, 90, 95]   speed=0.179  ← 接近边界
800k: [100,100,100]  speed=0.000  ← 撞墙！卡住
1M-2M: [100,100,100] speed=0.000  ← 永久卡住
```

Motor 持续放电 → 持续施力 → body 加速 → 到达 [100,100,100] 角落 → 弹回力 = 驱动力 → 净力≈0 → 停下来。

**撞墙前后的系统行为变化**：

| 指标 | 撞墙前 (200k-600k) | 撞墙后 (800k-2M) |
|------|-------------------|-----------------|
| Oto Xin | 120-244（巨大） | 10-11（微小） |
| Vest Xin | 12.6（恒定） | 12.0（恒定） |
| Therm Xin | 0.36-0.84 | 0.16-0.26 |
| Mitosis | 21-62/200k | 18-24/200k |
| Sprout | 56-58/200k | 54-58/200k |

**撞墙是一个相变点**：Oto 通路从"剧烈不可预测"变为"近似可预测"。

## §3 关于"时间、空间、运动的宽度"

你问的三个维度的实际测量：

### 运动（Motor）
- **宽度**: 3 → 286（×95），持续线性增长
- **增速**: ~24 neurons / 200k steps（恒定）
- **特征**: 宽度无限增长，但个体密度递减

### 空间（Otolith = 加速度 = 空间位移的二阶导）
- **宽度**: 恒定 = 9
- **Xin**: 从 244 衰减到 10（body 停止后）
- **特征**: 宽度不变，但信息含量随运动停止而骤降

### 时间（Vestibular = 振荡 + 外部输入的节律）
- **宽度**: 恒定 = 9
- **Xin**: 始终 ≈ 12（不依赖 body 运动状态）
- **特征**: 宽度和信息含量都完全稳定

### 它们之间的数学关系

$$W_{motor}(t) = W_0 + \alpha \cdot t, \quad \alpha \approx 24/200k = 1.2 \times 10^{-4}$$

$$W_{oto} = W_{vest} = \text{const} = 9$$

$$\frac{|\xi_{oto}|}{|\xi_{vest}|} \xrightarrow{body停止} 0.89$$

**核心发现**：

> 运动通路的**宽度**在增长，但感觉通路的**宽度**不变。
> 信息（Xin）在 body 停止后，Oto 和 Vest 趋向**等势**。

这意味着：
1. Motor 的"宽度增长"不是因为信息增加，而是因为 **Mitosis 不断分裂但每个后代承载的信息量递减**
2. 感觉通路的宽度恒定说明 **9 个神经元足够承载当前的信息流**
3. Oto/Vest Xin 的趋同说明 **在平衡态下，空间误差 ≈ 时间误差** — 这正是你说的"时间和空间互为 P/R"的数学证据

## §4 Weight Entropy 的分化

```
Motor:      0.000  ← 所有权重完全相同（无分化）
Otolith:    1.275  ← 中等分化
Vestibular: 2.833  ← 最大分化（接近 log2(20)=4.32 的上限）
Thermal:    1.000  ← 低分化
```

**Vestibular 的权重熵最高** = 权重分布最不均匀 = 最大的结构分化。

**Motor 的权重熵为零** = 所有 motor sprout 的权重完全相同 = 没有竞争分化 = 纯粹的克隆增殖。

这暗示 Motor 的 Mitosis 是一种**退化增殖**——不产生新功能，只是复制。

## §5 对时空环流理论的启示

| 你的理论预测 | 实验结果 | 吻合？ |
|------------|---------|--------|
| 时间和空间互为 P/R | Oto/Vest Xin 趋向 0.89 比 | ✅ 趋向等势 |
| 运动有独立生成机制 | Motor 宽度无限增长 | ✅ 但是退化增殖 |
| 时空有自己的生成机制 | Oto/Vest 宽度恒定 | ⚠️ 有稳定性但无"生成" |
| Xin 是轨迹超预期 | body 停止后 Xin 骤降 | ✅ 无运动 = 无超预期 |

**关键洞察**：当前系统的"空间"（Oto）完全依赖运动——body 一停，空间 Xin 就塌缩。这说明 **空间还没有独立的生成机制**。真正的空间表征需要"即使不动也知道自己在哪"——这就是你说的 PathIntegrator。
