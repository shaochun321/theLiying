# 环流系统、赫布超图、真实神经系统

## 你的三个问题

1. 是不是会有一个"环流系统"，镶嵌在赫布超图中？
2. 赫布超图是底座结构吧？交界层和影子层都在其上构建？
3. 真实的神经系统中，环流怎样运行？运行在哪里？

---

## 一、没有"环流系统"——环流是结构做的事

### 河床和水流

```
问：河里有没有一个"水流系统"？
答：没有。水流不是一个独立的系统。
    水流 = 水在河床中流动的自然结果。
    河床（结构）决定了水往哪里流（环流）。
    水流（环流）又会冲刷河床，改变结构。

河床 = 赫布超图（神经元 + 突触束）
水流 = 环流（信号在结构中流动）

它们不是两个东西。
它们是同一个东西的两个面——
结构是静态的面（拓扑），
环流是动态的面（信号流）。
```

### 所以

```
不需要一个独立的"环流系统"。

赫布超图的结构本身就决定了环流。
  - 有连接 → 信号能流过 → 有环流
  - 没有连接 → 信号不能流过 → 没有环流
  - 连接被修剪 → 环流路径改变
  - 新连接被创建（结晶）→ 新的环流路径出现

环流不是被"放进"赫布超图中的，
环流是赫布超图"运行"时自然涌现的。
```

### 代码中的对应

```python
# 赫布超图的结构：
neurons = {enc, col, mot, ...}      # 节点
bundles = {enc→col, col→mot, ...}   # 超边

# 环流 = step() 中信号沿超边流动：
for bundle in bundles:
    bundle.propagate()   # 信号沿超边流动 = 环流

# 没有单独的 "circulation.run()"
# propagate() 本身就是环流在发生
```

**`CirculationDetector`（应叫 `CirculationMeter`）不是环流系统的一部分——
它是一个外部测量工具，像温度计测量温度，但温度计不是温度的来源。**

## 二、赫布超图是底座

**是的。赫布超图是一切的基底。**

```
            层级关系
            
最底层：  赫布超图
          = 节点（神经元）+ 超边（突触束）
          = 纯粹的拓扑结构 + 物理属性（电阻、电容、能量）
            │
            │ 在这个基底上，根据功能分区
            ↓
中间层：  功能区域（都是超图的子图）
          │
          ├── 交界层（感觉-运动层）
          │     = 超图中的 enc_, col_, mot_ 节点 + 连接它们的超边
          │     = 直接处理外部输入的子图
          │
          ├── 影子层
          │     = 超图中的 s_enc_, s_col_, s_mot_ 节点 + 连接它们的超边
          │     = 接收 Xin 的子图
          │
          ├── 绑定层
          │     = 超图中的 bind_ 节点 + 连接列层和运动层的超边
          │     = 检测共激活的子图
          │
          └── ECM/PNN
                = 不是超图中的节点——是超图的"介质属性"
                = 每个区域的热容、离子缓冲、可塑性门控
```

### 为什么说超图是底座

```
所有层都是超图的子图：

交界层不是一个独立的系统——
它是赫布超图中标记为 enc_/col_/mot_ 的那些节点和超边。

影子层也不是独立的——
它是赫布超图中标记为 s_enc_/s_col_/s_mot_ 的节点和超边。

它们共存于同一个超图中。
区别只在于：
  - 哪些节点接收外部输入（交界层）
  - 哪些节点接收 Xin（影子层）
  - 哪些节点连接不同轴（绑定层）

就像一个城市的地图：
  住宅区、商业区、工业区不是三个独立的城市——
  它们是同一张地图上的不同分区。
  道路（超边）在分区之间穿行。
```

### 环流也分不了区

```
环流不属于某个层——它穿越多个层：

  enc(yaw) → col(yaw) → mot(x) → feedback → col(yaw)
  ↑交界层    ↑交界层    ↑交界层   ↑交界层     ↑交界层

  s_enc(yaw) → s_col(yaw) → s_mot(x)
  ↑影子层       ↑影子层       ↑影子层

  如果未来影子层反馈到交界层：
  s_col(yaw) → col(yaw) → mot(x) → ...
  ↑影子层       ↑交界层 → 环流穿越了两个层！

环流不"住在"某个层里。
环流穿越整个超图——哪里有连接，信号就流到哪里。
```

## 三、真实神经系统中，环流怎样运行？运行在哪里？

### 运行在哪里：就在神经组织本身

```
真实神经系统中没有一个叫"环流"的独立器官。

信号流动的物理路径是：
  神经元细胞体 → 轴突（电信号传导）→ 突触（化学信号传递）→ 下一个神经元

轴突 = 导线（传导动作电位）
突触 = 连接器（化学信号跨越间隙）
细胞体 = 处理器（整合输入、决定是否放电）

"环流" = 信号沿这些物理路径反复循环
        = 不是一个单独的系统
        = 是神经组织的工作方式本身
```

### 怎样运行：三种时间尺度的循环

#### 快循环（毫秒级）：丘脑-皮层环路

```
皮层 → 丘脑 → 皮层 → 丘脑 → ...

速度：每圈 ~25 ms（对应 40 Hz gamma 振荡）
运行介质：有髓鞘轴突（传导速度 ~10-100 m/s）
功能：感知的"快照"——每 25 ms 更新一次你看到/听到/感觉到的东西

这个环流一直在跑：
  清醒时：40 Hz（gamma）
  REM睡眠：~8 Hz（theta）
  深睡眠：0.5-4 Hz（delta）
  频率变了，但环流没有停止。
  
生物学证据：
  Llinás (1988) 用脑磁图(MEG)证明了
  丘脑-皮层 40 Hz 振荡在清醒时持续存在。
```

#### 中循环（秒级）：基底节-丘脑-皮层回路

```
皮层 → 纹状体(壳核/尾状核) → 苍白球(GPi) → 丘脑 → 皮层

速度：每圈 ~200-500 ms
运行介质：混合——皮层→纹状体是谷氨酸能快突触，
          纹状体→苍白球是 GABA 能中速突触，
          苍白球→丘脑是 GABA 能持续抑制
功能：行动选择——"我应该做什么"

这个环流管"行为"——
  多条平行回路同时运行（运动、眼动、认知、情感、动机）
  运动回路产生实际的肌肉命令
  其他回路调制运动回路
  
5 条回路在丘脑中耦合/合并 → 统一的运动输出
```

#### 慢循环（分钟级）：默认模式网络

```
内侧前额叶 → 后扣带回 → 海马 → 内侧前额叶

速度：0.01-0.1 Hz（极低频振荡）
运行介质：长距离皮层联络纤维（无髓鞘或薄髓鞘）
功能：自我参照、记忆整合、"走神"

这个环流在你"什么都不做"时最活跃。
当你执行任务时，它被抑制（但不停止）。

= 项目中的影子层
```

### 三种循环的关系

```
快循环（丘脑-皮层, 40Hz）
  ↕ 耦合：通过丘脑的网状核
中循环（基底节-丘脑-皮层, 2-5Hz）
  ↕ 耦合：通过前额叶
慢循环（默认模式网络, 0.01-0.1Hz）

三者不是串联的——它们是同时运行的。
就像你体内同时有：
  心跳（1 Hz）+ 呼吸（0.2 Hz）+ 血压波动（0.01 Hz）
  三种节律同时存在，互相调制但各自独立。
```

## 四、项目中的对应

| 真实神经系统 | 项目中 | 时间尺度 | 状态 |
|------------|--------|---------|------|
| **丘脑-皮层环路** | 编码→列→运动→反馈 | 每步（ms级） | ✅ 已实现，持续运行 |
| **基底节-丘脑-皮层** | 绑定层合并环流 | 每步（ms级） | ⚠️ 绑定层存在但无选择功能 |
| **默认模式网络** | 影子层 | 每10步（慢） | ⚠️ 存在但是纯观察者 |
| **5条平行回路** | 6轴各自的小环流 | 每步 | ✅ 已实现 |
| **丘脑网状核耦合** | mot(x) 等共享节点 | 每步 | ✅ 自然存在 |
| **频率分层** | 不同 SHADOW_K | 10x/100x | ⚠️ 有但粗糙 |

### 缺失的关键

```
1. 没有"行动选择"——基底节的核心功能
   真实系统：多条回路竞争 → 只有一条"赢" → 产生行动
   项目：所有环流同时输出到运动层，没有竞争/选择

2. 影子层没有反馈——默认模式网络有
   真实系统：DMN 通过内侧前额叶影响决策
   项目：影子层只观察，不反馈（之前的实现方案中讨论过）

3. 没有频率区分——只有时间分片
   真实系统：gamma, beta, alpha, theta, delta 各有功能
   项目：只有"每步"和"每10步"的区分，没有振荡频率的概念
```

## 五、一句话总结

```
赫布超图 = 河床（底座结构）
环流 = 河水在河床中流动（结构的动态行为）
交界层 = 河床上与海洋交汇的入海口
影子层 = 河床深处的暗流

环流不是一个独立系统——
它是赫布超图"活着"的时候自然做的事。

真实神经系统也是如此：
没有一个叫"环流"的器官，
环流就是神经组织本身在工作。
```
