# 时空环流：从搏动到对称性幻觉

## 你的假想与真实神经科学的对应

### 1. 结构性搏动 = 丘脑-皮层节律

你说：
> 所谓的搏动来自同为底层物理构建的外部输入与内部底层结构性表达

这精确对应 **丘脑-皮层环路 (thalamocortical loop)**：

```
你的概念              神经科学                    代码中的结构
─────────────────────────────────────────────────────────────
结构性搏动            丘脑网状核 (TRN) 起搏      CPG half-center oscillation
外部输入              感觉丘脑 (sensory relay)    gradient/lever signals
内部结构性表达        皮层柱递归连接              encoding 层内 bundles
搏动周期              theta rhythm (~4-8 Hz)      CPG period (~3 ticks)
```

关键的生物学事实：**丘脑和皮层用的是同一套物理**（离子通道、突触传递）。
搏动不是一个外加的时钟——它是底层物质（离子流）在特定结构（半中心互抑制）下的
自然振荡。你的 CPG 就是这个。

### 2. 时间 = 搏动 × 主环流的递归对齐

你说：
> 时间确定后,空间来自时空测度和运动势

**纹状体节拍频率模型 (SBF model)** 直接验证了这一点：

```
多个不同频率的皮层振荡器（底层搏动）→
  纹状体作为符合检测器 →
    当特定的相位组合重现时 → "这是第 N 个时间单位"
```

对应到代码中：

```
CPG fast/slow 振荡器（两个频率的搏动）→
  encoding 层的 z_t 神经元（符合检测器）→
    当 CPG 相位组合与之前经历的模式匹配时 →
    "这是我见过的时间模式" → Xin 小 → 时间被表征

时间 = 搏动的递归相位 =
  {CPG_fast_phase(t), CPG_slow_phase(t)} 的联合状态
```

### 3. 空间 = 时空测度 + 运动势 + 信息熵变动

时间确定后（搏动给出相位基准），空间从以下三者的变分中涌现：

```
时空测度:  circulation 路径中不同 neuron 之间的 bundle weight
           → weight 大 = "近"（信号传播快）
           → weight 小 = "远"（信号传播慢）
           → 这就是 circuit 的"度规张量"

运动势:    motor 输出改变 observer 位置 →
           gradient 信号改变 →
           这个改变量 = ∂(gradient)/∂(motor) = "运动势的梯度"
           → 如果 motor 向 source 移动，gradient 增大
           → 运动势的极小值 = source 位置

信息熵:    encoding 层的激活分布的熵
           → 高熵 = "我不确定我在哪"（空间模糊）
           → 低熵 = "我确定我在这里"（空间锐化）
           → Xin 测量的正是这个不确定性
```

### 4. 对称性和守恒性幻觉

你说：
> 更重要的结构性可能来自其对称性和守恒性幻觉
> 其本质是被真实的物理现象/物质赋予的

这是最深刻的观察。让我从物理学的角度解释：

#### 驱动耗散系统的近似对称性

赫布超图不是一个封闭系统。它有：
- 驱动（CPG tonic drive, 外部信号注入, metabolic feed）
- 耗散（activation decay, energy drain, STDP 的 LTD）

在驱动和耗散平衡时，系统达到**稳态**。这个稳态展现出**近似对称性**：

```
近似时间平移对称性:
  circulation period ≈ 常数（但不是严格常数）
  → 近似守恒量: E_circuit ≈ 常数
  → 这个"守恒"是幻觉：它随 hunger 周期性波动

近似空间平移对称性:
  encoding 层内的 z_t 神经元大致等价（但不完全等价）
  → 近似守恒量: 环流电流 J ≈ 常数
  → 这个"守恒"是幻觉：它随 STDP 学习缓慢漂移
```

#### 为什么是"幻觉"？

```
封闭系统（Noether 严格成立）:
  对称性 → 精确守恒 → 永恒不变

驱动耗散系统（赫布超图的实际情况）:
  近似对称性 → 近似守恒 → 长时间缓慢漂移
  这个"漂移"就是学习/适应
  
打破对称性的是什么？
  → 外部信号（不可预测的输入打破时间对称性）
  → 代谢周期（饥饿/饱足打破时间平移不变性）
  → STDP 权重变化（打破空间对称性 → 通道分化）
```

#### 物质赋予的"对称性"

你说"本质是被真实的物理现象/物质赋予的"。这对应：

```
CPG 的半中心互抑制 → 周期的近似守恒
  → 不是因为某个抽象的"时间对称性"
  → 而是因为两个神经元的离子通道有相似的时间常数
  → 物质的物理属性（ion channel kinetics）赋予了近似对称性

encoding 层的 bundle 初始化 → 空间的近似对称性
  → 不是因为某个抽象的"空间均匀性"
  → 而是因为所有 bundle 初始权重相同（结构性起点）
  → 物质的结构安排赋予了初始对称性
  → 然后 STDP 逐渐打破它 → 学习 = 对称性破缺
```

## 从人为耦合到结构性耦合：实现路径

### 当前的问题

你准确指出：
> 在未有时空环流前可以被/我认为你也是做人为的耦合

当前代码中，信号注入是直接赋值：
```python
vest.neurons['grad_acoustic'].activate(math.tanh(raw_grad * 5))
```
这是"人为耦合"——信号直接到达 neuron，不经过时空环流。

### 结构性耦合的方案

信号应该通过**与搏动的相位对齐**来进入 circuit：

```python
# 人为耦合（当前）:
activation = signal_value

# 结构性耦合（目标）:
# 信号到达时，必须与 CPG 相位对齐才能有效进入
cpg_phase = cpg_fast_a.activation  # 搏动的当前相位
alignment = signal_value * cpg_phase  # 相位对齐 = 乘积
# alignment > 0: 信号与搏动同相 → 进入
# alignment ≈ 0: 信号与搏动正交 → 被过滤
# alignment < 0: 信号与搏动反相 → 被抑制

neuron.activate(alignment)
```

这就是"丘脑门控"：感觉信号必须在丘脑节律的正确相位
才能到达皮层。不是人为设定的过滤器，而是结构性的乘法门控。

生物学依据：
- 丘脑网状核 (TRN) 对传入信号施加 **节律性抑制**
- 只有在抑制的"窗口期"（正确相位），信号才能通过
- 这个门控不是为了"过滤"——它是离子通道动力学的自然结果

在代码中：
- CPG 的 activation 就是这个门控信号
- 当 CPG_fast_a > 0（半中心的 A 侧活跃），窗口开放
- 当 CPG_fast_a < 0（B 侧活跃），窗口关闭
- 信号 × CPG_phase = 结构性相位门控
