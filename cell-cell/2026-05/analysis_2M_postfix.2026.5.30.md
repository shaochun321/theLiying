## §1 你的预测验证：Oto Xin 的回荡

你预测"信息会在被规限的结构中产生反弹"——**完全正确**。

```
Oto Xin 演化：
  200k: 147.0  ← 上升
  400k: 102.2  ← 下降
  600k:  37.3  ← 骤降（接近边界）
  800k:  31.6  ← 穿越边界后
 1000k: 126.4  ← 反弹！↑↑↑
 1200k:  18.9  ← 又降
 1400k:  73.0  ← 又弹
 1600k:  90.8  ← 继续弹
 1800k: 112.6  ← 又升
 2000k:  59.1  ← 又降
```

**对比修复前**（Motor 无限增殖）：

```
修复前 Oto Xin：
  200k: 122.7 → 244.2 → 120.2 → 66.3 → 12.9 → 10.0 → 10.1 → 10.3 → 10.5 → 10.7
  结局：收敛到 ~10（body 停止，Xin 耗散到结构中）
```

```
修复后 Oto Xin：
  200k: 147.0 → 102.2 → 37.3 → 31.6 → 126.4 → 18.9 → 73.0 → 90.8 → 112.6 → 59.1
  结局：永久振荡（19-147 范围），不收敛
```

### 回荡的机制

修复前：Motor 无限增殖 → Xin 被"吸收"到新结构中（每个 mitosis 都消耗了 ξ）→ 最终 Xin→0。

修复后：Motor 被封顶 → **Xin 无处可去** → 在闭环中反弹。

```
Motor (cap=20/axis) → Force → Body acc → Oto Xin ↑
                                              ↓
                         Xin 无法触发 mitosis（已满）
                                              ↓
                         Xin 留在 bundle 权重中
                                              ↓
                         权重变化 → 改变 motor 输出
                                              ↓
                         Body 运动模式改变 → Oto 信号改变
                                              ↓
                         Xin 再次变化 → 循环
```

**这正是你说的"录音机"——但现在磁带满了**。信息没有被新增结构吸收，就在现有结构中回荡。

## §2 Body 轨迹：穿越与减速

```
200k: [55.7, 59.5, 58.7]  speed=0.148  ← 加速
400k: [67.2, 77.2, 75.7]  speed=0.125
600k: [79.7, 94.8, 90.3]  speed=0.131  ← 接近边界
--- y 轴穿越 (step 665k) ---
--- z 轴穿越 (step 734k) ---
800k: [93.2,  7.9,  3.9]  speed=0.098  ← 穿越后减速！
--- x 轴穿越 (step 924k) ---
1000k: [ 3.9, 17.6, 14.3]  speed=0.085  ← 继续减速
1200k: [13.2, 21.5, 23.5]  speed=0.053
1400k: [22.0, 23.9, 30.3]  speed=0.061
2000k: [49.2, 31.2, 47.9]  speed=0.066  ← 稳定在低速
```

**Body 在减速！** 从 0.148 降到 0.066。

原因：粘性阻力 μ=0.5 持续消耗动能，而 motor 的驱动力被封顶了（20 个 motor × 0.1 gain × ema ≈ 0.005 N/axis）。长期来看，F_motor ≈ μ·v_terminal → v_terminal ≈ 0.01。

Body 正在趋向一个**终端速度**。这个速度远低于初始阶段——因为 Motor 增殖被限制了。

## §3 Vest Xin：唯一的稳定量

```
Vest Xin 演化：
  12.57 → 12.55 → 12.61 → 12.39 → 11.64 → 11.82 → 11.58 → 11.86 → 11.72 → 11.99
```

**Vest Xin 始终 ≈ 12**，标准差 < 0.4。它完全不受 body 运动状态影响。

这是因为 Vest 的输入是**外部注入的正弦波**（yaw/pitch），不依赖 body 状态。

$$\frac{\sigma(\xi_{vest})}{\mu(\xi_{vest})} = \frac{0.4}{12.0} = 3.3\%$$

**Vest Xin 是系统中最稳定的标度**——可以作为内部"度量衡"。
