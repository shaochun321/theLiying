# 地基 + 材料 — 统一分析

## 现在的问题是一座歪房子

```
你要盖的:
  一栋楼, 每层住一户人, 每户有独立管道

现在盖的:
  所有人住一间大通铺, 共用一根管子
  然后在通铺里装了最好的门锁和温控器

  门锁再好, 温控器再精密——
  大通铺的问题不是门锁能解决的
```

---

## Part 1: 地基 — 修复规模对应

### 现在: 大通铺

```
30 个粒子 (细胞)
    │
    ├── 粒子 1 的 V_m ─┐
    ├── 粒子 2 的 V_m ─┤
    ├── ...             ├──→ 求平均 ──→ 一个数: v_avg
    ├── 粒子 29 的 V_m ─┤
    └── 粒子 30 的 V_m ─┘
    
    30 个粒子 → 7 个统计量 → 6 个信号 → 7 个 z_t → 7 个 column
    
    每个 z_t 神经元不知道自己代表哪个粒子。
    它代表的是"所有粒子的某个统计特征"。
```

### 修正后: 独立房间

```
30 个粒子 (细胞)
    │
    ├── 粒子 1 ──→ 元单元 1 ──→ 管道 1 ──→ column 元单元 1
    ├── 粒子 2 ──→ 元单元 2 ──→ 管道 2 ──→ column 元单元 2  
    ├── ...
    ├── 粒子 29 ──→ 元单元 29 ──→ 管道 29 ──→ column 元单元 15
    └── 粒子 30 ──→ 元单元 30 ──→ 管道 30 ──→ column 元单元 15
    
    每个元单元知道: "我负责粒子 i"
    每个管道知道: "我连接的是粒子 i 的元单元 和 column j 的元单元"
    
    = 你原始设计的 1 cell : 1 pipeline : 0.5 T/O/P/R/Xin
```

### 这个修正改变了什么

```
之前: 元单元 1 收到的信号 = 所有粒子的平均值
      → 粒子 1 异常? 被平均掉了, 看不见
      → 粒子 1 和粒子 2 不同? 也被平均掉了

现在: 元单元 1 收到的信号 = 粒子 1 的完整信息
      → 粒子 1 异常? 立刻反映在元单元 1 上
      → 粒子 1 和粒子 2 不同? 元单元 1 和元单元 2 会有不同的激活
```

## Part 2: 材料在修正后的地基上怎么工作

### STDP (时序记忆) 在修正后的意义

```
之前 (大通铺):
  元单元 1 的 pre_trace = "某个统计量什么时候变了"
  → 不知道是哪个粒子导致的
  → 因果关系模糊
  → STDP 调整权重, 但调的是"统计量之间的因果"
  → 这种因果可能是虚假的 (辛普森悖论)

修正后 (独立房间):
  元单元 1 的 pre_trace = "粒子 1 什么时候激活了"
  → 明确知道是粒子 1 的信号
  → 如果粒子 1 激活后, column 1 也激活了 → 真因果!
  → STDP 增强 "粒子 1 → column 1" 这条管道
  → 这种因果是可靠的
```

**结论: STDP 需要清晰的个体信号才有意义。在大通铺里, STDP 学到的因果是混乱的。**

### 钙传感器 (自稳态) 在修正后的意义

```
之前 (大通铺):
  元单元 1 的 target_rate = 0.03
  → 它代表 "所有粒子的平均值" 的活跃目标
  → 如果有些粒子活跃有些沉默, 平均值可能正好 0.03
  → 钙传感器说 "一切正常!" → 但其实不正常
  → 部分粒子异常被平均隐藏了

修正后 (独立房间):
  元单元 1 的 target_rate = 0.03
  → 它只代表 "粒子 1" 的活跃目标
  → 粒子 1 突然沉默 → 元单元 1 的 calcium 下降
  → 钙传感器: "粒子 1 太安静了! 降低阈值!"
  → 元单元 1 变得更灵敏, 试图"听到"粒子 1 的微弱信号
  → 异常被检测到了
```

**结论: 自稳态需要知道"谁"在异常。在大通铺里, 个体异常被群体平均掩盖了。**

### 场效应管 (阈值) 在修正后的意义

```
之前 (大通铺):
  所有元单元的阈值由同一种统计量驱动
  → 活跃的粒子和沉默的粒子共用一个阈值
  → 阈值要么太高 (沉默的粒子永远听不见)
  → 要么太低 (活跃的粒子一直在喊, 淹没别人)

修正后 (独立房间):
  元单元 1 的阈值由粒子 1 的信号特性驱动
  → 粒子 1 是活跃型 → 元单元 1 的阈值自动升高 (滤噪)
  → 粒子 15 是沉默型 → 元单元 15 的阈值自动降低 (增敏)
  → 每个元单元自动适应自己负责的那个粒子
  → 这就是真正的分化 — 不是人为设参数, 是自稳态自动产生的
```

**结论: 分化不需要手动设置参数! 修复地基后, 每个元单元的钙传感器会自动把阈值调到适合它自己粒子的水平。**

## Part 3: T/O/P/R/Xin 在修正后的每个层级

### 单元级 T/O/P/R/Xin

```
元单元 i (负责粒子 i):

  T: 粒子 i 的信号到达 → 注入电容器
  O: 电容器电压 = 实际收到的信号强度
  P: 钙传感器记住的 target_rate = 我期望的活跃度
  R: 如果 O ≠ P → 调整场效应管阈值
  Xin: |O - P| 的热耗散 → 残差 → 
       如果残差大 → 果实 (标记"这里有问题")
```

### 管道级 T/O/P/R/Xin

```
管道 (元单元 i → column 元单元 j):

  T: 元单元 i 的输出, 经过管道的延迟和衰减, 到达 column j
  O: column j 实际收到的信号 (= 元单元 i 的输出 × 忆阻器权重 × 衰减)
  P: column j 根据之前的经验, 预期应该收到多少
  R: STDP 调整忆阻器权重:
     元单元 i 先激活, column j 后激活 → 增强 (因果确认)
     column j 先激活, 元单元 i 后激活 → 削弱 (非因果)
  Xin: 残差张力 (xin_tension) → 
       如果持续偏高 → 果实 → 等待第三因子 → 结晶
```

### 区域级 T/O/P/R/Xin

```
encoding 区域 (由 N 个元单元组成):

  T: 所有 encoding 元单元的输出, 通过管道群送往 column 区域
  O: encoding 层的整体激活模式 (哪些元单元活跃, 哪些沉默)
  P: 期望的激活模式 (由门控判断: 同步性+对比度是否正常)
  R: 层级温度调节 + 门控开关
     门控开 → 允许管道群学习
     门控关 → 冻结管道群 (巩固保护)
  Xin: 层级残差 → 
       如果整体模式异常 → 门控自动调整
       消亡的果实 → 沉淀层
```

### 全系统 T/O/P/R/Xin

```
整个系统:

  T: motor 输出 → 物理引擎 → 粒子系统运动
  O: 粒子运动后的感觉反馈 (per-cell 信号, 不再是聚合统计!)
  P: 系统整体预期 (自由能最小化目标)
  R: 全局学习率调整 + 能量分配
  Xin: 全局残差 → "系统的困惑度"
       高困惑 → 释放更多能量到 encoding → 让学习更积极
       低困惑 → 节约能量 → 维持现状
```

## Part 4: 修正后, 之前那些问题还存在吗

| 问题 | 修正前 | 修正后 |
|------|--------|--------|
| 学了就忘 | ❌ 所有内容写在同一组 7 个权重上 | ✅ 每个粒子有自己的权重, 不互相覆盖 |
| 容量不足 | ❌ 最多 34 个神经元 | ✅ N 个粒子 → 2N 个神经元, 随规模增长 |
| 对应断裂 | ❌ 30→7 压缩, 个体信息丢失 | ✅ 1:1 对应, 每个粒子可追溯 |
| 分化需要手调 | ❌ 手动设 5 组参数 | ✅ 自稳态自动分化 |
| 门控需要吗 | ⚠️ 有用但不够 | ✅ 仍然需要, 但在正确的地基上效果更好 |
| STDP 有效吗 | ⚠️ 因果模糊 | ✅ 清晰的 per-cell 因果 |

### 但有新问题

```
新问题 1: 规模增大
  修正前: 20 个神经元 → 计算快
  修正后: 2×30 = 60 个神经元 + 30 条管道 → 计算量 ×3
  
  → 真实数据 Allen Brain 有几百个 cells
  → 需要考虑计算效率

新问题 2: 管道数量爆炸
  如果每个 encoding 元单元都连每个 column 元单元:
  N encoding × N/2 column = N²/2 条管道
  30 个粒子 → 450 条管道
  
  → 需要稀疏连接 (不是全连接)
  → 每个 encoding 只连几个 column, 不是所有

新问题 3: 初始对应怎么定
  "粒子 1 连 column 1, 粒子 2 连 column 1"
  → 这个初始分配是随机的还是有依据的?
  → 需要一种自组织机制让连接自然形成
```

## 总结

```
地基修正 = 恢复 1:1 对应
  _compute_sensory() 不再压缩, 每个粒子保留独立信号
  build_circuit_from_cells(N) 替代 build_circuit_from_signal_transform()

材料重新安装:
  STDP → 在 per-cell 管道上才能检测真因果
  自稳态 → 在 per-cell 元单元上才能检测真异常
  场效应管 → 在 per-cell 信号上才能自动分化阈值
  门控 → 在区域级别控制"什么时候允许学习"
  果实 → 在 per-cell 管道上标记 per-cell 的预测错误
  结晶 → 在 per-cell 收敛检测上发现 per-cell 的关联

T/O/P/R/Xin → 在每个层级递归运转:
  单元级: 信号→电压→目标→调节→热
  管道级: 传输→到达→预期→STDP→张力
  区域级: 模式→激活→门控→冻结/学习→沉淀
  系统级: 动作→反馈→预期→全局调节→困惑度
```
