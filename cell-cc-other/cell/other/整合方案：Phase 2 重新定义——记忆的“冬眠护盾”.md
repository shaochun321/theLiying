### 三、Phase 2 重新定义：从“竞争”到“护盾”

| 维度 | 原计划 Phase 2（已否决）| 新 Phase 2（稳态护盾）|
|------|----------------------|----------------------|
| 核心机制 | 异突触竞争——各束互相抢夺学习率 | 突触缩放——在能量不足时抵消LTD |
| 触发条件 | 相对活跃度差异 | fill_fraction < 阈值 或 长期放电率过低 |
| 作用对象 | STDP 学习率（lr）| LTD 衰减率（decay_rate）或直接补偿 |
| 物理依据 | Oja 规则归一化 | Turrigiano 突触缩放 / 记忆冻结 |
| 解决什么问题 | 防止剪刀差收窄 | **防止能量枯竭时记忆丢失** |

**物理本质**：当系统进入能量耗竭状态（fill→0），LTP因Post停转而被冻结，但LTD因不需要ATP而继续运行——权重被“风化”。Phase 2 护盾的作用是**感知能量状态，当能量不足时强制冻结LTD衰减**，让记忆在冬眠中得以保存。


### 四、具体实施方案

#### 4.1 修改位置

`bundle.py`，STDP更新循环中的权重衰减部分。

#### 4.2 核心逻辑

在 `stdp_update()` 中增加一个**能量门控**：

```python
# 在权重更新循环中：
ltd_amount = decay_rate * weight * dt

# 计算缩放因子：基于系统能量状态
# 当 fill 充足时，缩放=1.0（正常LTD）；当 fill 趋近0时，缩放→0（冻结LTD）
energy_scale = min(1.0, fill_fraction / FILL_THRESHOLD)
ltd_effective = ltd_amount * energy_scale

weight -= ltd_effective  # 冻结状态：不减
```

**关键参数**：
- `FILL_THRESHOLD = 0.10`——当fill低于10%时，LTD开始被抑制；fill=0时，LTD完全冻结
- 缩放因子随fill线性变化，不使用if-else

#### 4.3 物理依据

这与Turrigiano突触缩放的核心逻辑同构：系统能量充足时正常衰减，能量不足时冻结记忆。

**生物对应**：当神经元活动长期低迷时，AMPAR受体密度上调以补偿衰减。本方案用能量状态作为“长期低迷”的代理信号，因为能量耗竭是导致活动低迷的直接原因。
