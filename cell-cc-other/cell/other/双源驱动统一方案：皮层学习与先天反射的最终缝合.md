## 第一章：两条路线的物理本质

延长线路线与平行路线在物理上不是对立关系，而是生物体发育的两个阶段：

| 维度 | 延长线（皮层学习） | 平行路线（先天反射） |
|------|-------------------|---------------------|
| 时间尺度 | 千步~万步（STDP成熟） | 单步~百步（即时响应） |
| 物理通路 | Enc→Col→Motor（可塑） | Skin→Reflex→Motor（硬连线） |
| 学习依赖 | 需要STDP权重成长 | 不需要，先天固化 |
| 功能角色 | 精细化、可适应的行为 | 快速、鲁棒的生存反应 |
| 生物对应 | 皮层-脊髓学习 | 脊髓反射 + 脑干趋向 |

生物体同时拥有两者。脊髓反射在毫秒级完成伤害回避，皮层学习在小时级优化运动策略。一个只有反射的系统是僵硬的，一个只有学习的系统在学会之前已经死了。

两份方案的最大公约数，是平行路线 Phase 3 发现的**拓扑盲区**：`therm_*`柱层没有轴特异束直达Motor——热信号被迫走cross-axis束（gain=0.7, w_max=0.15）。这是V8引擎点火后Δx仍然只有0.27的解剖学根因。


## 第二章：拓扑修补——热感轴特异束建立（全场MVP）

### 2.1 当前拓扑缺陷

```
yaw  → axis-specific (gain=10.0) → move_x
pitch → axis-specific (gain=10.0) → move_y
roll  → axis-specific (gain=10.0) → move_z

therm_front/back → 无轴特异束 → 只能走 cross-axis (gain=0.7)
therm_left/right → 无轴特异束 → 只能走 cross-axis (gain=0.7)
```

热感柱层没有直达Motor的高增益通路。200k步位移仅0.27的根本原因在此。

### 2.2 修复方案

建立热感→运动的轴特异束，赋予其与vestibular轴同等的V8配置：

```
therm_front/back → axis-specific (gain=10.0) → move_x
therm_left/right → axis-specific (gain=10.0) → move_y
```

参数设定：

| 参数 | 值 | 物理依据 |
|------|-----|----------|
| initial_weight | 0.2 | 低初始权重，让STDP自己学习 |
| weight_max | 0.5 | 与vestibular轴同等天花板 |
| synapse_gain | 10.0 | V8贝兹细胞扩音器——热趋性是生死攸关的生存行为 |
| stdp_lr | 0.05 | 关键期高可塑性，与战役四保持一致 |
| learning_rule | "stdp" | 允许经验优化 |
| bundle_role | "feedforward" | 标准前馈束 |

**此修复同时让反射层和皮层学习受益**：反射层获得高增益通路，皮层STDP获得可塑的物理介质。这是两份方案的最大公约数，也是解开Δx封印的关键一步。


## 第三章：三层运动驱动架构

### 3.1 架构总览

建立从快到慢的三层运动驱动体系：

| 层级 | 时间尺度 | 物理通路 | 功能 |
|------|----------|----------|------|
| **L2-反射层** | 单步~百步 | Skin→SpinalReflex→Motor（硬连线） | 即时伤害回避、饥饿趋向 |
| **L1-皮层学习** | 千步~万步 | Skin→Enc→Col→Motor（STDP可塑） | 经验优化、行为精细化 |
| **DA-调制层** | 跨尺度 | Shadow→DA→Reflex+STDP | 动机门控、学习率调制 |

三层同时向Motor注入电流，物理上是加法叠加：

$$I_{mot}^{total} = I_{reflex} + I_{col \to motor}^{STDP}$$

Motor不区分电流来源——膜上的基尔霍夫加法自动完成仲裁。

### 3.2 L2-反射层增强

**进食反射增益提升**：
- `hunger_approach_gain`: 0.3 → 0.6
- `hunger_gate_v_threshold`: 0.3 → 0.15（饥饿时更容易开启）
- 新增DA门控：DA浓度 × 反射增益（`da_min_gain=0.5`, `da_max_gain=2.0`）

**物理意义**：DA高时（高预测误差/高紧迫性），反射更强。DA低时（稳态），反射维持基线水平。这是基底核运动门控的物理实现。

**疼痛反射保持不变**：`REFLEX_GAIN=0.5` 已在L2.09中确认为有效参数。

### 3.3 L1-皮层学习：六刀成果保留

全部六刀成果保持不变：
- 战役三：XIN_GAIN 3→1，废除影子层人工放大器
- 影子归一：vr_base_rate 0.01→0.05
- 战役一：DA净空锁（shadow_to_da权重 1.0→0.1）
- 战役二：钙流填谷（Col CRI激活，Ca_ss≈0.31）
- 战役四：脊髓扩音（axis束 gain 5→10, lr 0.005→0.05）
- V8引擎点火：Langevin噪声注入半规管

### 3.4 三层协同的正反馈闭环

```
反射层提供即时运动 → 运动产生dT/dt信号
  → 持续的梯度攀登行为 → 产生丰富的STDP训练数据
    → 皮层学习通过新修复的轴特异束（gain=10.0）固化经验
      → 行为从反射驱动逐渐过渡到皮层驱动
        → 反射层的DA门控进一步调制学习状态
```


## 第四章：Phase 0——半导体物理Bug修复

在合入任何功能改动之前，修复两个物理漏洞：

### 4.1 Memristor电阻安全钳位

```python
@property
def resistance(self) -> float:
    w_safe = max(0.0, min(1.0, self.w))  # 防御性钳位
    return self.r_min + (self.r_max - self.r_min) * (1.0 - w_safe)
```

防止`w`因数值误差超过[0,1]区间时电阻变负。

### 4.2 Conductance物理上限

```python
@property
def conductance(self) -> float:
    g = 1.0 / max(self.resistance, self.r_min)
    return min(g, 1.0 / self.r_min)  # 硬上限 = 最小接触电阻的导纳
```

防止忆阻器进入“无限导通超导态”。两个修复共同确保半导体层在极端条件下仍然物理合法。
