## 二、因果链分析

### 已工作的链路（热趋性信号已到达）

```
SomatosensoryChain relay (front/right) → soma_to_da bundle → DA neurons
  → dopamine.concentration = 0.187（热场内，非致盲）

VitalOscillator（2Hz 三频心跳）→ Motor._membrane.inject()
  → body velocity ≈ 0.000165 m/s（基底随机游走建立）

Langevin OU 噪声 → mechanical_inputs['oto_x/y/z']
  → 打破对称死锁（body 非静止）
```

### 断裂的链路（定向运动未涌现）

```
HC(L2) → Aff(L3) → Enc(L4): ⚠️ Enc 发放率极低
  Enc pre_trace = 0.000（STDP 学习链断裂，DEG-007）
  Column 平均激活 = 0.000000（全部静默）
  Col→Motor 传播电流 = 0（无 nonzero 电流）

Motor 激活来源：
  当前: 仅来自 VitalOscillator 直接注入（20 spikes/10k）
  缺失: 来自 Col 的 STDP 驱动（weight 0.375，但无输入电流）
```

## 三、当前信号功率分析

### Motor 激活路径功率对比

| 路径 | 当前功率 | 状态 |
|------|----------|------|
| VitalOscillator → Motor | amplitude=0.023（有效）| ✅ 工作 |
| Col → Motor（STDP）| current=0（Col 全零）| ❌ 断裂 |
| soma_to_da → DA → Col 调制 | DA=0.187 但 Col 不放电 | ⚠️ 部分 |
| Langevin → Otolith | sigma≈0.0005（弱）| ✅ 工作 |

### 断裂根因：Aff→Enc 驱动不足

```
Aff 发放率: ~12.5 Hz（目标 50-100 Hz, DEG-004）
  → Aff 发放过低 → Enc 阈值未被突破 → Col 无输入 → Col 全零
  → Col→Motor = 0 → Motor 只靠 VitalOscillator
```

**Lateral inhibition**（gain=0.05）不是主要问题——即使减为 0，Col 也不会因此放电，因为它根本没有收到足够的 Enc 输入。

## 四、热趋性涌现的剩余障碍

### 必须解决

| # | 障碍 | 根因 | 修复方向 |
|---|------|------|----------|
| M1 | Enc pre_trace = 0（STDP 断链）| Aff→Enc 电流太弱 | 检查 Aff→Enc bundle 权重/gain，或降低 Enc 阈值 |
| M2 | Col 全零激活 | 无 Enc 输入电流 | 依赖 M1 |
| M3 | Col→Motor 功率 = 0 | 依赖 M2 | 依赖 M1+M2 |

### 当前方向性行为存在但极弱

test_thermotaxis_v2 显示 **9/9 approaching windows** — 方向性偏置是真实的，来自：
- soma_to_da 热梯度 → DA 调制
- VitalOscillator 提供基底能量（扩散的 STDP 机会）

但 Δx = +0.0094（90k步）约等于 **每小时移动 0.4 个世界单位**——太慢，无法在测试内达到明确的趋热行为标准。

---

## 五、下一步优先级

### 🔴 P0：修复 Enc 发放率（解锁完整信号链）

定位 Aff→Enc 电流不足的根因：
1. 检查 `bundles_aff_to_enc` 的 initial_weight 和 synapse_gain
2. 测量 Aff 实际传播电流（是否超过 Enc 阈值）
3. 若电流不足：调整 bundle 参数（需 BIO 文献依据）

路径：**Aff 发放率提升 → Enc 开始放电 → Col 激活 → Col→Motor 电流 → 体素定向运动**

### 🟡 P1：Langevin σ₀ 校准

当前 σ₀=0.005，T_bath≈0.01，sigma_k=0.0005——可能太弱。
按锚点 `σ* ≈ θ_mot/√2 ≈ 0.07`（Motor 阈值）校准。

### 🟢 P2：LangevinNoise 组件化

把 OU 过程封装为 `langevin_noise.py`，和 VitalOscillator 同等架构地位。

---

## 六、结论

> 热趋性的**信号基础设施已建立**（DA 梯度 + 皮肤感知 + 基底运动），方向性偏置已可测（9/9 approaching），但运动功率因 Aff→Enc 链路断裂而极弱。
>
> 修复 Enc 发放率是涌现完整热趋性行为的单一最高优先级任务。
