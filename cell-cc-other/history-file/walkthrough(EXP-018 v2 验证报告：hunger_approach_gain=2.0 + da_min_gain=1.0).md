## 信号链验证

```
skin patch gradient (front-back = +0.43)
  → thermoreceptor delta_x ≈ +0.45
    → hunger_drive = 0.45 × 2.0 × 1.0 × 0.53 ≈ 0.48 /step
      → motor V_ss = 0.48 × R_leak(5.0) = 2.4 >> v_th(0.15)
        → body velocity ≈ 0.003
          → monotonic approach
```

## 未解决问题

### 1. Thermal bundle 权重仍在衰减
```
therm_front_back → move_x: 0.102 → 0.094 (-7.5%)
therm_left_right → move_y: 0.097 → 0.089 (-8.3%)
```
STDP 支架效应未启动。虽然 motor 现在有活动了，但 bundle 权重的 STDP 需要 **pre-post 时序相关**。当前反射驱动是连续注入（非脉冲），不产生清晰的时序相关。

### 2. DA 仍然崩塌
DA=0 占 80% 步数。但由于 `da_min_gain=1.0`，这不再伤害反射。

### 3. 速度恒定（未加速）
0.003 速度线性进行，没有 STDP 学习带来的加速效应。当前纯靠反射层。
