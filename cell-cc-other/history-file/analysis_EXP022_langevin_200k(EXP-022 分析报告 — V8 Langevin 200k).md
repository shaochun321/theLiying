## 二、P1 FAIL 是预期行为，不是回归

**原因**：Langevin OU 噪声是热力学随机过程，不是定向信号。

前次测试（AGC 验证 20k 步）的输入是：
```python
oto_x = 200 * sin(2π × 0.5Hz × t)  # 机械正弦波，结构化驱动
```

EXP-022 的输入是：
```python
{}  # 空——仅依赖 Langevin η ∈ OU(σ₀=0.70, τ=0.5s)
```

OU 噪声的相关时间 τ=0.5s（500步），在 10k 步的时间窗口内是近似白噪声。
系统做的是**偏向随机游走（biased random walk）**，不是弹道运动。

x 轨迹验证了这一点：
```
x: 7→80→43→10→77→72→14→44→42→38...→77→67→58→32→97→68→61→34→36→96
```
振幅 ≈ ±40 单位，与随机游走在该时间尺度上的预期完全一致。

**最终 Δx=+46.04 是真实的趋热性涌现，来自以下通路的积累**：
- 皮肤 4-patch 温度梯度 → 体感链 → hunger reflex 方向偏置
- AGC 饱和 (gain=5.0) → 饥饿驱动持续放大
- Δw 持续正向（front > back 始终成立）→ 热源方向学习积累

## 三、P4/P5 FAIL 是已知代谢壁，非 V8 引入

fill 在第 90k 步耗尽（fill=0.0031），100k 步归零。

**这不是 V8 引入的退化**，而是已知的"代谢壁"问题：
- 之前 EXP-020（Phase 3）也在 100k 步内出现 fill~0.121 接近耗尽
- 差别在于：正弦波输入提供了更高效的定向运动，
  系统更早接近热源 → 更快吸收能量 → fill 维持更久

**V8 Langevin 的影响**：系统运动更"无序"，
到达热源的时间更长，期间能量净消耗更大。

fill 耗尽后（100k-200k），系统并未"死亡"：
- fill 在 0.000-0.008 间振荡（微量吸收 ↔ 消耗平衡）
- 运动继续（Δx 从 -11.69 增长到 +46.04）
- DA/AGC 仍然工作

这说明系统在"接近饥饿死亡"状态下，依然能通过 AGC 驱动维持运动。

## 四、P7 eta_mean 偏置分析

eta_mean = 0.099（应 ≈ 0）

**根本原因**：OU 过程的均值是 0，但 `eta_rms_now` 计算的是：
```python
eta_rms_now = sqrt(sum(e*e for e in c._langevin._eta) / 3.0)
```

这是**瞬时 RMS**，不是时间平均的期望值。
单步 `_eta` = [η_x, η_y, η_z]，各轴 η ∈ N(0, σ)，
瞬时 RMS 的期望值为 σ（不为零）。

**修正**：P7 的判据 `eta_mean < 0.05` 设计有误，
应该检查 `mean(η_x)` 或 `mean(η_y)` 的时间序列均值（应 ≈ 0），
而不是 `mean(RMS)`。这是诊断脚本的判据设计错误，不是系统问题。

## 五、与 EXP-021 基线对比（公平比较）

EXP-021 用的是机械正弦波 + 100k 步：

| 指标 | EXP-021（正弦波 100k） | EXP-022（Langevin 200k） |
|------|----------------------|-------------------------|

| fill@100k | 维持（AGC验证>0.1） | 0.0000（耗尽）         |
| 运动模式 | 定向弹道             | 随机游走+趋热性偏置     |

**关键结论**：Langevin 噪声使系统从"定向运动"变为"扩散探索"。
这在生物上是正确的：热噪声驱动的感觉不如机械驱动准确，
系统需要更长时间找到热源，但最终仍能到达。
