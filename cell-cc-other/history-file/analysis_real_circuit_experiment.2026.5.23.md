## 实验结果

### 1. 相平面 ✅ 极限环已确认

| 指标 | 5000 步 | 20000 步 |
|------|---------|----------|
| V_range | 0.296 | 0.222 |
| w_range | 0.009 | 0.005 |
| V_mean | 0.100 | 0.187 |
| w_mean | 0.001 | 0.021 |
| 极限环 | ✅ 是 | ✅ 是 |

**解读**：
- 运动层（mot_x）确认在极限环上运行
- V_range 从 0.296→0.222：**极限环在收缩**
- V_mean 从 0.100→0.187：**不动点在上移**
- w_mean 从 0.001→0.021：**适应电流在增强**

```
收缩的极限环 = 振荡在变小 = 系统在趋稳
上移的不动点 = 基础激活在提高 = 学习在积累
增强的适应 = 自我调节在加强 = 稳态机制生效
```

这与之前的分析一致：**极限环→STDP收敛→结晶→E_trend→0**

### 2. 权重学习 ✅ 明显变化

| 指标 | 5000 步 | 20000 步 |
|------|---------|----------|
| enc→col mid | 0.1500 | 0.1565 |

- 漂移 0.11 说明结构在显著变化

### 3. 能量 ✅ 正常消耗

- 消耗率在减慢：前 5k 消耗 8.41，后 15k 只消耗 3.0 → 趋稳

### 4. 环流 ⚠️ μ = 0

| 指标 | 5000 步 | 20000 步 |
|------|---------|----------|
| μ 均值 | 0.0 | 0.0 |
| 非零次数 | 0 | 0 |

**原因分析**：

环流测量路径：`Col → Bind → Mot → feedback → Col`

绑定层需要**多轴同时激活**才能产生非零输出。
实验中输入信号 yaw=0.5, pitch=0.3... 幅度递减，
但绑定层的 `co_activation_threshold` 可能没被超过。

```
可能原因：
A. 绑定层的共激活阈值太高
   → 列层激活不够同时达到阈值
B. CirculationMeter 的新阈值 0.01 过滤了弱信号
   → 之前 1e-8 能看到，现在看不到了
C. 20k 步不够
   → 列层权重还在增长中，需要更多步
```

> [!WARNING]
> 这是否意味着我们将阈值从 1e-8 提到 0.01 导致了回归？
> 需要用旧阈值重跑一次对比。

### 5. 影子层 ⚠️ ν = 0

影子层运动势为零 = 影子层内部没有产生有意义的动态。

**原因**：影子层是通过 Xin 驱动的。
Xin = 预测 - 实际。
如果权重在稳定变化（0.15→0.27），
那 Xin 应该不为零。

可能的问题：实验代码读取的是 `circulation_state.nu`（环流 ρ 向量的变化率），
不是影子层本身的运动势。需要直接读取 `shadow_sandbox` 的内部状态。
