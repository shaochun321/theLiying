| 6 | 10 | 0.3492 | 0.015 | 0.081 | 0.974 | 准静止 |

### 关键观察

```
所有 6 条轨迹的运动学特征几乎相同:
  速度 ≈ 0.34-0.36 (很低)
  速度方差 ≈ 0.013-0.016 (极低)
  相位速度 ≈ 0.078-0.082 (微弱漂移)
  直线度 ≈ 0.94-0.98 (几乎不转弯)

结论: 诊断数据中的细胞球处于准静止状态。
     6 条轨迹不是 6 种运动状态——
     它们是同一种运动状态(准静止)下的 6 个空间子区域。
```

## 这意味着什么

```
状态分离做的是: 把 50 个细胞分成了 6 组
  组 1: nodes [0,1,2,3]       (4 个细胞)
  组 2: nodes [28-38]         (11 个细胞)
  组 3: nodes [4-10]          (7 个细胞)
  组 4: nodes [39-49]         (11 个细胞)
  组 5: nodes [11-17]         (7 个细胞)
  组 6: nodes [18-27]         (10 个细胞)

这是空间分组 (哪些细胞彼此靠近, 信号特征相似),
不是运动类型分离 (哪些细胞在做不同运动)。

原因: 输入数据只有一种运动状态 (准静止的细胞球)。
     要分离出旋转/平移/振动, 数据中必须包含这些运动。
```

```
  5% noise:  stability = 0.958, Xin mass = 0.153  ✅ PASS

解读:
  - 遮蔽 30% 的输入后, 系统仍能以 98.4% 的稳定性分配到相同区域

```

## 要实现真正的运动状态分离, 需要什么

```
当前数据: 细胞球的诊断数据 = 准静止状态
         → 只能分离空间区域, 不能分离运动类型

要分离出 平移/旋转/振动/静止:

  方法 1: 修改细胞球模拟, 让它包含多种运动
    - 一组细胞在平移
    - 一组细胞在旋转
    - 一组细胞在振动
    - 一组细胞在静止
    → 运行状态分离 → 应该分出 4 组, 每组对应一种运动

  方法 2: 用多次运行, 每次施加不同运动
    - run1: 静止 → 产生基线
    - run2: 施加平移力 → 比较差异
    - run3: 施加旋转力 → 比较差异
    - run4: 施加振动力 → 比较差异
    → 对比不同 run 的轨迹特征差异

  方法 3: 你提到的球谐分析
    - 用球谐函数 Y_l^m 将细胞球表面信号分解
    - l=0: 呼吸模式 ↔ 静止?
    - l=1: 平移模式 ↔ 平移
    - l=2: 变形模式 ↔ 旋转/振动
    → 球谐分解天然给出运动类型分类
```
