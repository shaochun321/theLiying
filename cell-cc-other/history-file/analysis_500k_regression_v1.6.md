## 2. P_nu x H_flow: 准守恒 (CV=0.201)

| 指标 | 50k 初步 | 500k 长期 |
|---|---|---|
| avg | 3.18 | **3.78** |
| std | 0.59 | **0.76** |
| CV | 0.185 | **0.201** |
| 判定 | PASS | **边界** (刚超 0.2) |

### 按相位分解

| 相位 | 步数 | 输入 | P*H 范围 |
|---|---|---|---|
| Ph1 | 0-200k | oto_x + pitch | 2.54 - 3.86 |
| Ph2 | 200k-350k | oto_x half | 2.82 - 4.81 |
| Ph3 | 350k-500k | oto_x only | 3.03 - 4.52 |

### 漂移根因

```
H_flow: 4.47 → 5.13 (↑ 慢增长, 因为 sprouting 增加信号路径)
P_nu:   0.62 → 0.91 (↑ 增长, 因为 STDP 锐化轴特异权重)
```

两个量都在增长, 但速率不同 → 乘积不稳定。
**如果锁定 sprouting (no new bundles), H_flow 可能稳定, CV 可能 < 0.2。**

### 结论

P_nu x H_flow **不是严格守恒量**, 但是一个**强耦合关系**。
运动偏振和信号路由确实相互约束, 只是不到"不变量"的程度。
