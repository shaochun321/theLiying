## 一、我在这个过程中犯的错误

### 事故经过

```
原始状态:  gradient 用 activate()    → w_aco = 0.9       正确
我"修正":  gradient 改为直接赋值     → w_aco = 0.000     我搞坏了
我判断:   "之前的 0.9 是假的"                            错误判断
后来发现:  直接赋值不设 pre_trace → STDP 无法工作
最终修复:  所有外部神经元加 pre_trace → w_aco = 0.647     修回来了
```

### 错误的性质

我把 `activate()` 设置 `pre_trace` 当成了 **"副作用/bug"**，
实际上这是 **必要的生理功能**——神经元发放时必须产生突触前痕迹，
否则 STDP 看不到它的活动。

我把正确的东西拆了，然后误以为"发现了真相"。

### 影响范围

- w_aco 从 0.9 → 0.000 → 0.647 的波动 **全部是我的操作造成的**
- STDP 拓扑选择性结论 **不受影响**（一直是真的）
- Q4/Q1 = 3.5 不受学习影响 **不受影响**（一直是真的）
- 沉积层"装饰性"的结论 **被我的错误遮蔽了**（修复后变成 -54%）

## 四、距离觅食差什么

### 觅食的分解

```
觅食 = 检测方位 → 运动靠近 → 摄食 → 记住位置 → 返回/探索
         ✅          ❌         ✅       ⚠️          ❌
```

### 关键缺口：运动发育（Motor Development）

系统像一个**新生儿**：
- 大脑已经学会了"食物在那个方向"（w=0.65）
- 但四肢还在随机抽搐（CPG babbling 振幅 = 0.05）
- 大脑发出的信号（gradient 输出 = 0.003）太弱，四肢不听

在生物学中，这个问题通过**运动发育**解决：
- 新生儿：强烈的反射性运动模式（Moro 反射、吮吸反射）
- 3-6 个月：自主运动开始抑制反射
- 1 岁+：自主控制完全取代反射

**系统缺少一个"成长"机制**：随着学习进展，
逐步降低 CPG 振幅 / 提升 gradient 通路的增益，
让"学到的知识"逐渐接管"先天的随机运动"。

### 具体实现方案

```
CPG 振幅 = base_amplitude × (1 - maturity)
maturity = f(总学习量, 累积 grad_to_motor 权重变化)

当 grad_to_motor 的 STDP 稳定后 → maturity 增加 → CPG 降低
→ gradient 信号相对增强 → 运动开始受学习指导
→ 觅食行为涌现
```

这不是 hack——**这就是运动发育的计算模型**：
Bernstein (1967) "The Co-ordination and Regulation of Movements"
描述的"自由度解冻"（degrees of freedom problem）。

### 这个缺口对论文意味着什么

两种策略：

**策略 A：实现运动发育，论文声称觅食涌现**
- 工作量：中等（CPG 抑制机制 + 重跑消融 + 验证 Q4/Q1 改善）
- 风险：可能有效也可能暴露新问题

**策略 B：承认限制，论文聚焦已证明的贡献**
- 标题改为"振荡门控对 STDP 自组织的影响"
- 明确声明"内部表征学习已实现，行为涌现需要运动发育机制（future work）"
- 三个干净贡献：CPG-STDP 协同（-73%）、沉积层增强（-54%）、信号依赖性
