# 全局热力学-信息审计 — 裁决

## 全局图像

```
  信号 (高熵)                    
       │                         
       ▼                         
  encoding (S=0.85 bits) ──→ column (S=0.23 bits)
       │                         │
  权重学习 (ΔI ≈ 0 bits)         │
       │                         ▼
       └──── motor (S=0.00 bits)
                                  
  能量: 200 units 输入 → 0.09 consumed (0.045%)
  信息: -0.06 bits net (几乎没学到)
  效率: ≈ 0%
  第二定律: ✅ dS_total = +19.58
```

## 6 个定量发现

### 1. 能量预算：99.95% 被存储，0.045% 被消耗

```
输入:     200.0 units (feed)
消化吸收: 178.3 units (stomach → pool)
总消耗:   0.09 units
  基础代谢: 0.081 (88.6%)
  运动:     0.000 (0%)
  未分类:   0.010 (11.4%)
```

> 系统像一个**极度高效的储能器**——几乎不耗能。
> 这也是为什么 Szilard 效率 ≈ 0：不是因为浪费，是因为**消耗本身太小**。

### 2. 信息预算：激活熵压缩了，但权重熵没变

| 量 | 变化 | 含义 |
|----|------|------|
| S_weights | 3.04 → 3.10 (+0.06) | **权重几乎没学** |
| S_encoding (act) | 1.16 → 0.77 | ✅ 编码层压缩了 |
| S_column (act) | 0.83 → 0.00 | ✅ 柱层完全压缩 |
| S_config (总) | 6.69 → 5.37 (-1.32) | ✅ 总熵下降 |

**关键**: 信息积累发生在**激活模式**（S_col: 0.83→0.00），
不是在**权重**（S_w 几乎不变）。

权重是长期记忆，激活是工作记忆。系统在 500 tick 内只建立了工作记忆，
没有固化为长期记忆。

### 3. 第二定律

```
dS_system = -1.32 bits  (系统积累信息)
dS_env    = +20.90 bits (代谢散热 → 环境)
dS_total  = +19.58 bits ✅
```

第二定律满足。系统的信息积累 (-1.32 bits) 被代谢散热 (+20.9 bits) **远远覆盖**。

### 4. 信息流拓扑：逐层压缩

```
birth:         enc=1.16 → col=0.83 → mot=0.00
learning:      enc=0.77 → col=0.23 → mot=0.00
consolidation: enc=0.77 → col=0.09 → mot=0.00
stress:        enc=0.77 → col=0.00 → mot=0.00
recovery:      enc=0.77 → col=0.00 → mot=0.00
```

**Column 层从 0.83 bits 压缩到 0.00 bits** — 完全选择性编码。
但 motor 始终 = 0 — 运动输出没有被激活。

### 5. 瓶颈：inter-layer 权重未学习

```
encoding 内权重: 56 个, S=1.10 bits (已稀疏化)
inter-layer 权重: 49 个, S=3.94 bits (几乎均匀!)
```

层间连接 (encoding→column, column→motor) 的权重**几乎没有学习**。
这就是为什么 motor = 0：信号在层间传递时被高熵权重稀释了。

### 6. 各阶段行为

| Phase | ΔS_weights | 行为 |
|-------|-----------|------|
| birth | +0.36 | 初始权重分化 |
| learning | -0.40 | **学习发生**（唯一的信息积累期）|
| consolidation | +0.44 | 权重**回退**！稳态恢复使学到的结构退化 |
| stress | -0.14 | 饥饿略微压缩权重 |
| recovery | 0.00 | 恢复但不学习 |

> [!WARNING]
> **Consolidation 阶段权重回退** (+0.44 bits) 抵消了 learning 阶段的积累 (-0.40 bits)。
> 这就是为什么 net ΔI ≈ 0。
> 系统的 homeostasis 太强——它在"遗忘"学到的东西。

## 核心结论

1. **系统是一个信息流处理器，不是信息存储器**
   - 激活空间在压缩（工作记忆）
   - 权重空间没有稳定积累（长期记忆被 homeostasis 抹掉）

2. **Jaynes-Landauer bridge 在当前系统中不可测**
   - 消耗太小（0.09 / 200 = 0.045%）
   - 信息积累太少（net -0.06 bits）
   - 两者的比值没有统计意义

3. **真正的瓶颈不是热力学，是学习动力学**
   - inter-layer 权重没学
   - homeostasis 在抹除学到的结构
   - motor 层始终沉默

> [!IMPORTANT]
> 这是一个有价值的宏观诊断。问题不在 Landauer 公式或温度定义——
> 是在系统本身的学习能力不足以积累足够的信息让 bridge 可测。
> 
> 信息时空轨迹的框架是正确的（激活压缩证明了流处理有效），
> 但需要解决 homeostasis 导致的遗忘和 inter-layer 学习不足。
