# 修复后验证：衰减律现在真的起作用

## 修复内容

**结构修复**: 在 `compute_reflex` 中添加 taxis 反射。
系统现在直接响应量源梯度：低强度→接近，高强度→回避。

## 验证结果

### 修复前 vs 修复后

```
修复前: Position difference (有源 vs 无源) = 0.000000  ← 量源无效!
修复后: Position difference (有源 vs 无源) = 1.806044  ← 量源有效!
```

### 衰减律真正影响排序了吗？

| 测试 | n配置 | 最远 | 最近 | n 起作用？ |
|---|---|---|---|---|
| Baseline | aco=1, the=2, lum=2 | acoustic (11.1) | luminous (4.6) | - |
| All n=2 | 2, 2, 2 | **thermal** (9.3) | luminous (4.7) | ✅ acoustic 不再最远 |
| Swap n | aco=2, **the=1**, lum=2 | **thermal** (8.6) | luminous (5.2) | ✅ thermal 变最远 |

> [!IMPORTANT]
> **Test 6 (交换n) 是决定性证据**: 当 thermal 获得 n=1 时，它成为最远的源。
> 这证明 n 确实通过梯度→taxis反射→运动 影响行为。

### 但仍存在混杂变量：位置不对称

```
Test 3 (全等参数 A=5, n=2, λ=0.04):
  E[L]: aco=9.3  the=7.5  lum=4.7  ← 仍然不对称！
```

全等参数下应该 E[L] ≈ 相等，但不是。原因：

1. **源位置**: acoustic=(7.5,0,0), thermal=(0,7.5,0), luminous=(0,0,7.5)
2. **CPG 振幅**: x=0.2, y=0.1, z=0.05
3. **效应**: CPG 在 z 方向运动最弱 → 系统不容易离开 luminous → luminous 最近

### 效应分解

| 因素 | 对 E[L_aco] 的贡献 |
|---|---|
| n=1 vs n=2 (衰减律) | +2.0 (从 9.3 到 11.1) |
| 源位置×CPG偏差 | +9.3 (全等参数下的基线) |

> 结论: n 贡献约 **18%** 的 L_aco 增量，位置×CPG 贡献 **82%**。
> 但 n 是唯一能**改变排序**的因素。
