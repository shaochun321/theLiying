# 门控分化: 同一母本，不同表达

## 跨学科的分化机制

你的问题——"同一母本如何合理分化"——在多个学科中有成熟答案：

### 1. 神经科学: GABA 受体亚型

**同一种神经递质 (GABA), 6 种受体亚型, 不同脑区不同表达:**

| 亚型 | 脑区 | 时间常数 | 功能 |
|------|------|---------|------|
| GABA-A α1 | 大脑皮层 | τ=6ms (快) | 快速抑制, 精确时间窗 |
| GABA-A α2 | 海马 CA1 | τ=15ms (中) | 节律生成 |
| GABA-A α3 | 丘脑网状核 | τ=25ms (慢) | 睡眠纺锤波 |
| GABA-A α5 | 海马 DG | τ=40ms (极慢) | 记忆门控 |
| GABA-B | 广泛 | τ=200ms | 慢抑制, 代谢调节 |
| GABA-C | 视网膜 | τ=3ms (极快) | 亮度对比 |

**母本相同** (都是 GABA 受体, 都做抑制),
**参数不同** (τ, threshold, 药理学特性),
**结果不同** (快速精确 vs 慢速调节 vs 记忆门控).

> [!IMPORTANT]
> 分化的不是结构，是**参数**。
> 同一个 GABA 受体蛋白，换一个亚基 (α1→α5)，
> 时间常数从 6ms 变成 40ms，功能从"快速抑制"变成"记忆门控"。

### 2. 胚胎学: 形态发生素梯度

```
Sonic Hedgehog (SHH) 浓度梯度:

  高浓度          中浓度          低浓度
  ────→          ────→          ────→
  
  运动神经元       中间神经元       感觉神经元
  
同一种细胞, 相同基因, 但位置不同 → 浓度不同 → 分化方向不同
```

**分化信号 = 梯度位置。** 细胞不需要"知道"自己该变成什么，
它只需要读取局部信号浓度，然后用同样的基因做不同的事。

### 3. 免疫学: V(D)J 重组

```
同一套基因片段: V(50种) × D(27种) × J(6种)
组合产生: 50 × 27 × 6 = 8,100 种抗体可变区
加上随机接合: 8,100 × ~1000 = 8,100,000 种
```

**母本相同 (同一套基因), 分化机制 = 组合重排。**
不需要 800 万种基因来产生 800 万种抗体——
只需要 83 个基因片段 + 一个组合规则。

### 4. 材料科学: 碳的同素异形体

```
同一种元素 (碳), 不同排列:
  金刚石 → sp3 杂化 → 最硬的天然材料
  石墨   → sp2 层状 → 最好的润滑剂
  富勒烯 → sp2 球形 → 超导体
  石墨烯 → sp2 单层 → 最强的2D材料
```

**同一种原子, 相同的电子数, 但排列/拓扑不同 → 性质完全不同。**

## 门控的分化方案

基于上述原理, 门控的**母本模板**是:

```python
class GateTemplate:
    # 母本参数 (可分化)
    tau: float          # 时间常数 (GABA亚型的核心差异)
    theta: float        # 触发阈值 (灵敏度)
    contrast_weight: float  # 对比度权重 (0=只看同步, 1=只看方差)
    polarity: int       # +1=去抑制, -1=抑制
    
    # 共享逻辑 (母本, 不改)
    def compute_gate(self, synchrony, contrast):
        signal = synchrony * (1-self.contrast_weight) + \
                 contrast * self.contrast_weight
        # RC 积分 (时间常数决定响应速度)
        self.voltage += (signal - self.voltage) / self.tau
        # MOSFET 阈值
        if self.voltage > self.theta:
            return self.polarity * (self.voltage - self.theta)
        return 0.0
```

### 5 种分化变体 (对应系统中 5 个区域)

| 区域 | 变体 | τ | θ | contrast_w | polarity | 生物学对应 |
|------|------|---|---|-----------|----------|-----------|
| **encoding** | α1-fast | 3 | 0.3 | 0.8 | +1 | 皮层快速抑制 |
| **column** | α5-memory | 20 | 0.15 | 0.5 | +1 | 海马记忆门控 |
| **inter_layer** | α2-rhythm | 8 | 0.2 | 0.6 | +1 | CA1 节律生成 |
| **motor/CPG** | αC-retina | 2 | 0.5 | 0.9 | -1 | 视网膜快速对比 |
| **sediment** | αB-slow | 50 | 0.05 | 0.3 | +1 | 代谢调节 |

```
encoding:   τ=3, θ=0.3, contrast=0.8
  → 快速响应, 高对比要求
  → 只有强信号才开启学习
  → 防止噪声触发学习

column:     τ=20, θ=0.15, contrast=0.5
  → 慢速响应, 宽容阈值
  → 积累多个 tick 的证据后才决定
  → 适合巩固 (需要确认信号持续性)

inter_layer: τ=8, θ=0.2, contrast=0.6  
  → 中速响应, 中等阈值
  → 跨层连接的质量控制
  → 这是 Park et al. MLI2 的位置

motor/CPG:   τ=2, θ=0.5, contrast=0.9
  → 极快响应, 高阈值, 几乎只看对比度
  → motor 输出需要高确信度
  → 保守: 不确定就不动

sediment:    τ=50, θ=0.05, contrast=0.3
  → 极慢响应, 极低阈值
  → 长期积累的微弱信号也能触发
  → 这是 Xin "寻找更长时空窗口" 的体现
```

### 分化的决定因素

```
分化 ≠ 手动设参
分化 = 局部统计特性 → 自动选择参数

encoding 的信号方差大 (高对比输入)
  → 自然要求 contrast_weight 高
  → 否则噪声和信号不分

column 的信号变化慢 (巩固/积累)  
  → 自然要求 τ 大
  → 否则来不及积累

motor 的错误代价高 (动作不可逆)
  → 自然要求 θ 高
  → 否则错误动作频发

sediment 的信号极弱 (Xin 残余)
  → 自然要求 θ 低
  → 否则永远检测不到
```

**参数不是人为指定的——它是该区域功能需求的自然表达。**
就像 GABA-A α1 的 τ=6ms 不是谁设计的，
而是皮层需要快速精确抑制，这个需求驱动了 α1 亚型的进化。

## 与形态发生素梯度的对应

```
项目中 "位置" 的等价物:

  encoding → 信号入口 → "高浓度 SHH"
  column   → 中间处理 → "中浓度 SHH"  
  sediment → 深层存储 → "低浓度 SHH"

"浓度" 的等价物 = 该层的平均信号方差/频率

高方差 → 需要高对比灵敏度 → α1
低方差 → 需要长积累时间 → αB
```

**分化的"形态发生素"就是每层自身的信号统计特性。**
不需要外部指令告诉每层"你应该用什么参数"——
每层根据自己的输入统计自然调整。

## 实现策略: V(D)J 式组合

不是 5 种硬编码的变体，而是 **3 个独立的参数轴的组合**:

```
轴 1 (时间): τ ∈ {2, 5, 10, 20, 50}     → 5 种
轴 2 (阈值): θ ∈ {0.05, 0.15, 0.3, 0.5} → 4 种
轴 3 (对比): cw ∈ {0.3, 0.5, 0.8, 0.9}  → 4 种

组合: 5 × 4 × 4 = 80 种可能的变体
实际使用: 5 种 (每区域一种)
```

**80 种可能中只用 5 种，其余是潜在的分化空间。**
如果未来加入新区域，只需要从 80 种中选一个，不需要改母本。

## 关键原则

> [!IMPORTANT]
> **分化的三条规则:**
> 
> 1. **母本不动**: compute_gate() 的逻辑永远不改。
>    所有变体共享同一个计算流程。
> 
> 2. **参数决定命运**: τ, θ, contrast_weight 的组合决定行为。
>    就像 GABA-A 的 α1-6 亚基决定受体动力学。
> 
> 3. **局部统计驱动分化**: 每个区域根据自身信号特性
>    选择参数，不需要全局协调。
>    就像 SHH 梯度驱动细胞分化。
> 
> **改动时:**
> - ✅ 改一个区域的参数 → 只影响该区域
> - ✅ 加一种新变体 → 从组合空间选择
> - ❌ 改母本逻辑 → 影响所有区域 → 禁止
> - ❌ 为一个区域特殊处理 → if/else → 禁止
