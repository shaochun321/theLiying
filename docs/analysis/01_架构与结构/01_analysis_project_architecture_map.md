## 一、两大层的关系

项目把神经系统分成了**两层**：

```
┌────────────────────────────────────────────┐
│         感觉-运动层（交界层）                │
│   直接接触外部世界，处理输入，产生运动输出     │
│                                            │
│   MET → 毛细胞 → 传入 → 编码 → 列 → 运动    │
│   (1层)  (2层)   (3层)  (4层) (5层) (6层)    │
│                                            │
│   + 绑定层（检测多轴共激活）                  │
│   + 环流系统（信号循环流动）                  │
│   + 结晶系统（反复的模式固化为永久连接）       │
└──────────────────┬─────────────────────────┘
                   │
                   │ 主系统的"预测残差"（Xin 张力）
                   │ = 主系统说"我猜错了多少"
                   ↓
┌────────────────────────────────────────────┐
│         影子层（内省层/沉思层）              │
│   不直接看外部世界                          │
│   只看主系统的内部预测误差                   │
│                                            │
│   影子神经元 ← 驱动 ← 主系统的 Xin          │
│   影子连接 ← STDP学习 ← 影子神经元的活动     │
│   跨轴收缩 ← 权重变化 ← 相关的Xin模式       │
│                                            │
│   ds²（闵可夫斯基间隔）= 两个轴有多"远"      │
└────────────────────────────────────────────┘
```

### 你说的"交界层"

之前的对话中你问过："不是主，而是交界层？处于外部内部信息的汇聚区域？"

**对的。** 感觉-运动层不是"主人"，它是**外部世界和内部模型的交汇处**。
就像皮肤——不是身体的主人，而是身体与环境的交界面。

- 外部信号从 MET 进来（外→内）
- 运动命令从 Motor 输出（内→外）
- 中间的编码/列层 = **联合皮层**，汇聚多种信号

### 你说的"影子层/沉思层"

影子层 = 只观察交界层"猜错了多少"的**内省系统**。
它不直接感受外部世界，只感受"我的预测和现实的差距"。

## 二、学习在哪里发生？

| 学习位置 | 学习什么 | 怎么学 | 中文解释 |
|---------|---------|--------|---------|
| **交界层的 STDP** | 连接权重（哪些神经元之间的联系变强/变弱） | 先放电的连向后放电的 = 加强，反过来 = 减弱 | 条件反射：先听到铃声后看到食物 → 铃声→食物的连接变强 |
| **交界层的结晶** | 反复出现的共激活模式 → 创建永久连接 | 共激活矩阵累积超过阈值 → 创建新的 px_ 神经元 | 练习得够多 → 技能"刻进"大脑 |
| **影子层的跨轴收缩** | 哪些感觉轴之间有关联 | Xin 模式相似 → 跨轴权重增强 | "沉思"发现：每次转头(yaw)时也在点头(pitch) |
| **成熟阶段升级** | 神经元从"新兵"变成"老兵" | PNN成熟度 + 活动电位超过阈值 | 见下面详细解释 |

### 关键区别

- **交界层学习** = 学"怎么做"（哪个输入对应哪个输出）
- **影子层学习** = 学"什么和什么有关"（哪些感觉之间存在内在联系）
- **成熟升级** = 学"够了，可以锁定了"（从灵活变为稳固）

## 三、新兵升级系统（成熟阶段）

这就是你说的"新兵升级到什么什么"：

```
阶段 0: 新兵（Spine）
   │  学习率高（0.18），可塑性强
   │  像刚入伍的新兵——什么都在学
   │
   │  升级条件：PNN成熟度 > 0.3 且 活动电位 > 50
   ↓
阶段 1: 柱（Column）
   │  学习率中（0.01），开始稳定
   │  像经过训练的士兵——已经掌握基本技能
   │
   │  升级条件：PNN成熟度 > 0.7 且 活动电位 > 500
   ↓
阶段 2: 区域（Area）
      学习率低（0.001），基本锁定
      像老兵——技能固化，不再大幅改变
```

### 用大白话说

| 阶段 | 中文 | 比喻 | 学习速度 | 可塑性 |
|------|------|------|---------|--------|
| Spine | 树突棘/新兵 | 刚出生的婴儿 | 快（0.18） | 很高，容易改变 |
| Column | 柱/士兵 | 学会走路的小孩 | 中（0.01） | 中等，稳定但还能学 |
| Area | 区域/老兵 | 成年人 | 慢（0.001） | 低，基本固定 |

### 什么驱动升级？

两个条件同时满足：

1. **PNN成熟度**（围神经网络成熟度）
   - PNN = 围绕神经元的保护性蛋白网络
   - 就像混凝土逐渐凝固——慢慢包裹住神经元
   - 由 ECM（细胞外基质）系统管理
   - tau = 200秒 → 非常慢地增长

2. **活动电位 Φ**（potential_phi）
   - 神经元累积的总活动量
   - 活动越多 → Φ 越大 → 说明这个神经元被用得够多了

**两个都够高** = 这个神经元已经"练习够了"且"保护层已经长好了" → 可以升级。

## 四、所有结构的完整清单

### 交界层（感觉-运动层）的组件

| 组件 | 中文 | 干什么的 |
|------|------|---------|
| MET → HairCell → Afferent | 前庭链（1-3层） | 把机械运动变成神经信号 |
| Encoding（编码层，4层） | 每轴2个神经元（规则/不规则） | 把感觉信号编码为内部表示 |
| Column（列层，5层） | 每轴1个神经元 | 多通道汇聚，做"决策" |
| Motor（运动层，6层） | 3个运动神经元(x,y,z) | 产生运动命令 |
| Binding（绑定层） | 检测多轴共激活 | yaw+pitch 同时活跃 → 绑定节点激活 |
| Lateral Inhibition（侧向抑制） | 列层之间互相竞争 | 最强的轴压制其他轴 |
| Oscillator（振荡器） | 每轴一个 Kuramoto 振荡器 | 类似 CPG（中枢模式发生器） |
| ECM（细胞外基质） | 每层一个 | 管理温度、离子缓冲、PNN成熟 |
| Vascular（血管系统） | 全局一个 | 运送能量、带走热量 |
| Damper（阻尼器） | 每轴一个 | 防止振荡过大 |
| Router（路由器） | 编码→列的动态路由 | 根据活动相关性调整路由强度 |

### 影子层的组件

| 组件 | 中文 | 干什么的 |
|------|------|---------|
| Shadow Neuron（影子神经元） | 每个主系统神经元的影子副本 | 记录主系统的活动 |
| Shadow Bundle（影子连接） | 轴内连接 + 跨轴连接 | 轴内 = 编码→列的影子；跨轴 = yaw↔pitch 等 |
| ds²（闵可夫斯基间隔） | 两个轴之间的"时空距离" | 衡量两个轴到底有多不同 |
| 收缩 | 跨轴权重增强 | 相关的轴"靠拢"在影子空间中 |
| 下葬系统 | 弱化到消失的连接被记录 | 弃置的路径留下痕迹 |

### 过程/机制

| 机制 | 中文 | 涉及哪些组件 |
|------|------|------------|
| STDP（脉冲时序可塑性） | 谁先谁后决定连接强弱 | 所有 SynapticBundle |
| Maturation（成熟升级） | 新兵→柱→区域 | ECM 的 PNN + 神经元的 Φ |
| Crystallization（结晶） | 频繁的共激活模式固化 | 共激活矩阵 + 结晶条件 |
| Circulation（环流） | 信号在层间循环流动 | P路径（正向）、R路径（反射） |
| Xin（心/张力） | 每个连接的预测残差 | SynapticBundle 的 xin_tension |
| Homeostasis（稳态调节） | 能量低时自动降低活动 | 运动层的阈值适应 + 突触缩放 |

## 六、回答你的问题

### "学习的工作交给哪里？"

**三个地方同时在学，但学不同的东西：**

| 学什么 | 在哪学 | 怎么学 |
|--------|--------|--------|
| 输入→输出的映射 | 交界层的 STDP | 先后放电 → 权重变化 |
| 哪些感觉之间有关联 | 影子层的收缩 | Xin 相关 → 跨轴权重增强 |
| 技能固化 | 结晶 + 成熟升级 | 共激活累积 → px_ 创建；PNN 包裹 → 降低可塑性 |

### "新兵升级到什么什么？"

**新兵(Spine) → 柱(Column) → 区域(Area)**

就像一个人从婴儿到小孩到成人：学习速度越来越慢，技能越来越固定。
升级条件：(1) 保护层长够厚 (PNN > 阈值)，(2) 活动量够大 (Φ > 阈值)。

### "特殊的分化区域？"

目前代码中有这些特殊区域：
- **绑定层**（Binding）：只检测多轴共激活，不是普通的层
- **影子层**（Shadow）：不直接参与感觉-运动，只做内省
- **ECM/PNN**（细胞外基质/围神经网络）：管理成熟和保护，不是信号处理
- **血管系统**（Vascular）：全局能量供应，不是神经计算
