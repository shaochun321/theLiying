# nexus_v1 链路解剖 — 一根轴的完整信号路径

## "轴"是什么

"轴"(axis) = 一个感知方向。前庭系统感知 6 个方向的运动：

| 轴 ID | 中文 | 对应生物结构 | 感知什么 |
|-------|------|------------|---------|
| `yaw` | 偏航 | 水平半规管 | 左右转头 |
| `pitch` | 俯仰 | 前半规管 | 点头 |
| `roll` | 翻滚 | 后半规管 | 歪头 |
| `oto_x` | 前后 | 椭圆囊 | 前后加速（刹车/加速） |
| `oto_y` | 左右 | 椭圆囊 | 左右加速（转弯离心力） |
| `oto_z` | 上下 | 球囊 | 重力 / 上下加速（电梯） |

**每个轴有一条独立的信号链路。** 6 个轴 = 6 条平行链路，结构完全相同，只是方向不同。

## 单轴汇总

| 项目 | 数量 | 说明 |
|------|------|------|
| **元单元 (Neuron)** | **7** | 1 MET + 1 毛细胞 + 2 传入 + 2 编码 + 1 柱 |
| **管线束 (Bundle)** | **4** | MET→HC, HC→Aff, Aff→Enc(×2), Enc→Col |
| **突触 (Memristor)** | **6** | 1+2+2+2=7 (实际 6，aff→enc 各 1) |
| **MOSFET** | **9** | MET(1) + HC(4) + Aff(2) + Enc(2) |
| **Capacitor** | **8** | MET(1) + HC(2) + Aff(2) + Enc(2) + Col(1) |
| **PowerRail** | **7** | 每个元单元 1 个 |

| **共享突触** | **18** | 6 柱 × 3 motor |
| **Motor 元单元** | **3** | move_x, move_y, move_z |

## 极度简化的后果

### 生物系统 vs nexus_v1

在生物中，每个半规管有 **~3000 个毛细胞**，每个毛细胞参数略有不同。nexus_v1 每个轴只有 **1 个毛细胞**。

> [!WARNING]
> 1 个毛细胞 = 1 个数字。3000 个毛细胞 = 3000 个数字。
> 用 1 个数字代表 3000 个数字，意味着：

**丢失 1: 群体编码 (Population Coding)**
```
生物: 3000 个毛细胞，每个灵敏度略不同
     → 弱信号: 只有最敏感的 100 个响应 → 精确检测弱信号
     → 强信号: 全部 3000 个响应 → 宽动态范围

nexus_v1: 1 个毛细胞
     → 弱信号: 要么过阈值（全有），要么不过（全无）
     → 没有中间状态，丧失精度
```

**丢失 2: 噪声平均 (Noise Averaging)**
```
生物: 每个细胞都有独立噪声
     → 3000 个细胞取平均 → 噪声降低 √3000 ≈ 55 倍
     → 信噪比极高

nexus_v1: 1 个细胞
     → 噪声 = 信号 → 信噪比 = 1
     → 这就是之前测试中耳石被噪声淹没的原因之一
```

**丢失 3: 频率地图 (Tonotopy)**
```
生物: 不同毛细胞调谐到不同频率
     → 高频运动和低频运动被不同的细胞群编码
     → 可以分辨"快速抖动"和"缓慢转动"

nexus_v1: 1 个毛细胞只有 1 个频率响应
     → 所有频率看起来一样
```

### 为什么不回升（增加数量）

可以回升。结构上很简单——把每个轴的 1 个毛细胞改成 N 个：

```python
# 当前: 1 个
met_neurons["yaw"] = Neuron(_met_config("yaw"))

# 回升到 N=10:
for i in range(10):
    met_neurons[f"yaw_{i}"] = Neuron(_met_config(f"yaw_{i}", 
        threshold=0.01 + i*0.005))  # 每个阈值略不同
```

**但现在不应该回升的原因：**

1. **先让 1 个能通** — 目前信号只到了第 3 层（传入），编码层还是 DEAD。10 个也是 DEAD。
2. **先验证链路** — 用 1 个元单元验证信号从输入到输出是否贯通。
3. **回升是参数问题，不是架构问题** — 架构已经支持 N 个，只是循环 N 次。

> [!TIP]
> 优先级：先打通 1 条链路（6 层全亮），再讨论每层放多少个。

## 术语对照表

| 英文 | 中文 | 是什么 |
|------|------|--------|
| **Neuron** / 元单元 | 神经元 | 由多个半导体组件构成的计算单元 |
| **MOSFET** | 金属氧化物场效应管 | 阈值门控器（超过阈值才通电） |
| **Capacitor** | 电容器 | 电荷积累器（膜电位 = Q/C） |
| **PowerRail** | 电源轨 | 能量供给（限流器，防止过载） |
| **Memristor** | 忆阻器 | 可变电阻（= 突触权重） |
| **Bundle** / 管线束 | 突触束 | 一组 Memristor 的集合，连接两层元单元 |
| **STDP** | 脉冲时间依赖可塑性 | 学习规则：先后顺序决定权重增减 |
| **Spike** | 脉冲 | 元单元的"开火"事件（电压超峰值→重置） |
| **Activation** | 激活值 | 元单元的输出信号（连续或脉冲） |
| **Release rate** | 释放率 | Ca²⁺ 门控后的输出（噪声门） |
| **Adaptation** | 适应 | spike 后增加阈值，减少后续放电率 |
| **Regular afferent** | 规则传入 | 适应弱 → 持续放电 → 编码重力(DC) |
| **Irregular afferent** | 不规则传入 | 适应强 → 变化时才放电 → 编码运动(AC) |
