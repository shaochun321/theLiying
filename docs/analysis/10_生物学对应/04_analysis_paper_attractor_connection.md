# 论文结论与电路分化的连接（全中文版）

## 三篇论文验证了什么

| 论文 | 核心结论 | 对应的数学对象 |
|------|---------|--------------|
| **论文1** | 衰减指数 n 因果决定行为偏好 (p<0.002) | **序关系**: n₁<n₂ 意味着 平衡距离₁ > 平衡距离₂ |
| **论文2** | 衰减指数响应曲线：n → 平衡距离 的定量映射，三个区域 | **分岔**：三个区域的边界由噪声底限定义 |
| **论文3** | 介质物理 (质量,刚度,阻尼) 是第三控制轴 + 诺特异常 → 近似守恒 | **吸引子**：环流变化率 → 0 |

> [!IMPORTANT]
> 三篇论文验证的是 **练习引擎**（粒子系统+力场+介质+积分器+反射）的闭环行为。
> 它们不涉及赫布电路/前庭链。但论文发现的数学结构应该在赫布电路中也存在，
> 因为赫布电路是练习引擎的**神经基底层替代品**。

## 论文发现的三种动力学结构

### 1. 固定点吸引子（弹珠停在碗底）

论文2的衰减指数响应曲线公式：
```
平衡距离 L*(n) = (nAα√N/σ)^{1/(n+1)}
```

在第一区域 (n≤1)，系统收敛到稳定的平衡距离 L*。
弹珠最终停在碗底不动了——这就是**固定点吸引子**。

**nexus_v1 对应**：脉冲时序可塑性(STDP)权重收敛。
编码→列 的连接权重从 0.10 → 0.100265 然后不再变化。
50000 步后所有层的能量趋势 = 0，说明系统"停在碗底"了。

### 2. 极限环（钟摆来回摆）

论文3 §5.5 时空环流：
```
相位一致性 κ = 0.86（探索阶段）
环流周期 = 3 步
```

编码层的神经元按固定顺序轮流激活，形成闭合回路。
像钟摆一样——不停在一个点，而是反复走同一条路。

**nexus_v1 对应**：运动层的脉冲频率适应振荡。
适应电流 w_adapt 充电 → 放电 → w_adapt 放电 → 恢复 → 再放电...
当前 w_adapt 稳定在 0.05，脉冲间隔≈90步（约 11 赫兹）。
这是在 (膜电压, 适应电流) 相平面中的**极限环**。

### 3. 分岔/相变（碗突然变成山坡）

论文2 的三个区域：
```
n < 1:    梯度控制（弹珠停在碗底=固定点）
1 < n < 2.5: 反射饱和（碗变浅了但弹珠还能停住）
n > 2.5:  噪声主导（碗消失了，弹珠随机滚动）
```

当参数 n 越过临界值 n*≈2.5，平衡点**不再存在**。
碗突然变成了平面——弹珠不知道该往哪里去。
这就是**分岔**：一个参数的微小改变导致系统行为的质变。

**nexus_v1 对应**：我们刚修复的 Bug 就是一个分岔的例子！
- tau_w=50s 时：运动神经元放 18 次电后永久沉默（固定点：死亡）
- tau_w=1.0s 时：运动神经元持续以 11 赫兹放电（极限环：活着）
- 在 tau_w 从 50 变到 1 的过程中，某个临界值处发生了**分岔**

## 如何用论文的思路指导分化

### 之前的做法（反应式，一个一个修 Bug）

```
发现症状 → 追踪参数 → 修改参数 → 看有没有新症状 → 重复
```

这就像"哪里疼治哪里"，永远在追 Bug。

### 论文指向的做法（预测式，先画地图再走路）

```
第一步：画相图（地图）
  - 运动层：记录 (膜电压, 适应电流) 的轨迹 → 看是碗底还是钟摆
  - 列层：记录 (膜电压, STDP权重) → 看权重收敛到哪里
  - 影子层：记录 (跨轴权重, 能量) → 看收缩走向哪里

第二步：画分岔图（标注哪些区域安全）
  - 横轴：突触增益 ∈ [1, 10]
  - 纵轴：适应时间常数 ∈ [0.1, 10]
  - 颜色：这个参数组合下系统是什么行为
    🟢 极限环（持续放电）= 安全
    🔴 固定点在零（沉默/死亡）= 危险
    🟡 混沌（不规则放电）= 需要检查

第三步：选择参数区域
  - 在分岔图上找 🟢 区域的中心（最稳定的位置）
  - 这就是"分化后"的参数——不是试出来的，是计算出来的
```

> [!NOTE]
> 这和论文2做的事情完全一样：
> 论文2 先推导了公式 L*(n)，再用实验验证。
> 不是一个一个试 n 看效果，而是先有数学地图。

## 你的问题的直接回答

> "是否能为项目找到震荡范围？"

**是的。** 论文验证了三种动力学结构在 PracticeEngine 中存在。
nexus_v1 赫布电路中应该也存在相同的结构：
- 权重收敛 = 固定点吸引子
- 运动层脉冲振荡 = 极限环
- 参数临界值处的行为质变 = 分岔

"振荡范围"具体来说就是**极限环在参数空间中的存在域**——
在哪些（突触增益, 适应时间常数, 能量恢复速率...）的组合下，
运动神经元会维持稳定的周期性放电，而不是死亡或爆炸。

> "对应的项目构造是什么？"

| 数学对象 | 项目构造 | 当前状态 |
|---------|---------|---------|
| 固定点 | STDP 权重收敛（编码→列，列→运动） | ✅ 已观测到 |
| 极限环 | 运动层 (膜电压, 适应电流) 振荡 | ✅ 修复后存在 |
| 分岔 | 参数边界（如 tau_w 的临界值） | ⚠️ 需要画分岔图 |
| 诺特异常 | 全系统能量趋势 → 0 | ✅ 当前 E_trend ≈ 0 |
| 环流度量 | 信号在层间的循环强度 | ❌ 尚未在电路中实现 |
