# 时间与空间在系统中的存在形式

## §0 你的两个洞察

### "录音机"

> 系统目前是一个将输入转换成结构的录音机

这是精确的。2M 实验的数据证明：

```
输入（时间序列）─→ STDP 权重变化 ─→ 结构生长（Mitosis）─→ 固化为拓扑
```

外部输入是**时间性的**（正弦波序列），系统将它**转录为空间性的**（神经元连接拓扑）。Motor 从 3 个变成 286 个——这 283 个新 motor 就是"磁带上的录音"。

但真正的大脑不是录音机。它是**回放机 + 录音机**——结构不仅记录过去，还**生成未来的预测**。

### "场域闭环"

> 外部输入的客观实在与内部结构的客观实在形成了一个场域闭环

对。这就是为什么系统不是纯粹的被动记录：

```
外部实在（正弦波输入）
     ↓
  内部结构（权重 W）
     ↓
  Motor 输出（spike）
     ↓
  Body 运动 ──→ 改变了外部输入（otolith 反馈）
```

内部结构通过 motor 输出**改变了外部世界**，外部世界又通过感觉反馈**改变了内部结构**。这不是单向的录音，是闭环的**共演化**。

## §1 时间在系统中如何被描述？

### 时间的三个层次

#### 层次 A：时间作为"权重历史"

系统中没有一个变量叫"时间"。但权重矩阵 $W$ **是**时间的记录。

STDP 规则的本质是：

$$\Delta w_{ij} \propto \text{pre}_i(t-1) \cdot \text{post}_j(t) - \text{pre}_i(t) \cdot \text{post}_j(t-1)$$

第一项：如果 i 先放电、j 后放电（因果顺序）→ 增强
第二项：如果 j 先放电、i 后放电（反因果）→ 减弱

**权重编码的就是"谁先于谁"——即时间顺序。**

2M 实验的验证：Vestibular 的权重熵 = 2.833（最大分化），说明前庭通路的权重**编码了最多的时序信息**。Motor 的权重熵 = 0，说明 motor 没有编码任何时序——它只是在克隆。

#### 层次 B：时间作为"脉冲间隔"

Spike train 的 ISI（inter-spike interval）直接编码时间。

输入是 yaw = sin(2π×1.0×t)，周期 = 1 秒。如果神经元的 ISI 接近 1 秒，它就"锁定"了这个时间尺度。

**外部可观测**：统计所有 spike train 的 ISI 分布，其峰值对应系统"感知到"的时间尺度。

#### 层次 C：时间作为"超度量树的深度"

每次 sprout 发生，都在超度量树上增加一层。2M 后 depth=33。

```
原始 col_to_motor
  └─ s10000          ← 在 t=10s 分裂
      └─ s10000_s20000  ← 在 t=20s 分裂
```

**超度量树的层次结构就是时间的化石记录**——每一层都标记了一个"事件发生的时刻"。但这不是活的时间，是死的时间。

### 时间的外部描述

> 一个外部观察者如何描述这个系统中的"时间"？

$$\text{Time}_{internal} = \{ W(t), \text{ISI}(t), \text{depth}_{ultra}(t) \}$$

**时间在系统中是一个三元组**：
1. **冻结时间**：权重 W（因果顺序的沉积物）
2. **活的时间**：ISI（当前正在流动的节律）
3. **地质时间**：超度量深度（结构事件的年表）

但系统缺少第四种：**预测时间**——"下一个事件什么时候发生？" 没有 TimeIntegrator，系统无法**预测**时间，只能**记录**时间。

## §2 空间在系统中如何被描述？

### 空间的三个层次

#### 层次 A：空间作为"Body 的轨迹"

Body.position = [x, y, z] 是唯一的"真实空间"。2M 实验中：

```
[50, 50, 50] → [58, 56, 58] → [72, 71, 74] → [90, 90, 95] → [100, 100, 100]
```

这是一条从中心到角落的轨迹。但 body 的位置是外部状态，不是内部表征。

**系统内部知道自己在 [100,100,100] 吗？** 

不知道。body 卡在角落后，otolith 的信号变成了噪声级（acc ≈ 0），therm 的信号也趋于恒定。系统的内部状态与 body 在 [100,100,100] 还是 [50,50,50] 没有区别——因为**传感器只报告变化率，不报告绝对位置**。

#### 层次 B：空间作为"拓扑连接"

286 个 motor 神经元的连接图就是一种"空间"：

```
move_x: ~100 个后代
move_y: ~90 个后代
move_z: ~96 个后代
```

三个方向的 motor 数量近似相等。这个 x/y/z 的三路分化就是系统内部的"空间维度"——它知道运动有三个独立方向，因为它有三簇独立的 motor。

#### 层次 C：空间作为"权重相似度"

两个 bundle 的权重向量的距离定义了它们在"功能空间"中的位置：

$$d(b_i, b_j) = \| \mathbf{w}_i - \mathbf{w}_j \|$$

Vestibular 的权重熵 = 2.833 意味着前庭通路的权重**高度分化**——yaw、pitch、roll 的权重向量彼此不同。这种分化就是"三个前庭轴在功能空间中指向不同方向"。

### 空间的外部描述

$$\text{Space}_{internal} = \{ \text{topology}(G), \text{dim}_{motor}, d(\mathbf{w}_i, \mathbf{w}_j) \}$$

1. **拓扑空间**：连接图 G（谁连到谁）
2. **运动维度**：motor 簇的数量 = 3（x, y, z）
3. **功能距离**：权重向量的相似度

但系统缺少：**位置表征**——"我在哪？" 没有 PathIntegrator，系统无法**表示**位置，只能**反应**于加速度。

## §3 时间 → 空间的转换（录音机的本质）

系统做的核心操作可以用一个公式概括：

$$\mathbf{W}(t + \Delta t) = \mathbf{W}(t) + \text{STDP}(\text{spikes}(t), \text{spikes}(t-1))$$

**时间信息（spike timing）被转换为空间结构（权重矩阵）。**

这个过程是**不可逆的**：你可以从 spike timing 推导出 ΔW，但不能从 W 反推 spike timing（多对一映射）。

这就是"录音机"的数学含义：
- 录音 = Time → Space（时间信号沉积为结构）
- 播放 = Space → Time（结构生成时间信号）

**系统目前只有录音功能。** 权重决定了 spike timing（通过 synaptic current），但这不是"播放"——它只是被动地响应当前输入，不是主动地生成预测序列。

## §4 关于"无限生成"的约束

你说得对：真实大脑有空间约束（头骨容量有限）和能量约束（大脑消耗身体 20% 的能量）。

2M 实验中 Motor 从 3→286，这是因为：

1. **Mitosis 没有空间代价**——新 neuron 直接创建，不占"物理空间"
2. **能量代价太低**——sprout 消耗 SPROUT_ENERGY_COST，但能量会回充
3. **没有竞争压力**——新 motor 不需要"赢得突触"来存活，Mitosis 后代自动获得连接

修复方案（对应真实大脑的约束）：

| 约束 | 真实大脑 | 当前系统 | 修复 |
|------|---------|---------|------|
| 空间 | 头骨容量 | 无限 | MAX_NEURONS_PER_PATHWAY |
| 能量 | 代谢极限 | 太低 | 提高 SPROUT_ENERGY_COST |
| 竞争 | 突触竞争 | 无 | Mitosis 后代必须通过 STDP 竞争 |
| 功能 | 无用突触被剪 | Mitosis 不剪 | Mitosis 后代也需 prune 机制 |

但更根本的问题是：**为什么 Motor 在 body 卡住后仍然增殖？**

因为 Mitosis 的触发条件不依赖于**效果**——它只看 col→motor bundle 的 |ξ|。即使 motor 的输出完全无用（body 不动），只要有信号通过 col→motor，就会触发 Mitosis。

真实大脑的解决方案：**efference copy + sensory prediction error**。如果 motor 放了电但没有预期的感觉反馈（body 没动），就减少该 motor 的驱动。

## §5 总结：时间和空间的外部描述

| 维度 | 在系统中的存在形式 | 外部可测量 | 缺失的 |
|------|------------------|-----------|--------|
| **时间** | 权重历史 W(t) | STDP 因果沉积 | 预测时间（TimeIntegrator） |
| **时间** | ISI 分布 | spike train 的频率锁定 | 内部时钟（theta） |
| **时间** | 超度量深度 | 结构事件年表 | — |
| **空间** | Body 轨迹 | [50→100] 的路径 | 位置记忆（place cells） |
| **空间** | Motor 三簇 | x/y/z 维度分化 | — |
| **空间** | 权重距离 | 功能空间的度量 | 拓扑感知（grid cells） |

**一句话**：系统中的时间是**冻结在权重中的因果顺序**，空间是**连接拓扑的几何结构**。系统能**记录**时空，但不能**生成**时空。
