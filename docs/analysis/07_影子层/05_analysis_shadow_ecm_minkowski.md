# 影子层的基质与闵可夫斯基时空

## 问题 1：影子层有没有基质系统（ECM）？

### 需要。理由如下

影子层由真实 Neuron + Bundle 实例构成。Neuron 的 step() 产生热（`heat_output`），Bundle 的 learn() 改变权重。如果没有 ECM：

- 热无处散 → 温度无限上升 → 数值发散
- PNN 不成熟 → 学习永远不冻结 → 收缩永远不停

但影子层的 ECM **不等于** 感觉-运动层的 ECM。

### 影子层 ECM 的特殊性

| 属性 | 感觉-运动层 ECM | 影子层 ECM | 理由 |
|------|----------------|-----------|------|
| 热容 | 1.0 | **5.0**（更高） | 影子层运算更慢，热积累更缓慢 |
| PNN 成熟速率 | 0.0001/step | **0.00005**（更慢） | 影子层需要更长时间才能冻结——它的学习窗口比主系统长 |
| 初始温度 | 0.1 | 0.1 | 相同 |
| 离子浓度 | 有 | **简化** | 影子不直接参与感觉转导，离子动力学次要 |

**关键**：影子层的 PNN 成熟**独立于**感觉-运动层。两层的学习窗口关闭时间不同——感觉-运动层可能已经结晶，影子层仍在重塑。这对应生物学中**皮层可塑性比海马更长**的事实。

### 影子层需要自己的 Vascular 吗？

需要。影子层有热产生 → 需要散热。但流量更低（活动更低）。

可以用现有的 `create_brainstem_vascular()` 配置一个低流量版本：

```python
shadow_vascular = VascularCooling(
    base_flow=0.2,    # 主系统的 1/5
    nvc_gain=0.5,     # 更弱的血管-神经耦合
    heat_capacity=4.0,
    energy_efficiency=0.3,
)
```

## 问题 2：闵可夫斯基时空与影子层

### 为什么这个问题自然地出现

影子层和感觉-运动层有**不同的度规**：

- 感觉-运动层：度规由硬件拓扑决定（MET→HC→Aff→Enc→Col→Mot），**刚性**
- 影子层：度规随 Xin 相关性收缩，**动态**

两个度规之间的关系——一个刚性、一个可变——恰好是相对论要处理的核心问题。

### 神经"光锥"

每个 bundle 有传播延迟（由 RC 时间常数 τ = C × R 决定）和衰减（由权重 w 决定）。定义**神经信号的最大传播速度**：

$$c_{\text{neural}} = \frac{1}{\tau_{\min}} = \frac{1}{\min_k (C_k \times R_k)}$$

从神经元 $i$ 在时刻 $t$ 发出信号，到时刻 $t + \Delta t$，**能影响到哪些神经元**？

这定义了一个**神经光锥**：

```
                    future
                     /|\
                    / | \
                   /  |  \
                  / causal \
         --------i----------  ← "现在"
                  \  |  /
                   \ | /
                    \|/
                    past
```

光锥内的神经元 = **因果可达**（信号能在 Δt 内传到）
光锥外的神经元 = **因果隔离**（信号无法到达）

### 感觉-运动层的光锥

在感觉-运动层中，拓扑是线性链：MET→HC→Aff→Enc→Col→Mot

从 Enc_yaw 出发，1 步内能到达的只有 Col_yaw（直接 bundle）。
3 步内能到达的包括 Col→Mot 和可能的反馈路径。

**跨轴**：Enc_yaw 到 Enc_pitch 没有直接 bundle → 它们是**类空间隔**（spacelike separated）。在感觉-运动层的因果结构中，yaw 和 pitch 的 encoding 层之间**没有直接因果关系**。

### 影子层的光锥变形

影子层的度规收缩改变了因果结构：

```
感觉-运动层:                    影子层（收缩后）:
  Enc_yaw ····· Enc_pitch        Shadow_yaw ───── Shadow_pitch
  (类空间隔, 无因果链接)            (类时间隔, 有因果链接!)
```

当影子层的跨轴 bundle 权重从 0.001 增强到 0.18：
- **传播延迟不变**（同样的 RC 常数）
- **衰减大幅减小**（权重增加 = 信号不再被衰减到零）
- 原来信号到不了的地方现在到得了

**因此：影子层的度规收缩 = 类空间隔 → 类时间隔的转变 = 光锥展宽。**

### 闵可夫斯基间隔

定义**神经时空间隔**（外部审计的描述工具）：

$$ds^2 = -(c_{\text{neural}} \cdot \Delta t)^2 + d_{\text{structural}}^2$$

其中：
- $c_{\text{neural}} \cdot \Delta t$ = 信号在 $\Delta t$ 内能走的最远距离
- $d_{\text{structural}}$ = 两个神经元之间的结构距离（bundle 链长度 × 衰减）

| 情况 | $ds^2$ | 含义 |
|------|--------|------|
| $ds^2 < 0$（类时间） | 信号能到达 | 因果连接 |
| $ds^2 = 0$（类光） | 信号恰好到达 | 因果边界 |
| $ds^2 > 0$（类空间） | 信号到不了 | 因果隔离 |

**感觉-运动层**：跨轴对 $ds^2 > 0$（类空间）——yaw 和 pitch 的 encoding 因果隔离

**影子层（收缩后）**：跨轴对 $ds^2 < 0$（类时间）——因果连接出现！

### "洛伦兹因子"

两个度规之间的比值：

$$\gamma_{ij} = \frac{d_{\text{main}}(i,j)}{d_{\text{shadow}}(i,j)}$$

当 $\gamma > 1$：影子层中距离更近 = 影子层的"尺缩"
当 $\gamma = 1$：两个度规一致 = 无收缩
当 $\gamma \to \infty$：影子层中距离趋零 = 完全耦合

同时，影子层的时间膨胀：

$$\tau_{\text{shadow}} = k \cdot \tau_{\text{main}}$$

**影子层中 $\gamma > 1$ 且 $k > 1$**——**空间收缩 + 时间膨胀**——这正是相对论中运动观察者的特征！

### 这意味着什么？

影子层就像一个**运动中的参考系**：
- 它看到的空间距离比"静止"的感觉-运动层更短（Xin 相关的节点"靠近"了）
- 它经历的时间比感觉-运动层更慢（每 k 步更新一次）

**类空间隔 → 类时间隔的转变 = 新的因果路径涌现。**

这些新的因果路径——在感觉-运动层中不存在、只在影子层中存在的因果连接——就是你在构想中说的**"深层跨链接"**。

### 但这是外部描述，不是内部机制

> [!IMPORTANT]
> 闵可夫斯基间隔 $ds^2$ 是**外部审计工具**描述影子层行为的数学语言。
> 
> 系统内部不知道也不需要知道什么是"类时间隔"。它只有：
> - 结构（哪些 bundle 连接哪些 neuron）
> - 物理（STDP 改变权重、能量消耗、热产生）
> 
> 光锥展宽是**涌现的结果**，不是编程的输入。
> 
> CirculationDetector 可以计算 $ds^2$ 来**审计**这种涌现——但它不会告诉系统该怎么做。

### 实施建议

在 `CirculationDetector` 或单独的审计工具中，添加：

```python
def compute_neural_interval(self, circuit, i, j, dt):
    """Compute Minkowski-like neural spacetime interval.
    
    ds² < 0: timelike (causal)
    ds² > 0: spacelike (acausal)
    """
    c_neural = 1.0 / min_tau  # max signal speed
    d_struct = structural_distance(i, j)  # bundle chain distance
    ds2 = -(c_neural * dt)**2 + d_struct**2
    return ds2
```

然后对比感觉-运动层和影子层的 $ds^2$ 分布：如果影子层中有更多的类时间对 → 收缩正在创造新的因果路径。

## 总结

| 问题 | 答案 |
|------|------|
| 影子层有 ECM？ | **有**，独立的 ECM + Vascular，参数更慢（热容高、PNN 慢） |
| 闵可夫斯基时空？ | **可以**用作外部审计框架，描述收缩造成的因果拓扑变化 |
| 内部用相对论？ | **不**。内部只有结构+物理。光锥展宽是涌现结果，不是编程输入 |
| 实施方式？ | ECM = 组件配置；$ds^2$ = CirculationDetector 的审计输出 |
