# P/R 关系与 Xin 递归特性分析

> 结合 2026.5.24 变分批判文档 + 用户的递归链理论

---

## §1 批判文档的核心要点

该批判假设目标是建立**基于变分的数学模型**，然后指出三个断层：

```
断层1: 离散系统不能直接用连续诺特定理
断层2: 耗散系统不能用保守变分 S = ∫(T-V)dt
断层3: 分线性门控 (max(0,x)) 让变分/勒让德崩溃
```

批判给出的修补方案：
```
→ 离散变分原理 (Variational Integrators)
→ 瑞利耗散函数 R = ½ Σ γ_i ȧ_i²
→ 噪声光滑化: ReLU → Softplus (用噪声磨平不可微点)
```

**但这整个方案的前提是：系统应该用变分来描述。**

如果按用户的理论框架——T/O/P/R/Xin 递归本身就是数学工具，
不需要外挂变分——那么批判的三个断层变成了：

```
断层1（诺特）:
  批判: 离散 → 不能用连续诺特
  回应: 项目不需要诺特推导守恒量
        环流守恒是结构的直接行为 (C-001.2)
        守恒量 = 环流稳定时的 ρ
        不需要对称性推导 → 不需要诺特

断层2（耗散）:
  批判: 耗散系统需要瑞利耗散函数
  回应: 耗散 = RC 泄漏 = 代码中的 leak()
        Xin 已经包含了耗散的效果:
        predicted - actual 的差距中包含了泄漏造成的衰减
        不需要在拉格朗日中显式写耗散项

断层3（分线性）:
  批判: max(0,x) 不可微 → 变分崩溃
  回应: 如果不用变分，分线性不是问题
        MOSFET 的分线性特性 = 度规的退化边界
        这不是"近似"——这是系统的物理特征
        用噪声磨平它 = 人为修改系统来适应数学工具
        应该是数学工具适应系统，不是反过来
```

## §2 但批判的两个公理漏洞是有效的

批判的两个公理问题**不依赖于变分假设**，是独立有效的：

### 2.1 度规不对称、不正定

```
当前: g_ij = w_cross(i,j)
问题: w_cross 由 STDP/BCM 更新 → 不保证 w(i,j) = w(j,i)
                                → 不保证正定

如果不对称: ds² 不是真正的距离（从 A 到 B ≠ 从 B 到 A）
如果不正定: ds² 可能为负（"距离"为虚数）

修正: g_ij = ½(w_ij + w_ji) + g_baseline
  对称化: ½(w + wᵀ)
  基线保证正定: g_baseline > 0 (来自基线活动)
```

### 2.2 |a_i| 不是微分 da_i

```
当前: ds² = Σ w_cross × |a_i| × |a_j|
问题: |a_i| 是绝对值（恒正），不是偏移量 δa_i

  |a_i| → 只有大小，没有方向
  δa_i = a_i - baseline → 有正有负 → 有方向

修正: ds² = Σ g_ij × δa_i × δa_j
  其中 δa_i = a_i - a_i_baseline

  这与用户说的"输入让基线偏移"完全吻合
  基线 = a_baseline
  信号 = δa = 偏移
  度规 = 偏移的二次型
```

## §3 P/R 在项目中的数学关系

### 3.1 当前代码中的 P 和 R

```
P (Predict):
  bundle.compute_xin():
    predicted = Σ W_ij × a_i(t-1)    ← 用上一步的激活和当前权重预测
    actual = |a_j(t)|                 ← 当前实际激活
    xin_tension += (predicted - actual) × dt

R (Regulate):
  VoltageRegulator: 调节能量供给
  Homeostasis (_motor_homeostasis): 调节 Motor 增益
  PNN: 调节可塑性门控 g_ℓ
  NDR: 调节兴奋性
  MOSFET.adapt_threshold: 调节阈值
```

### 3.2 P 和 R 的数学关系

```
P 产生残差: ξ = ŷ - a
R 消费残差: R 的各机制响应 ξ 的不同方面

具体地:
  P: ξ = Σ W × a(t-1) - a(t)

  ξ 的分解:
  ξ = [W 误差] + [a 误差] + [耗散误差]

  其中:
  [W 误差] = W 不准确 → STDP/BCM 学习修正 W → 这是 P 自我修正
  [a 误差] = 输入变化了 → 新事件 → Xin 脉冲
  [耗散误差] = RC 泄漏、能量耗尽 → VR/Homeostasis 补偿

  P 只负责检测 ξ
  R 负责归因 ξ 的来源并调节对应参数:
    ξ 来自 W 错误 → 学习修正 W (其实这混在 P 中)
    ξ 来自能量不足 → VR 增加供能
    ξ 来自过度兴奋 → Homeostasis 降增益
    ξ 来自系统老化 → PNN 冻结权重
```

### 3.3 问题: P 和 R 没有形式化分离

```
当前实现中 P 和 R 的界限不清:

  bundle.learn() 既是 P（权重更新基于 pre × post）
                也是 R（PNN 门控、成熟率控制）
  
  compute_xin() 只是记录 ξ → 不驱动学习
  learn() 不看 ξ → 直接用 pre × post

  → P 在计算 ξ 但没人用
  → learn() 在做"学习"但不受 ξ 指导
  → R 的各机制各自为政
  
  这就是 C-005 §4 说的 "P 相在主电路中无实效"
```

## §4 新知识不摧毁旧知识

### 4.1 递归链的身份

```
用户说:
  "以固化的概率知识有其特有的反证与屏蔽公式"
  "基于的 T/O/P/R/Xin 递归严格推导且可逆"
  "无法证明为同一条递归链的新 T → 新 Xin → 新 O/P/R"

翻译:
  旧知识 = 通过某条 T/O/P/R/Xin 递归链固化的度规 g_ij
  新事件 = 新的 T (输入)
  
  两种情况:
    1. 新 T 属于同一递归链 → 沿已有度规测地线预测
       → Xin ≈ 0 → 不需要新学习 → 旧知识屏蔽新干扰
    
    2. 新 T 不属于已知递归链 → 无法用旧度规预测
       → Xin ≠ 0 → 但不知道旧度规是否适用
       → 不应修改旧度规 → 应该创建新的 O/P/R
```

### 4.2 "反证与屏蔽"

```
已固化的知识 (PNN 成熟 → g_ℓ ≈ 0):
  → 权重几乎不可改 → 度规几乎不变
  → 新 Xin 不能修改这部分度规
  → 这就是"屏蔽"

反证:
  如果新 T 持续产生大 Xin 且与旧递归链的预测矛盾
  → 持续矛盾 = 旧度规可能错误
  → 但 PNN 保护下仍不改 → 需要更强的机制来"反证"
  
  当前项目没有"反证"机制:
    PNN 冻结了就冻结了，没有解冻条件
    生物学中: 情绪应激 → 糖皮质激素 → PNN 降解 → 解冻
    项目中: Modulator (DA/5-HT) 存在但未接入 PNN
```

### 4.3 递归链的可逆性

```
T/O/P/R/Xin 可逆?

正向: T → O → P → R → Xin
反向: Xin → R⁻¹ → P⁻¹ → O⁻¹ → T⁻¹

可逆的条件:
  每个映射都是双射 (可逆)
  T: 传播 → 可逆? 如果知道权重和输出, 可以反推输入 (W⁻¹ × output)
  O: 观测 → 可逆? 如果知道阈值, 可以反推膜电位
  P: 预测 → 可逆? 如果知道 ξ 和 actual, 可以反推 predicted
  R: 调节 → 可逆? 如果知道调节后和调节前的参数差

  在理想情况下 (无信息丢失): 每一步都可逆
  在实际中: MOSFET 阈值以下 → 信息丢失 → 不可逆
  
  但如果用 δa (偏移) 代替 |a| (绝对值):
    偏移可以是正或负 → 保留了方向信息
    如果 MOSFET 用 Softplus 或亚阈值指数 → 不完全截断
    → 信息衰减但不丢失 → 弱可逆

  "弱可逆" = 反向链产生的是"虚弱波"
  这正是用户之前说的:
    "反向激活引发的虚弱波可以作为 Xin 来验证判断是否正确"
```

## §5 Xin 递归衰减的意义

```
用户说: "xin 在定量场景中只会越来越弱"

数学上:
  xin(t+1) = f(xin(t))  其中 f 的增益 < 1 (因为每层都在消化 xin)
  → xin → 0 (不动点)

但 Xin 不是用来做定量驱动的:
  Xin 是检验器: ξ ≈ 0 → 当前度规正确
  Xin 是事件标记: ξ 突变 → 新事件
  Xin 的结构 (在哪些维度) 才是信息

递归衰减 ≠ 失效:
  衰减到 0 = 系统已学会 = 度规收敛 = 正确行为
  突然脱离 0 = 新事件 = 触发新 T/O/P/R 循环

  生物类比:
    适应后的神经元: 持续刺激 → 反应 → 0
    新刺激: 反应 = 峰值
    信息在峰值的时刻和方向, 不在持续的幅度
```
