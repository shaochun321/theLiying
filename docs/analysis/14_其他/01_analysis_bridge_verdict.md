# Jaynes-Landauer Bridge — 诚实裁决

## 三个实验结果

| 实验 | 方法 | R值 | 裁决 |
|------|------|-----|------|
| **差分** (学习 ON vs OFF) | R(残差, -TΔS差) | **-0.17** | ❌ 无因果 |
| **高对比** (放大 ΔS) | R(残差, -TΔS) | **0.38** | ⚠️ 微弱信号 |
| **公式诊断** | 0.1 vs S×ln2 | **差 20.7×** | ❌ 公式错误 |

## 核心发现

### 1. 自由能公式 `F = ΣE - T × 0.1` 不是热力学自由能

```
公式中:  α = 0.1 (人为常数)
实际值:  α = S_weights × ln2 ≈ 2.07 (差 20.7 倍)
```

这个 `0.1` 不是从信息量导出的，是人为选择的标量。
它让 F 看起来像热力学量，但**不携带任何信息论内容**。

### 2. Bridge 在差分实验中被否决

学习 ON 和学习 OFF 的自由能差异 **不能被 TΔS 解释**：
- R = -0.17 (负相关！)
- 残差中的变异主要来自**非信息源**（代谢波动、CPG 干扰）

### 3. 但高对比实验中有微弱正信号

- R = 0.38 (弱正相关)
- 信号比 = 1.6 (残差和 TΔS 量级可比)

这说明：**信息-热力学耦合存在，但被噪声淹没。**

## 为什么 P1 (Landauer bound) 始终满足？

> Q >> kT ln2 ΔS (约 100 倍裕量)

因为系统中 I²R 热耗散**远大于**信息处理所需的最低热量。
就像一台柴油发动机——热效率只有 3%，97% 是废热。
Landauer bound 对这种"浪费型"系统永远满足，**没有区分力**。


## 下一步选项

> 这是一个真实的、有价值的否定结果。它精确地指出了**哪里**有问题。

### 选项 A: 修正自由能公式

```python
# 旧 (人为):
self._free_energy = total_energy - self._temperature * 0.1

# 新 (信息论导出):
S_w = shannon_entropy(all_weights)
self._free_energy = total_energy - self._temperature * S_w * math.log(2)
```

然后重跑三个实验，看修正后 bridge 是否成立。

### 选项 B: 接受当前状态

当前系统的信息-热力学耦合是**存在但微弱的**。
Landauer bound 始终满足，但没有区分力。
作为论文贡献：这是一个**诚实的负结果** — 说明需要更高的信息密度才能让 bridge 可测。

### 选项 C: 增强信息密度

问题不在公式，而在系统的**信息效率太低**（97% 废热）。
需要让系统更接近 Landauer bound：
- 降低 R_leak → 减少无用耗散
- 增大学习率 → 每 tick 更多信息变化
- 降温 → 使 kT ln2 更小，bound 更紧
