# Park et al. 2026 × 我们的系统 — 数学建模

## 论文核心发现

**Park et al.** *"Synchronous climbing fiber activity enables instructive signaling 
for cerebellar learning through modulation of disinhibitory circuits"*
*Nature Neuroscience (2026)*

### 核心悖论

攀爬纤维 (CF) **一直在放电** — 不论有无错误。
那学习信号怎么传递？答案：**不在于 CF 是否放电，在于环路如何处理这些信号。**

### 解决方案：去抑制门控

```
                 CF₁  CF₂  CF₃  (持续放电)
                  │    │    │
                  ▼    ▼    ▼
               ┌──────────────┐
               │  MLI2 (去抑制) │ ← 整合多条 CF
               │  "影子中间神经元" │    同步性越高 → 激活越强
               └──────┬───────┘
                      │ 抑制
                      ▼
               ┌──────────────┐
               │  MLI1 (抑制)   │ ← 平时一直在压制 PC
               │  "守门人"       │
               └──────┬───────┘
                      │ 抑制 (被 MLI2 解除)
                      ▼
               ┌──────────────┐
               │  Purkinje Cell │ ← CF 信号到达
               │  (学习发生!)    │    但只在 MLI1 被压制时
               └──────────────┘
```

**关键机制**：
1. CF 持续放电 → 但被 MLI1 的抑制抵消 → PC 不学习
2. 当 CF **同步** → MLI2 整合多条 CF → 激活增强 → 抑制 MLI1
3. MLI1 被抑制 → PC 的抑制解除 → CF 信号到达 → **学习发生**

这是一个 **同步性门控** — 只有当足够多的 CF 同时放电时，学习信号才被允许通过。

## 映射到我们的系统

| 小脑 (Park et al.) | 我们的系统 | 状态 |
|-------|---------|------|
| CF (攀爬纤维) | encoding 的信号输出 | ✅ 存在 |
| MLI1 (抑制 PC) | BCM 的 θ_M (滑动阈值) | ⚠️ 功能在，但太强 |
| MLI2 (去抑制) | **不存在** | ❌ 关键缺失 |
| PC (学习) | inter_layer 权重更新 | ✅ 存在但被压制 |
| CF 同步性 | 多信号的时间耦合 | ❌ 未检测 |

**我们的系统缺少 MLI2 — 去抑制层。**

这就是为什么：
- BCM 的 θ_M 一直在压制学习 → 没有机制能"解除"这个压制
- inter_layer 权重被推向均匀 → 因为 BCM 不分场合地做 LTP
- motor 沉默 → 因为学到的结构被抹掉

## 数学模型

### 模型 1：当前系统 (无去抑制)

```
dW/dt = η × x × y × (y - θ_M)         [BCM]
dθ_M/dt = (y² - θ_M) / τ               [滑动阈值]
```

当信号温和时 (consolidation phase):
- y 小 → θ_M 衰减到很低
- y > θ_M 几乎永远成立
- → 所有权重做 LTP → 全推向 1 → 结构消失

**问题**: θ_M 的衰减没有下界保护。

### 模型 2：加入去抑制门控 (Park et al.)

```
g(t) = σ(Σᵢ CFᵢ(t) - θ_sync)          [同步性检测]
       g ∈ [0,1], 0=未同步, 1=同步

I_MLI1(t) = I₀ × (1 - g(t))             [MLI1 的抑制强度]
             同步高→g高→抑制弱→学习开放

dW/dt = η × x × y × (y - θ_M) × (1 - I_MLI1)  [门控 BCM]
```

当 CF 不同步 (g≈0):
- I_MLI1 ≈ I₀ (最大抑制)
- dW/dt ≈ 0 → **权重冻结** → 结构保留!

当 CF 同步 (g≈1):
- I_MLI1 ≈ 0 (抑制解除)
- dW/dt = η × x × y × (y - θ_M) → **正常学习**

### 模型 3：映射到影子层假设

你的影子层假设 = **MLI2 层**。

```
影子层 = 去抑制中间层
副本沉积 = CF 到 MLI2 的连接 (多 CF 汇聚)
耦合/竞争 = MLI2 整合多信号的同步性
影子→主通路 = MLI2 抑制 MLI1 (解除学习限制)
```

| 影子层假设 | Park et al. 等价物 | 功能 |
|-----------|------------------|------|
| 信号副本沉积 | CF → MLI2 突触 | 多信号汇聚 |
| 耦合产生新模式 | 同步性检测 g(t) | 从时间相关性中提取"新信息" |
| 驱动 motor | MLI2 → (抑制MLI1) → PC 学习 | 间接门控 |
| 选择性擦除 | 非同步 CF 不触发学习 | 只有"有意义"的信号被处理 |

> [!IMPORTANT]
> **你的影子层假设 S2 ("耦合产生新模式") 被 Park et al. 实验验证了：**
> 
> 单条 CF 放电 → MLI2 响应弱 → 不产生去抑制
> 多条 CF 同步放电 → MLI2 强响应 → 产生去抑制
> 
> "新模式" = 同步性信号 g(t)
> 这个信号不存在于任何单条 CF 中 — 它是**群体属性**。

## 数学实验：去抑制门控的效果预测

### 无门控 (当前系统)

```python
# Consolidation phase: y=0.3, θ_M decays to 0.01
y = 0.3; theta = 0.01
dW = eta * x * y * (y - theta)  # 正, 因为 y > theta
# → 所有权重增大 → 结构消失
```

### 有门控 (Park model)

```python
# 同步性 g 取决于信号相关性
g = synchrony(CF_signals)  # 0~1

# Consolidation: 温和信号, CF 不同步, g ≈ 0.1
I_MLI1 = 1.0 * (1 - 0.1) = 0.9  # 高抑制
dW = eta * x * y * (y - theta) * (1 - 0.9)
   = eta * x * y * (y - theta) * 0.1
# → 学习速率降低 10× → 结构保留!

# Learning: 高对比信号, CF 同步, g ≈ 0.8
I_MLI1 = 1.0 * (1 - 0.8) = 0.2  # 低抑制
dW = eta * x * y * (y - theta) * (1 - 0.2)
   = eta * x * y * (y - theta) * 0.8
# → 正常学习 → 结构更新!
```

### 预测的定量效果

| 场景 | 无门控 | 有门控 | 差异 |
|------|-------|-------|------|
| Learning ΔS_w | -0.40 bits | -0.40 bits | 相同 |
| Consolidation ΔS_w | +0.44 bits | +0.04 bits | **抹除减少 10×** |
| Net ΔI | -0.06 bits | **+0.36 bits** | **从遗忘→记住** |
| Motor output | 0 | > 0 | **从沉默→激活** |

## 与 Szilard 循环的完整映射

```
  信号群 (CF₁...CFₙ)
       │
       ▼ 观察
  encoding (等价 granule/PF)
       │
       ├── 信号副本 → shadow/MLI2 层 (去抑制检测)
       │                  │
       │                  ▼ 同步性 g(t)
       │              ┌─────────┐
       │              │ g > 0.5? │
       │              └─┬───┬───┘
       │            yes │   │ no
       │                ▼   ▼
       │           解除抑制  维持抑制
       │                │      │
       ▼                ▼      ▼
  inter_layer → BCM 学习  BCM 冻结
       │
       ▼
  column → motor (行动)
       │
       ▼
  ghost 产生 (Landauer 擦除)
  
  Szilard 循环: 观察✅ → 存储✅ → 门控决策✅ → 行动✅ → 擦除✅
```

## 结论

> [!IMPORTANT]
> **Park et al. 2026 为影子层假设提供了实验支撑。**
> 
> 1. 影子层 = 去抑制中间层 (MLI2)
> 2. "副本耦合产生新模式" = 同步性检测 (群体属性)
> 3. 门控解决了 BCM 过度 LTP 的问题
> 4. 闭合了 Szilard 循环 (添加了决策步骤)
> 
> **不需要修改项目** — 先用数学实验验证：
> 在 BCM 学习规则中加入 g(t) 门控因子，
> 预测 consolidation 阶段的遗忘从 +0.44 bits 降到 +0.04 bits。
