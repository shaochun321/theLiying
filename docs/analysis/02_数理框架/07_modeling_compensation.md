## 1. 傅里叶频谱分析

对 Column 激活信号（t > 1500 ms 的活跃窗口，3500 个采样点）做离散傅里叶变换：

### DFT 结果

| 排名 | 频率 (Hz) | 周期 (ms) | 幅度 |
|------|-----------|-----------|------|
| 1 | 0.29 | 3500 | 0.5521 |
| **2** | **11.43** | **87.5** | **0.4394** |
| 3 | 12.29 | 81.4 | 0.3810 |
| 4 | 11.14 | 89.7 | 0.3579 |
| 5 | 10.86 | 92.1 | 0.3517 |

### 频谱解读

```
                        ┌─ 低频包络 (0.29 Hz) = 能量耗竭-恢复周期
   Column 频谱         │  
   幅度 ↑              │  ┌─ 高频簇 (~11 Hz) = 传入放电驱动
        │ ██            │  │
        │ ██  ████████  │  │
        │ ██  ████████  │  │
        └─┴───┴────────→ 频率 (Hz)
          0.3   11-13
```

**关键发现**：
1. **0.29 Hz（T=3.5s）**：整体能量耗竭-恢复的振荡。这不是信号振荡，是**系统崩溃-恢复的呼吸频率**。
2. **11-13 Hz 簇**：与传入放电率（12.5 Hz）吻合。这是**信号驱动频率**。
3. 频谱中没有目标的 50-100 Hz 生物频率 → 传入放电偏慢。

## 3. 能量平衡分析

核心方程：

$$\frac{dE}{dt} = P_{metabolic} - P_{dissipation}$$

### 各层能量收支

| 层 | 平均耗散 | 峰值耗散 | 当前恢复 | 亏缺比 |
|----|---------|---------|---------|--------|
| Encoding | 1.37 / step | 10.69 / step | 0.001 / step | 1,368× |
| **Column** | **30.32 / step** | **133.80 / step** | **0.001 / step** | **30,322×** |
| Motor | 3.50 / step | 22.78 / step | 0.001 / step | 3,499× |

> [!CAUTION]
> Column 层的能量亏缺比达到 **30,322 倍**！
> 这意味着代谢恢复速率只有耗散速率的 0.003%。
> 任何真实系统在这种失衡下都会立即崩溃。

### 时间常数 vs 脉冲间隔

```
Encoding τ_RC = R × C = 2.0 × 0.5 = 1.0 ms
Column   τ_RC = R × C = 3.0 × 1.0 = 3.0 ms
Afferent ISI = 79.8 ms
```

**两次脉冲之间的电压保留率**：

$$V_{retain} = e^{-ISI/\tau_{RC}}$$

| 层 | τ (ms) | ISI (ms) | 保留率 |
|----|--------|----------|--------|
| Encoding | 1.0 | 79.8 | e^{-79.8} ≈ **0%** |
| Column | 3.0 | 79.8 | e^{-26.6} ≈ **0%** |

> [!IMPORTANT]
> 关键矛盾：ISI = 79.8 ms >> τ_RC = 1-3 ms
> 
> 在两次脉冲之间，Encoding 和 Column 的膜电压**完全衰减到零**。
> 每次传入脉冲到来时，下游从零开始充电。
> 这就是为什么 Column 在 0 ↔ 6 之间跳变而不是平稳波动。

## 4. 半导体风格补偿方案

基于以上数学分析，推导 4 种补偿机制，全部用半导体等效件实现：

### A. 电压调节器 (LDO) → 活动依赖代谢

**生物**: 线粒体 ATP 产出速率与神经活动正相关。

**半导体**: LDO (Low Dropout Regulator) — 输出随负载自动调节。

**数学推导**:
```
稳态条件: P_metabolic = P_dissipation
P_dissipation ∝ |activation|  (经验关系)

Encoding: coeff = mean_heat / mean_act = 1.37 / 3.67 = 0.37
Column:   coeff = mean_heat / mean_act = 30.32 / 1.44 = 21.13
```

**公式**:
```python
# 代替固定 metabolic_recovery = 0.001
recovery = base_rate + activity_coeff * abs(activation)
# Encoding: recovery = 0.001 + 0.37 * abs(activation)
# Column:   recovery = 0.001 + 21.13 * abs(activation)
```

**但直接用 activity_coeff=21.13 太大了**。因为 I²R 耗散源头是膜电压过高（col_v=6.0），根本解法是限制电压而非无限补偿。

### B. 去耦电容器 → 能量缓冲 / 膜电压平滑

**生物**: 突触囊泡池、树突电容缓冲。

**半导体**: 去耦电容并联在负载旁，吸收瞬态电流。

**数学推导**:
```
需要 τ_buffer >> ISI
τ_buffer = 5 × ISI = 5 × 79.8 = 399 ms
如果 R_buffer = 10 → C_buffer = 39.9
```

**nexus_v1 实现**: 在 Encoding/Column 增大膜电容 C 或增大 r_leak 使 τ >> ISI:
```
所需 τ_RC ≥ 2 × ISI = 160 ms
如 C=1.0 → r_leak = 160
如 C=10 → r_leak = 16
```

> [!IMPORTANT]
> 这是最直接的修复：**增大 τ_RC 使其远超 ISI**。
> 当前 τ = 1-3 ms，ISI = 80 ms → τ/ISI = 0.01
> 目标 τ ≥ 160 ms → τ/ISI ≥ 2.0

### C. 偏置电流源 → 张力性自发放电

**生物**: 前庭传入即使无输入也以 ~70 Hz 自发放电（钾泄漏电流驱动）。

**半导体**: 恒流偏置 (Bias Current Source)。

**数学推导**:
```
目标: V_bias = 50% × V_threshold = 0.5 × 0.1 = 0.05
稳态: V_ss = I_bias × R_leak
I_bias = V_target / R_leak
```

对 Encoding (R=2.0): `I_bias = 0.05 / 2.0 = 0.025`
对 Column  (R=3.0): `I_bias = 0.05 / 3.0 = 0.017`

**nexus_v1 实现**: 在 neuron.step() 的输入电流中加一个固定偏置：
```python
injected = external_current + i_bias  # bias keeps neuron near threshold
```

### D. 自动增益控制 (AGC) → 稳态可塑性

**生物**: BCM 规则、突触缩放、内在兴奋性调节。

**半导体**: AGC 反馈环路 — 输出太大则降低增益，太小则提高。

**数学推导**:
```
gain(t) = K / (1 + ā(t) / a_target)

ā(t) 是活动的指数滑动平均:
ā(t+1) = ā(t) × (1 - α) + |activation| × α
α = 0.01 (τ_AGC = 100 steps = 100 ms)
K = 20 (基础增益)
a_target = 1.0 (期望激活水平)
```

| 状态 | ā | gain | 效果 |
|------|---|------|------|
| 沉默 | 0 | 20.0 | 高敏感度，容易激活 |
| 正常 | 1.0 | 10.0 | 标准增益 |
| 过度兴奋 | 5.0 | 3.3 | 压制增益，防止爆发 |

## 5. 推荐方案：优先级排序

基于数学分析，**方案 B（增大 τ_RC）是根本解**：

| 优先级 | 方案 | 根因 | 复杂度 | 效果 |
|--------|------|------|--------|------|
| **P0** | **B: 增大 τ_RC** | τ/ISI = 0.01 (应 ≥ 2) | 改 2 个参数 | **解决跳变** |
| P1 | A: 活动依赖代谢 | energy 归零 | 改 step() | 解决能量耗尽 |
| P2 | C: 偏置电流 | 零基线 | 加 1 行代码 | 解决间隔沉默 |
| P3 | D: AGC | 增益不适应 | 新组件 | 解决长期稳定 |

### P0 的具体参数（增大 τ_RC）

```python
# Encoding: τ 从 1.0 ms → 160 ms
_encoding_config: capacitance=10.0, r_leak=16.0  # τ=160 ms

# Column: τ 从 3.0 ms → 240 ms
_column_config: capacitance=12.0, r_leak=20.0    # τ=240 ms
```

但膜电容增大意味着**充电变慢**，电压幅度变小：
```
ΔV = I × dt / C
C = 10 → ΔV = I × 0.001 / 10 = I × 0.0001
```

需要同步提高 synapse_gain 或降低阈值来补偿：
```
v_threshold = 0.01 (从 0.1 降低)
```

## 6. 数学验证：P0 修改后的预期行为

### 修改前（当前）

```
           ISI = 80 ms
    ┌──┐                    ┌──┐
    │  │                    │  │        ← 脉冲
────┘  └────────────────────┘  └──── t
       │← τ=1ms →│                       ← 1ms 后衰减到 0
       完全遗忘
```

### 修改后（τ = 160 ms）

```
           ISI = 80 ms
    ┌──┐                    ┌──┐
    │  │                    │  │        ← 脉冲
────┘  └──────────────┐     │  │     ← 保留 60%
                      └─────┘  └──── 
       │← τ=160ms ─────────→│       ← 慢衰减
       保留 exp(-80/160) = 61%
```

叠加效应使电压趋向稳态值：

$$V_{ss} = \frac{V_{spike}}{1 - e^{-ISI/\tau}} \approx \frac{V_{spike}}{0.39} \approx 2.56 \times V_{spike}$$

这将产生**平滑的持续激活**而非 0 ↔ 6 的跳变。
