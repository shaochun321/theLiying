## §1 层级结构

```
┌─────────────────────────────────────────────────────────────┐
│  最高层: 高维存在                                             │
│  时间/空间/信息 = 高维结构的投影                                │
│  项目的目标 = 从投影中重建结构                                  │
└──────────────────────────┬──────────────────────────────────┘
                           │ 投影
┌──────────────────────────▼──────────────────────────────────┐
│  约束层: 物理原理作为泛函约束                                   │
│                                                             │
│  诺特定理: 对称性 → 守恒量                                     │
│    → ds² 在某变换下不变 → 存在守恒的环流                        │
│    → 约束: δ∫μ dt = 0 当且仅当 ds² 具有该对称性                │
│                                                             │
│  相对论: 坐标无关性                                            │
│    → ds² 在观测者变换下不变 → 项目的空间知识是内禀的             │
│    → 约束: ds² 只能从 a_i × a_j (输入流) 构建, 不能引用坐标     │
│                                                             │
│  变分原理: 最小作用量                                          │
│    → δS = 0, S = ∫L dt = ∫(T - V) dt                       │
│    → T = ½ g_ij v^i v^j (动能 = 度规 × 速度)                 │
│    → V = -a_therm(x) (势能 = 负热场)                         │
│    → 约束: 项目的运动倾向于沿测地线（最小 Xin 路径）             │
│                                                             │
│  信息几何: Fisher 度规                                        │
│    → ds²_Fisher = E[(∂log p/∂θ)²] dθ²                      │
│    → 约束: 学习 = 在统计流形上沿 Fisher 测地线移动              │
└──────────────────────────┬──────────────────────────────────┘
                           │ 约束
┌──────────────────────────▼──────────────────────────────────┐
│  主方法层: 时空测度 ds² 和运动势 ν                              │
│                                                             │
│  ds² = Σ g_ij × da_i × da_j                                │
│    g_ij = w_cross(i,j) = 影子层跨轴权重                       │
│    da_i = 输入偏移量（从基线的偏差）                            │
│                                                             │
│  ν = dρ/dt                                                  │
│    ρ = μ_p / μ_total (环流占比)                              │
│    ν = 占比变化率 = "急迫程度"                                 │
│                                                             │
│  ds² ↔ ν 的连接:                                             │
│    T = ½ g_ij (da_i/dt)(da_j/dt) = "度规空间中的动能"         │
│    ν = ∂L/∂(dρ) 或等价地 ν ∝ √(ds²/dt²)                    │

└──────────────────────────┬──────────────────────────────────┘
                           │ 表达
┌──────────────────────────▼──────────────────────────────────┐
```

## §2 项目理念如何统一到 ds²/ν

### 2.1 MOSFET 阈值 → 度规的退化边界

```
MOSFET: activation = max(0, V - V_th)

在度规语言中:
  V_th 定义了度规空间的"地平线"
  V < V_th → 度规退化 (det g = 0) → 该方向的度量消失
  V > V_th → 度规有限 → 该方向可被测量

  物理类比: 黑洞视界 → 视界内的信息不可及
  项目类比: 阈值以下的信号对系统不存在

问题: V_th = 0.3 是固定的工程常数
理想: V_th 应该从 ds² 的结构中涌现
  → 当某方向的度规分量太小时, 该方向自然"不存在"
  → 不需要硬阈值 → 度规退化就是阈值
```

### 2.2 STDP/BCM 学习 → 度规更新规则

```
STDP: Δw = η × pre × post - λw
BCM:  Δw = η × |a_i| × |a_j| × (|a_j| - θ)

在度规语言中:
  w = g_ij (度规分量)
  学习 = 从经验中更新度规

  STDP: Δg_ij ∝ (共活) - (衰减)
    = 两个方向共变时, 度规增大 → 这两个方向"更近"
    = 度规的经验性构建

  BCM: Δg_ij ∝ (共活) × (活跃度 - 阈值)
    = 只有超过平均活跃度的方向才增长
    = 竞争性度规构建 → 稀疏表示
```

### 2.3 Xin → 测地线偏差

```
Xin = predicted - actual
    = ŷ - a(t)
    = "按当前度规预测的位置" - "实际到达的位置"

在度规语言中:
  预测 = 沿当前度规的测地线外推
  实际 = 真实的位置
  Xin = 测地线偏差 = 曲率的局部测量

  如果度规完美 → 测地线 = 实际轨迹 → Xin = 0
  如果度规有误 → 测地线偏离 → Xin > 0

  Xin 驱动度规更新 → 学习 = 减小测地线偏差
  = 自由能最小化的几何表达
```

### 2.4 环流 μ → 度规空间中的体积流

```
μ_p = a_col × a_bind × w × f_feedback

在度规语言中:
  μ_p = 度规空间中沿路径 p 的"体积流"
  体积 = √(det g) × 坐标体积
  高度规 → 大体积 → 大流量

  μ_total = 总体积流 = 度规空间的总"容量"
  ρ = μ_p / μ_total = 路径 p 占总容量的比例
  ν = dρ/dt = 容量分配的变化率
```

### 2.5 影子层 κ → 度规的缩减

```
κ = (Σ w_cross × |a_i| × |a_j|) / (Σ|a_i| × Σ|a_j|)

在度规语言中:
  分子 = ds² (在当前激活模式下的度规值)
  分母 = 如果度规是单位矩阵的 ds² (最大可能值)
  κ = ds²_actual / ds²_max = 度规相对于单位矩阵的缩减程度

  κ → 1: 度规接近单位矩阵 → 各方向等权 → "没有结构"
  κ → 0: 度规高度各向异性 → 某些方向被强化 → "有结构"

  影子层的缩减 = 度规偏离"无结构"的程度
  = 项目内部空间结构的量度
```

### 2.6 热膜适应 → 势能场的基线校准

```
M → T (甲基化追踪温度)
therm = T - M (偏差)

在度规语言中:
  T = 势能场 V(x) 的当前值
  M = 势能场的基线（长期平均）
  therm = V(x) - V̄ = 势能偏差

  只有偏差（而非绝对值）进入度规计算
  = "度规只编码变化, 不编码常数"
  = 等效于广义相对论中"局部惯性系中引力消失"
```

### 2.7 绑定层 → 度规的非对角分量

```
bind(oto_x, therm) 激活条件:
  col_oto_x > θ AND col_therm > θ

在度规语言中:
  绑定 = 度规的非对角分量 g_ij (i≠j) 是否有效
  只有两个方向同时活跃, 非对角分量才参与 ds² 计算
  = 度规是条件性的（不总是完整的）
  
  无绑定: ds² = Σ g_ii (da_i)² (对角, 各方向独立)
  有绑定: ds² = Σ g_ij da_i da_j (完整, 方向间有关联)
```

### 2.8 PNN 成熟 → 度规冻结

```
g_ℓ = 1 - PNN (可塑性门控)
PNN 增长 → g_ℓ 降低 → Δg_ij 减小

在度规语言中:
  PNN = 度规的刚度 (rigidity)
  低 PNN: 度规柔软 → 快速学习 → 空间结构不稳定
  高 PNN: 度规刚性 → 缓慢学习 → 空间结构稳定

  = 度规从流体到固体的相变
  = "空间结构晶化"
```

## §3 分线性工具与光滑数学的关系

```
项目当前:
  MOSFET: max(0, V - V_th)     → 分线性
  绝对值: |a|                   → 分线性
  clip: max(min, min(max, x))   → 分线性
  EMA: x += (y-x) × α          → 线性

光滑数学需要:
  黎曼度规: g_ij 光滑 → ds² 可微
  变分: δS = 0 需要 L 可微
  勒让德: 需要 L 严格凸

关系:
  分线性 = 光滑的分段近似
  max(0, x) ≈ softplus(x) = log(1 + exp(x))
  |x| ≈ sqrt(x² + ε)
  
  MOSFET 的亚阈值区域已经是光滑的:
    I ∝ exp((V - V_th) / (n·VT))  → 指数 → 光滑!
  
  → 项目实际上有两个区域:
     亚阈值: 光滑 (指数) → 变分可用
     超阈值: 线性 → 分线性
     
  → 在阈值附近, 两者连续拼接
  → 如果信号主要在阈值附近波动, 光滑部分主导
  → 如果信号远超阈值, 线性部分主导
```

## §4 缺失: 基线活动作为垫支

```
用户指出:
  "前庭、热感受、甚至整个项目本身一直是被激活的状态"
  "是输入让激活基线偏移"

在度规语言中:
  基线活动 = 度规空间中的"背景曲率"
  无输入时: ds² = ds²_baseline > 0 (空间存在)
  有输入时: ds² = ds²_baseline + δds² (空间被扰动)

  信息在 δds² 中, 不在 ds² 的绝对值中
  = 广义协变性: 物理在背景度规的选择下不变

当前项目:
  无输入 → activation = 0 → ds² = 0 → "空间不存在"
  
  这是错的.
  应该是:
  无输入 → activation = baseline → ds²_baseline > 0
  有输入 → activation = baseline + δ → ds² = ds²_baseline + δds²
```

## §5 尺度问题的度规表达

```
用户指出:
  "以大型生物的系统作指导, 却以小型生命体的体量施加力"

在度规语言中:
  度规的"尺度" = g_ij 的数值范围
  前庭链的增益链 = 度规从物理空间到神经空间的映射
  
  物理空间: 加速度 ~0.01 单位/s²
  前庭增益链: MET → HC → Aff → Enc → Col
    每层阈值 ~0.3, 增益 ~1.0
  神经空间: 需要 activation > 0.3

  映射链: 0.01 → (×?) → 需要 > 0.3
    → 需要总增益 > 30 倍
    → 但每层都有阈值损耗
    → 净增益可能 < 1

  = 度规映射退化
  = 物理空间中可分辨的事件在神经空间中不可分辨
  
  body.total_volume = 9 (写在代码里)
  body.mass = 1.0 (默认值)
  → 这些数字不参与度规映射的计算
  → 身体的物理属性与神经系统的信号尺度断开

  需要: 让身体的质量/体积参与信号增益的物理计算
    → 惯性质量决定加速度尺度
    → 身体尺度决定前庭灵敏度
    → 这些是度规映射的物理约束
```
