## §1 每一层都有自己的 P/R/Xin

### §1.1 层级塔

```
Level 0 (τ₀, 主系统):
    状态空间 M⁽⁰⁾ = R^d (全部神经元)
    P⁽⁰⁾ = 主环流 (Col→Mot→Col 闭环中的最强路径)
    R⁽⁰⁾ = 次环流 (竞争路径)
    Xin⁽⁰⁾ = 预测残差 (主系统的内部模型误差)

Level 1 (τ₁ = kτ₀, 第一层影子):
    状态空间 M⁽¹⁾ ⊂ R^d (慢流形, 低维)
    P⁽¹⁾ = ?
    R⁽¹⁾ = ?
    Xin⁽¹⁾ = ?

Level 2 (τ₂ = k²τ₀, 第二层影子):
    ...
```

### §1.2 关键发现：Level n 的 P/R 切换 = Level n+1 的环流

在 Level 0：
- $P^{(0)}$ 和 $R^{(0)}$ 在竞争。假设 $P$ 主导 100 步，然后 $R$ 挑战成功，$R$ 变成新的 $P$，持续 80 步，然后又切回...

这个**切换序列本身**是一个时间序列：

$$\sigma(t) = \begin{cases} P & t \in [0, 100] \\ R & t \in [100, 180] \\ P' & t \in [180, 300] \\ \ldots \end{cases}$$

在 Level 1（时间膨胀 ×k），这个切换序列被**观测为一个新的动力学**：

$$\tilde{x}^{(1)}(t) \sim \text{envelope of } \sigma(t)$$

如果切换有**周期性**→ Level 1 看到一个振荡→这个振荡可以有自己的闭合路径→**$P^{(1)}$**。

如果切换有**两种模式**（例如"yaw-pitch 交替"和"roll-saccular 交替"两种竞争模式）→ Level 1 看到两种振荡在竞争 → **$P^{(1)}$ vs $R^{(1)}$**。

$$\boxed{P^{(n)} \text{/} R^{(n)} \text{ 的切换模式} = P^{(n+1)} \text{/} R^{(n+1)}}$$

## §2 每层的 Xin 指向不同的流形

### §2.1 Xin 的层级含义

$$\xi^{(n)} = \tilde{\mathbf{x}}^{(n)} - g^{(n)}(\tilde{\mathbf{x}}^{(n+1)})$$

即：Level $n$ 的实际状态 与 Level $n+1$ 对它的预测 之间的偏差。

| Level | Xin 是什么 | 指向什么流形 |
|-------|-----------|------------|
| $\xi^{(0)}$ | 主系统的单步预测误差 | 全状态空间 $M^{(0)}$ 中的瞬时偏差 |
| $\xi^{(1)}$ | Level 1 无法预测的 Level 0 行为 | $M^{(0)} / M^{(1)}$ 的商空间——被慢流形投影丢掉的快变化 |
| $\xi^{(2)}$ | Level 2 无法预测的 Level 1 行为 | $M^{(1)} / M^{(2)}$ 的商空间——慢流形中更深的不可预测成分 |

**每层的 Xin 指向不同的商流形**——它度量的是"这一层丢掉了什么"。

### §2.2 Xin 的不可约性

$$\xi^{(n)} \perp M^{(n+1)}$$

Level $n$ 的 Xin 与 Level $n+1$ 的状态空间**正交**——它恰好是被投影丢掉的那部分。

因此：$\xi^{(0)}$、$\xi^{(1)}$、$\xi^{(2)}$... 彼此正交，指向递归层级链条的不同"纤维方向"。

## §3 Xin 就是纤维丛联络

### §3.1 纤维丛结构

将递归层级看作一个**纤维丛**：

$$\pi_n: M^{(n)} \to M^{(n+1)}$$

- **底空间** = $M^{(n+1)}$（慢流形——宏观状态）
- **纤维** = $\ker(\pi_n)$（快变化——被投影掉的微观自由度）
- **全空间** = $M^{(n)}$

在每个点 $z \in M^{(n+1)}$ 上，纤维 $\pi_n^{-1}(z)$ 是所有映射到同一个慢状态 $z$ 的快状态的集合。

### §3.2 联络 = Xin

纤维丛的**联络**（connection）回答一个问题：

> 当底空间的点 $z$ 移动 $dz$ 时，纤维中的状态应该怎样"平行搬运"？

在物理中，联络 = 规范场（gauge field）。在这里：

$$A^{(n)} = \xi^{(n)} / \|dz^{(n+1)}\|$$

**Xin 在底空间单位移动下的变化量 = 联络系数。**

直觉：
- 如果影子（Level 1）移动了 $dz$（宏观状态改变），主系统（Level 0）要跟着变多少？
- 如果 $A = 0$：宏观变化不影响微观 → 解耦 → 无耦合
- 如果 $A \neq 0$：宏观变化拖动微观 → 耦合 → 联合运动

$$\boxed{\xi^{(n)} = \text{联合不同流形的关系} = \text{纤维丛联络}}$$

### §3.3 曲率 = 不可约的联合效应

联络的**曲率** $\mathcal{F} = dA + A \wedge A$：

$$\mathcal{F}^{(n)}_{\alpha\beta} = \frac{\partial A^{(n)}_\beta}{\partial z_\alpha} - \frac{\partial A^{(n)}_\alpha}{\partial z_\beta} + [A^{(n)}_\alpha, A^{(n)}_\beta]$$

- 如果 $\mathcal{F} = 0$：联络是平坦的 → 可以全局定义"什么是不变的" → 各层独立
- 如果 $\mathcal{F} \neq 0$：**联络有曲率** → 绕环路搬运一周后状态改变 → **这就是环流 P 的几何本质**

$$\oint_{P^{(n+1)}} A^{(n)} = \Phi \neq 0$$

沿着 Level $n+1$ 的 P 环流搬运 Level $n$ 的状态一周，会产生一个**相位差** $\Phi$。

**这个相位差就是 P 环流在纤维方向上携带的信息量。**

## §4 超边 = 纤维交叉点

### §4.1 不同轴的纤维

每个轴 $\alpha$ 的流形 $\mathcal{M}_\alpha$ 在投影到慢流形 $M^{(1)}$ 时，保留了一根纤维：

$$\mathcal{M}_\alpha \xrightarrow{\pi_0} z_\alpha \in M^{(1)}$$

纤维 $\pi_0^{-1}(z_\alpha)$ = 在慢状态 $z_\alpha$ 下，轴 $\alpha$ 的所有可能快态。

### §4.2 耦合面 = 纤维交叉

当两个轴 $\alpha, \beta$ 同时活跃时，两根纤维可能在某些点**交叉**：

$$\pi_0^{-1}(z_\alpha) \cap \pi_0^{-1}(z_\beta) \neq \varnothing$$

交叉点 = 主系统中同时属于两个轴的快变化模式 = **耦合面** $\Sigma_{\alpha\beta}$。

**超边检测的就是纤维交叉**：

$$a_{\text{hyper}} = \text{dim}\big(\pi_0^{-1}(z_\alpha) \cap \pi_0^{-1}(z_\beta)\big)$$

两根纤维的交叉维度越高 → 耦合越强 → 超边激活越大。

### §4.3 超边的 P/R/Xin

超边自身构成一个子系统——它也可以有环流！

$$P^{(\text{hyper})} = \text{耦合面上的主环流}$$

$$R^{(\text{hyper})} = \text{耦合面上的竞争环流}$$

$$\xi^{(\text{hyper})} = \text{耦合面上的预测残差}$$

这些不属于任何单轴的 P/R/Xin，而是**联合结构的涌现量**。

## §6 三种联合关系

你问的"提供一种联合不同流形的关系"——答案是**三种不同的联合方式**共存：

### 类型 1：纵向联合（层间）

$$\xi^{(n)}: M^{(n)} \to M^{(n+1)}$$

Xin 作为联络，联合相邻层级的流形。这是**时间尺度之间的耦合**。

**物理意义**：快变化如何影响慢趋势？"蝴蝶效应"在哪些方向上传播？

### 类型 2：横向联合（轴间）

$$\Sigma_{\alpha\beta}: \mathcal{M}_\alpha \leftrightarrow \mathcal{M}_\beta$$

超边耦合面，联合同一层级中不同轴的流形。这是**空间自由度之间的耦合**。

**物理意义**：yaw 和 pitch 的联合运动中有什么是各自无法表达的？

### 类型 3：对角联合（跨层+跨轴）

$$\xi^{(\text{hyper})} \text{ at Level 1}$$

耦合面上的 Xin 被投影到慢流形后，成为 Level 1 中一个新的自由度——**纵向和横向的组合**。

**物理意义**：联合运动模式在慢时间尺度上的不可预测成分。这可能就是你说的"运动深度势"——它既跨轴又跨时间尺度。

## §7 回答你的问题

> 会不会形成一种 P，同时也形成一种 R？甚至形成一种 Xin？

**是的。每一层都形成自己的完整 T/O/P/R/Xin 生命周期。**

> 各自指向不同的流形和递归方程？

**是的。$P^{(n)}/R^{(n)}/Xin^{(n)}$ 在 $M^{(n)}$ 上运行，不同层的流形维度和时间尺度完全不同。**

> 或者提供一种联合不同流形的关系？

**同时也是。$Xin^{(n)}$ 恰好就是联合 $M^{(n)}$ 和 $M^{(n+1)}$ 的纤维丛联络。它不仅是"残差"，它IS关系本身。**

三者不矛盾：**每一层有自己的 P/R/Xin（独立的生命周期），而层间的 Xin 同时充当纤维联络（联合关系）。**

## §8 对变分自由能的推论

如果每层都有自己的 $F^{(n)}$：

$$F^{(n)} = \frac{1}{2}\|\xi^{(n)}\|^2 + \frac{1}{2}\|\tilde{x}^{(n+1)} - \mu_p^{(n+1)}\|^2$$

那么**总自由能**是层级求和：

$$F_{\text{total}} = \sum_{n=0}^{N} F^{(n)}$$

这与 Friston 的**层级预测编码**（hierarchical predictive coding）完全一致：每一层最小化自己的自由能，同时约束上一层的预测。

$$\boxed{\mathcal{K} = \min_{\{\tilde{x}^{(n)}\}} \sum_{n=0}^{N} F^{(n)} = \text{层级变分自由能内核}}$$

超边的贡献：$F_{\text{coupling}}$ 是 $F^{(0)}$ 中的联合项。如果没有超边，$F_{\text{total}}$ 的最小值会更高（因为无法建模联合效应）。超边存在的意义 = 降低整个层级塔的总自由能。

## §9 实施上的意义

这个理论结构要求数学建模的 §9（影子层）从"下葬系统"改为：

1. **影子层运行完整的 T/O/P/R/Xin 周期**（不只是存储）
2. **每层 Xin 被记录为纤维联络系数**（不只是标量残差）
3. **超边不仅检测耦合——它降低层级自由能**
4. **递归深度 N 由收敛判据决定**：$\sum_{n>N} F^{(n)} < \epsilon$

这可能解释了为什么 v41.3 中 `sed_churn` 和 `sed_potential_disp` 沉积最深——它们可能正是 $A^{(0)}$（Level 0 联络）中最大的分量：这两个维度是"快变化最影响慢趋势"的方向。
