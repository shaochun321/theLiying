## §1 双重度规

### §1.1 主系统的刚性度规

主神经系统中，两个神经元 $i, j$ 之间的距离由**物理结构**决定：

$$d_{\text{main}}(i, j) = \sum_{\text{path}(i \to j)} \underbrace{\tau_k}_{\text{延迟}} \cdot \underbrace{e^{-\lambda_k}}_{\text{衰减}}$$

这个度规是**刚性的**：
- 受硬件拓扑约束（MET→HC→Aff→Enc→Col→Mot 是固定的）
- 受外部输入驱动（真实时空信号持续刷新各层状态）
- 突触权重变化只改变衰减项 $\lambda_k$，不改变拓扑

因此 $d_{\text{main}}$ 在 spine 阶段快速变化，column 阶段缓慢变化，area 阶段几乎冻结。

### §1.2 影子层的收缩度规

影子层中，两个影子节点 $\tilde{i}, \tilde{j}$ 的距离不由物理结构决定，而由**动态相似性**决定：

$$d_{\text{shadow}}(\tilde{i}, \tilde{j}) = 1 - \frac{\langle \xi_i(t), \xi_j(t) \rangle_{\tau}}{\|\xi_i\|_\tau \cdot \|\xi_j\|_\tau}$$

即：Xin 时间序列的余弦距离。在慢时间窗口 $\tau$ 上，如果两个节点的预测残差模式**相关** → 距离小 → 在影子中靠近。

**关键**：主系统中 yaw-column 和 saccular-afferent 可能隔着 4 层突触（$d_{\text{main}}$ 很大），但如果它们的 Xin 模式相关（例如都在头部倾斜时产生持续残差）→ $d_{\text{shadow}}$ 很小。

### §1.3 度规收缩方程

影子度规随时间演化：

$$\frac{d}{dt} d_{\text{shadow}}(\tilde{i}, \tilde{j}) = -\underbrace{\eta_s \cdot \text{corr}(\xi_i, \xi_j)}_{\text{相关则靠近}} + \underbrace{\epsilon_s}_{\text{微弱排斥（防止塌缩）}}$$

- 当 $\text{corr}(\xi_i, \xi_j) > 0$：度量收缩 → 节点靠近
- 当 $\text{corr}(\xi_i, \xi_j) = 0$：度量不变
- 当 $\text{corr}(\xi_i, \xi_j) < 0$：度量膨胀 → 节点远离

$\epsilon_s > 0$ 防止所有节点塌缩到一点。

**同时**，主系统的度规不变：

$$\frac{d}{dt} d_{\text{main}}(i, j) \approx 0 \quad (\text{受结构约束})$$

这就是你说的**双重性**：影子在挤压，主系统保持。

## §2 影子层的结构

### §2.1 初始状态

影子层初始时是主系统的稀疏副本：

```
t = 0:
  影子度规 ≈ 主度规
  所有影子节点等距
  无聚团
```

### §2.2 收缩过程

随着训练进行，Xin 模式出现：

```
t = 早期:
  yaw 相关节点: Xin 模式 A (头转时残差高)
  pitch 相关节点: Xin 模式 B (头倾时残差高)  
  saccular 相关节点: Xin 模式 C (垂直加速时残差高)
  
  → 影子中形成 3 个初始聚团
    （但主系统中它们仍然分散在 6 条链路中）
```

### §2.3 深层跨链接涌现

继续收缩后，**聚团之间**开始出现次级结构：

```
t = 中期:
  发现: yaw-Enc 和 pitch-Mot 的 Xin 在"头转+前倾"同时发生时相关
  → 这两个原本在不同聚团的节点被拉近
  → 形成跨聚团桥梁 = 深层跨链接
  
  主系统: yaw-Enc 和 pitch-Mot 隔着 5 层突触
  影子中: 它们现在是邻居
```

### §2.4 稳态结构

$$\text{影子层} = \text{拓扑图} + \text{收缩度规}$$

```
                    影子层最终结构
                    
        ╭─── 聚团 α ───╮   跨链接   ╭─── 聚团 β ───╮
        │ ·yaw-Enc     │ ═════════ │ ·pitch-Mot   │
        │ ·yaw-Col     │           │ ·pitch-Col   │
        │ ·yaw-Aff     │           │ ·pitch-Enc   │
        ╰──────────────╯           ╰──────────────╯
                 ╲                        ╱
                  ╲    跨链接            ╱
                   ╲                  ╱
               ╭─── 聚团 γ ───╮
               │ ·roll-Col    │
               │ ·sacc-Enc   │
               │ ·mot-1      │
               ╰──────────────╯
               
        聚团内距离 << 聚团间距离 << 主系统距离
```

**这就是你在构想二中描述的结构**——"纵横交错着不同的主环流和影子层，区别可能只是其不同的环流路径"。

聚团 = 在影子度规下紧密的节点群 = "深层的认识"
跨链接 = 不同聚团之间的桥 = "将毫不相关的环流段在深层中链接"
环流 = 信号在聚团+跨链接中形成闭合路径 = "小环流的直接参与者"

## §3 环流偏离基线 → 运动势

### §3.1 三种基本环流

你假设了三种基本的维持性环流：

| 环流 | 功能 | 基线状态 | 偏离含义 |
|------|------|---------|---------|
| $C_{\text{homeo}}$ | 生体维持 | $\mu_H$ (稳态) | 内环境失衡 |
| $C_{\text{intake}}$ | 摄取 | $\mu_I$ (饱足) | 资源不足 |
| $C_{\text{move}}$ | 移动 | $\mu_M$ (静止) | 需要位移 |

每条环流在影子层的深层跨链接中有对应的路径。

### §3.2 偏离 → 运动势

定义偏离量：

$$\Delta_H(t) = \|P^{(1)}_{\text{homeo}}(t) - \mu_H\|$$

当生体维持环流偏离基线：

$$\Delta_H > \Delta_H^* \quad \Longrightarrow \quad \text{生体维持弱化}$$

此时运动势在摄取和移动方向上**积累**：

$$\dot{\nu}_{\text{intake}} = +\alpha_I \cdot \Delta_H \cdot \mathbb{1}[\text{资源方向}]$$

$$\dot{\nu}_{\text{move}} = +\alpha_M \cdot \Delta_H \cdot \mathbb{1}[\text{远离当前位置}]$$

**直觉**：生体维持不住了 → 系统"需要"做什么来恢复 → 摄取（补充资源）或移动（寻找资源）。

### §3.3 运动势的方向由深层跨链接决定

**关键问题**：系统怎么知道该吃还是该走？

答案：**由影子层中哪个跨链接最先不稳定决定**。

```
情况 A: 
  homeo-intake 跨链接先不稳定
  → Xin^(1) 在 intake 方向上最大
  → ν_intake 积累最快
  → 系统倾向于"摄取"

情况 B:
  homeo-move 跨链接先不稳定
  → Xin^(1) 在 move 方向上最大
  → ν_move 积累最快
  → 系统倾向于"移动"
```

**不是硬编码的决策逻辑——是深层跨链接的不稳定性方向决定了运动势的梯度方向。**

### §3.4 统一方程

$$\nu_\alpha = \int_0^t \sum_{\text{crosslinks } \ell} A^{(1)}_\ell \cdot \Delta_\ell(\tau) \cdot \mathbf{e}_\alpha \, d\tau$$

其中：
- $A^{(1)}_\ell$ = 跨链接 $\ell$ 上的联络系数（纤维丛联络）
- $\Delta_\ell$ = 跨链接 $\ell$ 上的环流偏离基线量
- $\mathbf{e}_\alpha$ = 行为方向的单位向量（摄取/移动/...）
- $\nu_\alpha$ = 行为 $\alpha$ 方向上的运动势

**运动势 = 联络系数 × 偏离量 的时间积分。**

## §5 影子层不是记录器——是压力生成器

### §5.1 重新理解影子的功能

```
旧理解: 影子 = 墓地 / 日志 / 存档
         下葬 → 衰减 → 偶尔共振

新理解: 影子 = 压缩的沙盒 + 运动势源
         收缩度规 → 深层跨链接 → 环流偏离检测
         → 通过纤维联络向主系统施加"压力"
         → 这个压力就是运动势
```

影子层不是被动的——它在**主动压缩时空**，通过压缩发现主系统中被物理距离隐藏的关联，然后将这些关联的不稳定性**反投射**为主系统的行为驱动力。

### §5.2 因果链

```
外部输入 → 主系统各层激活 → Xin 模式
                                ↓
                          影子接收 Xin
                                ↓
                          度规收缩 → 聚团+跨链接
                                ↓
                          影子自身 P^(1)/R^(1) 竞争
                                ↓
                     检测: 哪条跨链接环流偏离基线？
                                ↓
                          Xin^(1) = 纤维联络
                                ↓
                     运动势 ν 在特定方向积累
                                ↓
                     主系统 Motor 层受 ν 调制
                                ↓
                          行为输出（摄取/移动/...）
```

### §5.3 "另一种方式"

你说"可能是另一种方式"——是的。运动势的积累不需要是简单的"不稳定 → 吃/走"的线性关系。它可以是：

**变分方式**：运动势 = 降低 $\mathcal{K}$ 的梯度方向

$$\nu_\alpha = -\frac{\partial \mathcal{K}}{\partial z_\alpha}$$

系统"选择"的行为是使总自由能下降最快的方向。不是"因为饿所以吃"，而是"因为吃能最大幅度降低整个层级塔的预测误差"。

这把三种驱动统一了：

| 驱动 | 为什么降低 $\mathcal{K}$ |
|------|----------------------|
| 摄取 | 补充能量 → 神经元 E↑ → 预测精度↑ → $F^{(0)}$↓ |
| 生体维持 | 恢复基线 → $\Delta_H$↓ → $F^{(1)}$↓ |
| 移动 | 改变输入 → 新信息 → 可能降低 $F^{(0)}$（探索）或升高（风险） |

**移动的特殊性**：它不一定降低 $\mathcal{K}$——它可能升高。但如果当前位置的 $\mathcal{K}$ 已经很高（预测全错），移动至少提供了**降低的可能性**。这就是"探索"的变分解释。
