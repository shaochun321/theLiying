# Morphosphere v39 — 能力、泛化与鲁棒性评估

> 以下评估基于代码审计和实际运行结果，力求诚实，不美化。

## 一、项目目前能做什么

### 核心能力

| 能力 | 具体实现 | 数据来源 |
|------|---------|---------|
| **运动模式识别** | 8 种模式的贝叶斯分类器 | Allen Brain 真实 ΔF/F + 合成数据 |
| **赫布信号变换** | 6 维信号 → W_signal → 7 维 z_t 测度坐标 | 从真实钙信号中学习 |
| **变分 GMM** | ELBO 单调收敛的高斯混合分解 | z_t 特征空间 |
| **最优传输** | Wasserstein 距离度量窗口间细胞分布变化 | 真实细胞坐标 |
| **影子超图** | 死亡结构的 z_t 墓碑存储 + 余弦共振检测 | 管线内部 |
| **拓扑迟滞** | d_σ_t 泄漏积分门控，防止虚假相变 | 管线内部 |
| **外部熵账本** | 4 项 Shannon 熵 + 自动异常检测 | 管线审计面 |

## 二、泛化能力 — 诚实评估

### ✅ 已验证的泛化

| 测试 | 设计 | 结果 | 真实程度 |
|------|------|------|---------|
| **跨窗口** | train/val/holdout 60/20/20% 分割 | 验证集无学习，模式一致 | ★★★★☆ 真实 |
| **跨脑区** | VISp 训练 → VISl (35 cells) + VISpm (128 cells) 测试 | JSD < 0.17, 模式多样性保持 | ★★★☆☆ 见下文 |
| **跨种子** | 10 个独立随机种子 | std = 0.023, 一致 | ★★★★★ 真实 |

### ⚠️ 泛化的诚实局限

**1. 跨脑区泛化衡量的是"分布相似性"，不是"分类准确率"**

跨域测试测的是：在 VISl 上检测到的模式分布与 VISp 的 Jensen-Shannon 散度。这证明了**分类器不会退化为单一模式**，但 JSD < 0.17 不等于"分类正确"——因为我们没有 VISl/VISpm 的**真实标签**。

> 诚实表述：系统能在未见脑区产生**多样且稳定**的分类输出，但我们无法验证这些输出是否"正确"。

**2. 分类准确率 0.849 是在合成标签上测的**

8 种运动模式（stationary, slow_drift, fast_drift, oscillation, jump, diffusion, burst_firing, sustained_activity）由 `BayesianMotionRecognizer` 在**合成的 ground truth** 上评估。系统用 Hebbian 学习从真实信号中推断模式，但"正确答案"是由同一个系统的先验定义的。

> 诚实表述：0.849 证明的是**系统内部一致性**（先验 → 学习 → 推断 是自洽的），不是与外部生物学标注的对比。

**3. 真实数据 → 统一接口的信息压缩**

Allen Brain 的 214 cells × 3003 timepoints ΔF/F 矩阵是**真实数据**。但经过 `AllenBrainAdapter` 映射后：
- `V_mean` ← 窗口内 ΔF/F 均值
- `spike_rate` ← 超过 2σ 的比例
- `x, y` ← 真实 ROI 质心坐标
- `z` ← 固定 0.0

这个映射是合理的（保留了关键统计量），但丢弃了**时序动态**（如瞬态事件、连续 burst 模式）。

## 三、鲁棒性 — 诚实评估

### ✅ 已实现的鲁棒性机制

| 机制 | 防御对象 | 实际效果 |
|------|---------|---------|
| **拓扑迟滞** | 单次 Xin 冲击导致相变 | d_σ_t 积分器 + 阈值 3.0，需持续积累才允许 ΔW |
| **马尔可夫毯铁律** | P 跳过 R 直接冻结 | 硬编码：P_frozen 必须有 R_frozen 前驱 |
| **影子层单向性** | 反馈环路导致振荡 | `shadow_influences_pipeline=False` 硬编码 |
| **W_signal 冻结** | 推断时权重漂移 | 校准后冻结，验证期不更新 |
| **Oja 自正则化** | 赫布权重无限增长 | 权重更新含 -η·y²·w 衰减项 |
| **熵异常检测** | 沉默失败 | z-score > 2.5σ → `entropy_spike` 警报 |

### ⚠️ 鲁棒性的诚实局限

**1. 没有对抗测试**

拓扑迟滞的设计是正确的，但从未被真正的对抗输入测试过。我们不知道：
- 精心构造的 Xin 序列能否绕过积分器？
- 梯度型攻击（缓慢持续的 d_σ_t 注入）是否会累积穿透？

**2. 影子共振的物理意义不完全清晰**

cos = 0.99 的 strong_echo 说明验证窗口的 z_t 与校准窗口的 z_t 非常相似。但由于 W_signal 是冻结的，所有通过同一组权重变换的相似信号 **必然** 会产生相似的 z_t。这意味着高共振可能是**平凡的数学性质**，而非有意义的结构记忆。

> 真正有意义的共振应该是：死亡结构的 z_t 与一个**不同模式**的活跃 z_t 产生回声——这说明不同结构占据了相似的测度空间位置。目前的 2/19 强回声都来自同源信号。

**3. pipeline_engine.py 1900+ 行未分解**

这是最大的工程鲁棒性风险。所有 T/O/P/R/Xin 逻辑在一个文件里，修改任何一个环节都有连锁影响风险。

## 四、能力边界的坦率总结

### 系统是什么

```
一个具有赫布超图架构的时空测度引擎，能够：
  ✓ 从真实钙成像数据中学习信号→运动模式的映射
  ✓ 在冻结权重下对未见数据产生稳定推断
  ✓ 通过影子层保留结构历史
  ✓ 通过迟滞机制抵抗瞬态扰动
  ✓ 通过熵账本提供工程审计能力
```

### 系统不是什么

```
  ✗ 不是经过外部标注验证的神经科学工具
  ✗ 不是端到端的生物信号处理管线（有信息压缩）
  ✗ 不是经过对抗测试的安全系统
  ✗ 不是模块化的（核心仍是单体架构）
```

### 泛化能力打分

| 维度 | 评分 | 理由 |
|------|------|------|
| 跨窗口稳定性 | ★★★★☆ | 60/20/20 分割，验证无学习，表现一致 |
| 跨脑区迁移 | ★★★☆☆ | 分布稳定但无外部标签验证准确率 |
| 跨种子可重复 | ★★★★★ | 10 seeds, std=0.023 |
| 抗噪声能力 | ★★★☆☆ | 迟滞机制存在但未被压力测试 |
| 结构记忆 | ★★☆☆☆ | 影子层功能完整但物理意义需进一步确认 |
