### U 型曲线解释

```
Motor EMA
 0.65 ┤●                                    ●●
      │  ●
 0.60 ┤    ●                 ●●
      │                       
 0.55 ┤
      │         ●                        
 0.50 ┤            ●
      └──┬──┬──┬──┬──┬──┬──┬──
        0.0 .05 .10 .20 .30 .50 .80 1.0  DA
```

**DA < 0.1 (baseline)**:
- gain < 1.0 → da_current NEGATIVE → inhibitory injection into columns
- (da_gain-1)*50 = (0.97-1)*50 = **-1.5** → 抑制 Column
- 但 Column 已被 enc→col 突触驱动饱和 → 抑制效果有限

**DA ≈ 0.1-0.2**:
- gain ≈ 1.0 → da_current ≈ 0 → Column 处于"自然"工作点
- Motor 最低点（0.518）在此区域

**DA > 0.3**:
- gain > 1.0 → da_current POSITIVE → 兴奋 Column
- Column 迅速饱和到 EMA=1.0 → Motor 输出回到最高

## 新发现的 Bug

### 1. xin_to_da initial_weight=5.0 被静默钳位

```
BundleConfig 默认: weight_max = 1.0
initial_weight=5.0 → Memristor w0 = min(1.0, 5.0*(1+var)) = 1.0
实际 mean_weight = 0.918 (而非 5.0)
```

> [!WARNING]
> 需要同时设置 `weight_max=5.0` 或降低 `initial_weight` 到 ≤ 1.0 的合理范围。
> 当前行为：xin→DA 权重被截断为 ~1.0，与 shadow→DA 的 ~0.89 接近，xin 通路未获得预期的 50x 加粗。

### 2. DA neuron da_vta_1 负电压

```
da_vta_1: V_mem = -0.182, activation = 0.000
```

D2R 自受体的 GIRK 电流可能将部分 DA 神经元推入深度抑制。这是合法的物理行为（D2 负反馈），但导致 mean_da 被拉低。

## 下一步建议

**DA 已觉醒但受困于 Column 饱和天花板。** 可选：

1. **降低 DA gain 系数**：50 → 20。减轻饱和，拓宽线性区间到 DA 0.0-0.8。
2. **提高 Column v_peak**：0.15 → 0.25。给 Column 更多 headroom。
3. **接受 U 型并继续**：U 型意味着偏离 baseline（无论高低）都增加 Motor 活动。这在动理学（Klinokinesis）中仍然有效——任何 DA 偏离都引发运动。

> [!IMPORTANT]
> 从生物学看，U 型实际上可能是**正确的**：DA 低于基线=奖赏缺失（负预测误差）→ 探索行为增加；DA 高于基线=新奇刺激 → 探索增加。只有 DA ≈ baseline 时动物"满意"，运动最少。这与 Klinokinesis 完美兼容。
