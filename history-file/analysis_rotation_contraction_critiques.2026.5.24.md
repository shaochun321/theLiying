## §1 运动势 ν 如何对应旋转/周期

### 1.1 当前实现

项目中 ν 有两个不同的计算位置：

**CirculationMeter (circulation.py L188-197):**
```python
ν[key] = (ρ[key](t) - ρ[key](t-1)) / dt_ticks
```
其中 ρ 是环流路径的占比向量：
```
ρ = {p_c, p_b, r_c, r_b, m_b, x, u}
  = {P晶化, P带宽, R晶化, R带宽, 遮蔽, Xin, 未解决}
```

**ShadowSandbox (_update_free_energy, shadow_sandbox.py L433-445):**
```python
K = Σ ξ²        (自由能核 = Xin 总平方)
K_ema += 0.01 × (K - K_ema)   (指数移动平均)
ν = K_ema(t) - K_ema(t-1)     (K 的变化率)
```

### 1.2 与旋转/周期的关系

```
当前 ν 是标量变化率: dρ/dt 或 dK/dt
它只描述 "变了多少"，不描述 "怎么变的"

旋转 = 在某个闭合路径上，状态周期性回到同一区域
周期 = 状态变量的振荡

ν 能否描述旋转？

  ν = dρ/dt 是一维的（每个路径分量各一个 ν）
  旋转需要至少二维: (ρ_i, ρ_j) 在相空间中绕某点
  
  如果 ρ_i 和 ρ_j 同时振荡且相位差 = π/2:
    → 在 (ρ_i, ρ_j) 平面上画圆
    → ν_i 和 ν_j 也振荡
    → 但从单独的 ν_i 看不出旋转，只看到振荡

  需要: 环流的旋度 (curl) 或角动量
    ω = ρ_i × ν_j - ρ_j × ν_i
    = 相空间中的面积扫过率
    = 旋转的测度
```

### 1.3 项目中旋转对应什么？

```
物理旋转:
  yaw, pitch, roll = 三个旋转轴
  前庭的半规管 = 旋转感受器
  振荡器 (VPO) = 产生周期性信号

环流旋转:
  信号在闭合路径上循环:
    Col → Bind → Mot → feedback → Col
  一圈 = 一个周期
  P_frequency = 零交叉次数 / (2 × 窗口长度)

  但这个 "频率" 测量的是流量振荡的频率
  不是信号在环路中实际绕行的周期

实际的环路周期:
  Col → Bind → Mot → feedback → Col
  每一步都传播一次 → 一圈 ≈ 1 step (dt)
  不是振荡 → 是持续流
  
  振荡来自:
    VPO (振荡器): 正弦波调制前庭增益
    STDP: 突触前后不对称 → 权重振荡
    BCM θ: 滑动阈值追踪活跃度 → 延迟振荡
```

## §2 时空测度 ds² 能否描述卷缩

### 2.1 当前 ds² 实现

**shadow_sandbox.py L526-550:**
```python
ds² = -(c_neural × Δt)² + (d_struct/100)²
其中:
  c_neural = 1/τ_RC = 1/15
  d_struct = 1/w (权重越大 → 距离越小)
  Δt = shadow dt = 0.01
```

这是一个 **Minkowski 式** 的度规：时间项为负，空间项为正。
但它是静态的——每对神经元只有一个固定的 ds²。

### 2.2 卷缩需要什么

```
卷缩 (contraction/curling):
  度规从"展开"状态变为"收缩"状态
  某些方向的距离变短 → 该方向的 g_ij 分量减小

在项目中:
  影子层的 "contraction" 正是这个概念:
    w_cross 增大 → d_struct = 1/w 缩短
    = 两个轴在度规空间中"靠近"了
    = 卷缩

  收缩度 κ:
    κ = (Σ w_cross × |a_i| × |a_j|) / (Σ|a_i| × Σ|a_j|)
    κ 增大 → 更多跨轴耦合 → 空间更紧凑
    κ → 1: 完全卷缩（所有方向等价）
    κ → 0: 完全展开（各方向独立）

  但 κ 当前用的是 |a_i| 而非 δa_i
  → 如批判文档指出: 丢失了方向信息
  → 不是真正的度规卷缩, 是幅度加权平均
```

### 2.3 真正的度规卷缩

```
如果用正确的 ds²:
  ds² = Σ g_ij × δa_i × δa_j
  其中 g_ij = ½(w_ij + w_ji) + g_baseline

卷缩 = g_ij 的特征值分布变化:
  特征值均匀 → 各向同性 → 球形 → 展开
  特征值差异大 → 各向异性 → 椭球/扁平 → 卷缩到某些方向

  det(g) = Π(特征值)
  det(g) 大 → 体积大 → 展开
  det(g) 小 → 体积小 → 卷缩

  卷缩方向 = 最小特征值对应的特征向量
  = 被"压缩"的方向
  = 度规在该方向退化

旋转的出现:
  如果 g_ij 的非对角元素有符号交替:
    g_12 > 0 但 g_21 < 0 (在对称化之前)
    → 不对称 → 旋转分量
    对称化后: g̃_12 = ½(g_12 + g_21) ≈ 0
    反对称部分: ω_12 = ½(g_12 - g_21) ≠ 0
    
  ω_ij = 度规的反对称部分 = 挠率 (torsion)
  挠率 ≠ 0 → 空间有旋转 → 卷缩带有螺旋性
```

## §3 两份批判的共同指向

### 3.1 第一份批判 (2026.5.24.txt) 的核心

```
三个致命断层:
  1. 离散 vs 连续诺特
  2. 耗散 vs 保守变分
  3. 分线性 vs 可微性

两个公理漏洞:
  4. g_ij 不对称不正定
  5. |a_i| 不是 δa_i
```

### 3.2 第二份批判 (2026.5.24.1.txt) 的核心

```
三个工程断裂:
  6. P 算了 ξ 但 R 不用
  7. 需要归因机制: 向量化 Xin + R 的感受野
  8. 需要结构重组触发规则: Xin 模式 → 新链/解冻
  + 重申: 度规必须纯化 (g_ij 对称化 + δa 替代 |a|)
```

### 3.3 共同指向

```
两份批判从不同角度到达同一个结论:

  第一份 (数学): 变分要求光滑、对称、正定
  第二份 (工程): P/R 断裂、归因缺失、度规误用

  共同结论:
  ┌─────────────────────────────────────────────┐
  │ 1. 度规 g_ij 必须对称化 + 基线正定化          │
  │ 2. ds² 必须基于偏移 δa, 不是绝对值 |a|       │
  │ 3. Xin 必须接入学习 (P→R 闭合)              │
  │ 4. 需要归因: 区分 "度规错" vs "新事件"         │
  └─────────────────────────────────────────────┘
```

### 3.4 但两份批判的分歧

```
第一份假设: 应该用变分 → 修补光滑性
第二份假设: 应该修工程 → 让 Xin 驱动学习

用户的立场: T/O/P/R/Xin 本身是数学工具
  → 不需要外挂变分
  → 但度规定义的公理漏洞仍然有效
  → P/R 断裂是最急迫的工程问题

  优先级:
    [必须] 度规公理修正 (对称化 + δa)
    [必须] P→R 闭合 (Xin 驱动学习)
    [可选] 变分形式化 (对外解释用, 非运行必需)
```

## §4 综合: ν 与 ds² 的完整关系

```
如果度规正确定义:
  g_ij = ½(w_ij + w_ji) + g_base
  ds² = g_ij δa_i δa_j

那么:
  动能 T = ½ g_ij (δa_i/dt)(δa_j/dt)
  运动势 ν = ∂T/∂(δa_i/dt) = g_ij (δa_j/dt)
    (ν 是 δa 的时间导数在度规空间中的对偶)

  旋转:
    ω_ij = ½(∂ν_i/∂a_j - ∂ν_j/∂a_i)
    = 运动势的旋度
    = 相空间中的角速度

  卷缩:
    ∇·ν = Σ ∂ν_i/∂a_i
    = 运动势的散度
    < 0: 收缩 (卷缩)
    > 0: 膨胀
    = 0: 体积不变

  周期:
    τ = 2π / |ω|
    = 旋转一圈的时间
    = ρ 分量振荡的周期

但在项目当前的离散 + 分线性框架中:
  这些连续算子 (旋度、散度) 需要离散化
  → 差分旋度: ω_ij ≈ (Δν_i/Δa_j - Δν_j/Δa_i) / 2
  → 差分散度: ∇·ν ≈ Σ Δν_i/Δa_i
  
  不需要变分 → 直接在 ρ/ν 的时间序列上计算
```
