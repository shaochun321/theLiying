# 自由能公式诊断 — 数据裁决

## 核心数据

### α 拟合结果 (F = ΣE - T × α)

| 条件 | α 中位 | CV(变异系数) | S_weights | α/S |
|------|--------|-------------|-----------|-----|
| **Baseline** | 0.008 | 535% | 2.98 | 0.003 |
| **Low waste** | 0.014 | 811% | 1.37 | 0.010 |
| **High learning** | -0.023 | 1107% | 2.93 | -0.008 |

α 在三个条件下**不一致**（甚至有负值），且 CV > 500%。
**它不是物理常数，不是 S_weights，不是任何稳定的量。**

### 熵项贡献

```
T × 0.1 / ΣE ≈ 0.005% ~ 0.08%
→ F ≈ ΣE — 熵项是装饰
```

## 根本诊断

```
┌──────────────────────────────────────────────────────┐
│  当前 F = ΣE - T × 0.1 有两个根本错误:               │
│                                                       │
│  1. T ≠ 温度                                          │
│     T = _total_heat / tick = 平均热输出速率             │
│     温度 = (∂S_config/∂E)^{-1}                        │
│     二者是完全不同的物理量                               │
│                                                       │
│  2. 0.1 ≠ 熵                                          │
│     0.1 是人为常数                                     │
│     系统的真实熵 S_config ≈ 2~4 bits                   │
│     两者差 20-40 倍                                    │
│                                                       │
│  结论: 这个公式不是 F = E - TS 的任何形式               │
│  它是 ΣE 加了一个 <0.1% 的噪声项                      │
└──────────────────────────────────────────────────────┘
```

## 怎么办

要建立真正的信息-热力学 bridge，需要**从零推导**:

### Step 1: 定义构型空间

系统的微观态 = 所有权重 + 所有激活的组合:
```
microstate = (w₁₁, w₁₂, ..., w_nm, a₁, a₂, ..., a_k)
```

### Step 2: 定义构型熵

```
S_config = -Σ p(microstate) log p(microstate)
```

实际不可能枚举所有微态，但可以分解:
```
S_config ≈ S_weights + S_activations + S_coupling
```
其中 S_weights 已有 (Shannon entropy of weight histogram)，S_activations 也可测。

### Step 3: 定义温度

从微正则系综:
```
1/T_stat = ∂S_config / ∂E
```

实操: 微扰 E → 观察 ΔS_config → 数值求导。

### Step 4: 自由能自然导出

```
F_true = E - T_stat × S_config
```

> [!IMPORTANT]
> 这不是"修改公式中的常数"。
> 这是重新定义 T 和 S 的计算方法——从模拟的热力学中自举出来，
> 而不是从外部假设灌入。
