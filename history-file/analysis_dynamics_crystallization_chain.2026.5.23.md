| 项目概念 | 中文 | 它是什么 |
|---------|------|---------|

## 它们是什么关系？

### 一条完整的因果链

```
运动层的极限环（钟摆在摆）
        │
        │ 运动神经元持续产生脉冲（约11赫兹）
        ↓
列→运动 的 STDP 权重收敛（碗底的弹珠）
        │
        │ 权重稳定 → 每当列层活跃，运动层都按固定模式响应
        ↓
共激活矩阵累积（运动+积分器同时活跃）
        │
        │ 累积超过阈值（age>50, strength>0.01）
        ↓
结晶条件满足 → 创建新的 px_ 神经元和连接
        │
        │ 新连接改变了信号的循环路径
        ↓
环流度量 μ(G) 增大 → 信号循环更完整
        │
        │ μ 不再变化 → dμ/dt → 0
        ↓
诺特异常 A → 0（系统达到近似守恒）
        │
        │ 这在电路中表现为：
        ↓
全系统 E_trend → 0（能量趋势稳定）← 我们当前观测到的！
```

### 用简单的比喻说

1. **极限环 = 心脏在跳**
   - 运动神经元的周期性放电就像心跳
   - 如果心脏不跳（tau_w=50时的死亡），整个系统没有输出
   - 修复 tau_w 后，心脏恢复了稳定的跳动（11赫兹的极限环）

2. **固定点 = 习惯养成了**
   - STDP 权重收敛 = 系统学会了一种固定的"输入→输出"映射
   - 就像学会骑自行车后，不再需要改变神经连接
   - 权重不再变化 = 碗底的弹珠 = 固定点吸引子

3. **结晶 = 习惯被"刻进"结构**
   - 当共激活足够持久（>50步）且足够强（>0.01），系统创建新神经元
   - 就像反复练习后，大脑中生长出新的突触
   - 结晶是从"临时的统计相关"变成"永久的结构连接"

4. **E_trend → 0 = 一切都稳定了**
   - 心脏在跳（极限环）✓
   - 习惯养成了（固定点）✓
   - 结构已固化（结晶）✓
   - 能量不再变化（守恒）✓

## 层级关系

```
底层（物质基础）
  └─ 运动层极限环（心跳）
      ├─ 如果极限环不存在 → 整条链断裂
      ├─ 如果极限环不稳定 → 结晶不会发生
      └─ FIX-016 修复了这个底层问题

中层（学习过程）
  └─ STDP 权重收敛（固定点吸引子）
      ├─ 依赖底层的极限环提供稳定的脉冲输出
      ├─ 权重收敛速度取决于突触增益和学习率
      └─ 当前 enc_to_col 权重已收敛到 0.100265

上层（结构创建）
  └─ 结晶
      ├─ 依赖中层的权重收敛提供稳定的共激活模式
      ├─ 当前 nexus_v1 的 _is_crystallized() 检查权重方差 < 0.01
      └─ 需要成熟度 ≥ 2（maturation_stage）

顶层（系统守恒）
  └─ E_trend → 0 / 诺特异常 → 0
      ├─ 所有下层都稳定后的必然结果
      └─ 当前全系统已达到
```

## 关键洞察：之前修 Bug 其实是在修"底层"

之前修的 Bug 全都在**底层**：

| Bug | 修了什么 | 影响了哪一层 |
|-----|---------|------------|
| DEG-009 | 运动层能量耗尽 | 底层：极限环不存在（心脏停了） |
| DEG-013 | 绑定层爆炸 | 底层：极限环被外部干扰摧毁 |
| DEG-014 | 适应电流不衰减 | 底层：极限环被内部积累杀死 |

**修完底层后，上面的层才有可能工作。**

这就是为什么之前结晶一直不发生（老项目中 `Origin 不结晶` 的问题）——
底层的极限环就没有稳定过，中层的权重就无法收敛，
上层的共激活就不会累积，结晶条件就永远达不到。

## 影子层在哪里？

影子层是**外部审计者**，观察但不参与以上链条：

```
感觉-运动层（内部系统）          影子层（外部观察）
  极限环 → 权重收敛 → 结晶        ↗ 影子神经元记录激活
                                  ↗ 影子连接记录相关性
                                  ↗ 跨轴收缩 = 检测分岔
                                  ↗ ds² = 检测度量结构
```

影子层的 **跨轴收缩**（ds² 从 80→77→79）就是在检测：
内部系统的吸引子结构是否在变化。

当 ds² 稳定不再变化时 = 影子层"确认"内部系统已经收敛到吸引子。
