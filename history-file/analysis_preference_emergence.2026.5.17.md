# 偏好行为涌现分析：为什么靠近光源、远离声源？

## 结论

**这个偏好不是设计的，是涌现的。** 根因是**随机结构不对称 + 正反馈锁定**。

## 因果链

```
lognormal gain 随机采样  (结构不对称)
         ↓
积分器累积不对称  (不同通道的"记忆强度"不同)
         ↓
integ→motor 正反馈  (放大初始偏差)
         ↓
STDP 锁定  (赫布学习固化偏差)
         ↓
行为偏好  (靠近光/远离声)
```

## 第1层：随机结构不对称

积分器的增益是从 `lognormal(μ=0, σ=0.3)` 中**一次性采样**的：

| 量源 | gain | 含义 |
|---|---|---|
| acoustic | **0.958** | 中等放大 |
| thermal | **1.568** | 高放大 |
| luminous | **0.730** | **低放大** ← 关键 |

luminous 的增益最低 → 同样的 lever_rate 变化，luminous 积分器的累积最弱。

## 第2层：保留率差异

| 量源 | λ_eff | retention | 含义 |
|---|---|---|---|
| acoustic | 0.017 | **96.4%** | 记忆力最强 |
| luminous | 0.027 | **95.5%** | 中等 |
| thermal | 0.037 | **91.5%** | 记忆力最弱 |

acoustic 的保留率最高 → 一旦积累了"远离"的状态，不容易衰减回来。

## 第3层：正反馈锁定

```
babbling 阶段 (tick 0~100):
  随机运动 → acoustic integ 积累到 +0.85 (远离声源)
  随机运动 → luminous integ 积累到 -0.37 (靠近光源)
                              ↑ 初始偏差！
```

初始偏差通过 `integ→motor` 束 (w=0.08) 被放大：

- acoustic integ = +0.85 → 正向motor信号 → 继续远离 → integ继续增加 → **正反馈**
- luminous integ = -0.37 → 负向motor信号 → 继续靠近 → integ继续下降

但 luminous gain=0.73 (低) → 靠近信号的累积速度慢 → 不会过度接近 → 在某个距离**达到平衡**

## 第4层：物理衰减律的次级效应

| 量源 | 衰减律 | 中距离recv | 效应 |
|---|---|---|---|
| acoustic | **1/r** | 0.436 (强) | 远离后仍能感知 → 无需靠近 |
| luminous | **1/r²** | 0.095 (弱) | 必须靠近才能感知 → 靠近有信息增益 |
| thermal | **1/r²** | 0.042 (很弱) | 高gain=1.57补偿弱信号 → 振荡 |

## 第5层：做功方向验证

```
acoustic:  work = -18.3 → -20.7  (持续远离声源做功)
thermal:   work =  -4.9 →  -5.2  (轻微远离)
luminous:  work = +16.0 → +19.7  (持续向光源做功!) ← 偏好行为
```

## 生物学类比

这与真实神经系统中的**偏好形成机制**惊人相似：

1. **随机突触强度不对称** (lognormal gain)
   ≈ 神经发育中随机的突触权重初始化
2. **正反馈锁定** (integ→motor)
   ≈ 赫布学习 "fire together, wire together"
3. **偏好稳定化** (STDP 固化)
   ≈ 经验依赖的突触可塑性固化行为偏好
4. **1/r vs 1/r² 差异**
   ≈ 不同感觉模态的信号衰减特性 (听觉vs视觉)

> **核心洞察**: 没有任何奖励函数、没有任何目标设定。
> 纯粹从 **随机结构 + 物理约束 + 正反馈** 中涌现出了行为偏好。
> 这是一种 "embodied preference" — 身体结构决定了行为倾向。
