# 系统叙事：从运动到结构

## 世界

有一个世界，是一个 100×100×100 的立方体空间。空间是环形的——从一边出去，从另一边进来，没有墙壁，也没有尽头。空间中有一个热源，固定在某个位置。

世界中有一个"身体"。它是一个没有形状的点，有质量（1.0），有位置、速度、加速度。它浸泡在粘稠的介质中（类似蜂蜜），运动越快阻力越大。

身体有三条"肌肉"，分别在 x、y、z 方向上施力。肌肉不能自主收缩——它们等待神经信号。

---

## 第一步：无中生有

系统启动时，什么都没有发生。身体静止在中心 [50, 50, 50]。神经元静默。

但世界对身体施加了两件事：

**第一件事：外部振荡。** 像海浪一样，世界向身体的前庭系统注入一个正弦波信号。yaw 轴以 1 Hz 振荡，pitch 轴以 0.7 Hz 振荡。这不是身体自己的运动，而是**客观实在的外部扰动**——好比一条鱼在水中感受到洋流的涌动。

**第二件事：热梯度。** 身体离热源越近，皮肤感受到的温度越高。这是一个连续的、缓慢变化的标量信号。

这两个信号——振荡和温度——是系统的**全部外部输入**。一切结构都从这两个信号的处理中生长出来。

---

## 信号的旅程：从物理到电流到脉冲

### 第一关：机械转导（Met）

外部振荡到达身体的前庭器官。前庭器官的入口是一个**阈值门**：只有信号强度超过 0.3 时，门才打开，信号才能通过。如果信号太弱，门关着，下游什么都收不到。

这个阈值门就像一个水闸——水位（信号）不够高，闸门不开。这是系统对客观实在做的**第一次切分**：低于 0.3 的扰动被忽略，高于 0.3 的才被承认为"存在"。

**注意**：这意味着系统对时间和空间的感知是有分辨率的。它不感知一切——它感知的是超过阈值的那些。

### 第二关：毛细胞放大（HC）

通过阈值门的信号到达毛细胞。毛细胞有三个通道（像三扇门），分别响应不同的电压范围。信号在这里被**非线性放大**：微弱的振荡变成显著的电流，而且放大的程度取决于信号的"音量"。

同时，毛细胞内部有一个钙离子池。信号触发钙的积累 → 钙达到阈值 → 释放一股神经递质 → 传递给下一层。钙池有容量限制：连续释放后会耗竭，需要时间恢复。

**生物比喻**：毛细胞像一个蓄水池，信号是打开闸门的钥匙，水（钙）流出后需要时间重新蓄满。这个"蓄满"的过程本身就编码了**时间**——系统通过"耗竭-恢复"的节奏来"测量"信号的频率。

### 第三关：传入神经元的分叉（Aff）

毛细胞的输出到达传入神经元。这里发生了一次**关键的分叉**：

- **正则传入（Aff_reg）**：放电间隔均匀。它编码的是**恒定的东西**——重力方向、持续的倾斜。这就是**空间测度的种子**。

- **不正则传入（Aff_irr）**：放电间隔不规则。它编码的是**变化的东西**——加速度的突然改变、振荡的拍频。这就是**时间测度的种子**。

这是系统做的**第二次切分**：它把客观实在的信号分成了"恒定的"和"变化的"两个通道。恒定的变成空间的坐标轴，变化的变成时间的刻度。

**Aff 产生的是脉冲（spike）**——不再是连续信号，而是离散的"嘟嘟嘟"。每一个 spike 就像一声钟响，标记着"此刻有事发生"。spike 的时间间隔（ISI）就是系统内部的**时间单位**。

### 第四关：编码融合（Enc）

正则和不正则两条通路汇合到编码层。编码层有一个 bundle（束），连接着两种传入：

```
T (拓扑): Aff_reg 和 Aff_irr 通过 bundle 连接到 Enc
O (操作): Enc 接收两个 spike 流，将它们叠加为一个综合信号
P (预测): bundle 的权重代表"我预期这两个信号的关系是这样的"
R (实际): 信号实际到达时的模式
Xin = |P - R|: 如果权重预测的关系与实际不符，就产生张力
```

**STDP 在这里工作**：如果 Aff_reg 在 t-1 放电，Enc 在 t 放电（因果顺序），连接增强。如果反过来，连接减弱。

**STDP 编码的就是"谁先于谁"——这是系统对时间因果的内部表示。** 权重矩阵 W 是时间因果的沉积物。

### 第五关：柱状整合（Col）

每个轴（yaw, pitch, roll, oto_x, oto_y, oto_z, therm）有一个独立的编码层。现在，柱状层接收某一个轴的全部编码输出，把它们整合成一个决策信号。

Col 是**运动状态判别的输出端**。它的激活强度代表"这个轴正在发生多大的变化"。所有 Col 的激活加在一起 = **运动势（motion_potential）**——"整个身体当前在动多少"。

同时，从 Col 的信号中可以提取两个测度：

**时间测度** = Enc_irr 的 EMA（不正则通路的平滑放电率）。变化越快的轴，时间测度越高。

**空间测度** = Enc_reg 的 EMA（正则通路的平滑放电率）。在某个方向上有持续倾斜（如重力），空间测度就高。

**运动势** = Σ |Col_activation|。所有轴的变化强度之和。

> 到此为止，系统完成了从"客观实在的物理扰动"到"内部可用的三个测度"的转换。这就是你一直在构建的**运动状态判别结构**。

## 从判别到运动：中间决策层

Col 的输出不应该直接变成运动命令——但目前部分地直接这样做了。

### 直接通路（已存在，有问题）

Col 通过 STDP bundle 直接连接到 Motor。这条通路**只学习了时序因果**——"Col 放电后 Motor 跟着放电"。它不知道运动的效果好不好，只知道放电的先后顺序。

这就像一个人学会了"先挥手再迈腿"的节奏，但不知道"往哪走才对"。节奏是小脑的工作，方向是基底核的工作。目前这两者混在了一起。

### CPG 节律调制（刚转正）

在 Motor 输出到肌肉之前，经过一个节律发生器（CPG）：三个耦合振荡器，像三台咬合的钟摆，保持 120° 的相位差。

效果：x 轴用力时，y 轴在休息，z 轴在恢复。就像游泳时左划-右划-蹬腿的节奏。

CPG 有自己的固有频率（1 Hz），但能被前庭信号**锁频**——身体动得越快，CPG 也越快。这叫**携入**（entrainment）。

### 方向选择（空占位）

应该在这里有一个机制决定"加强 x 方向还是 y 方向"。目前不存在。

### 空间导航（空占位）

应该在这里积分加速度信号来估计"我在哪"。目前不存在。

## 运动的闭环

Motor 放电 → 肌肉收缩 → 身体移动 → 加速度改变。

加速度被放大 500 倍后注入前庭通路（otolith）。这就是**闭环的关键一步**——运动的结果变成了下一步的输入。

```
Motor 放电 → 身体加速 → otolith 感受到加速度 → 信号重新进入 Met → HC → Aff → Enc → Col → Motor'

Motor' ≠ Motor —— 因为 STDP 在每一步都在修改权重
```

**这个闭环就是你说的"场域闭环"**：外部实在（身体的物理运动）和内部实在（权重矩阵）通过这个环路共同演化。

### Xin 在闭环中的角色

每个 bundle 都有一个 ξ（xi）——累积的预测误差。

$$\xi(t+1) = \xi(t) + (P(t) - R(t)) \cdot dt$$

P = 权重 × 前一层的输出（我预测下一步会怎样）

R = 实际到达的信号（实际发生了什么）

**Xin = |ξ|**，是这个差距的绝对值。

当系统运转时，每一步都在累积预测误差。如果预测准确，ξ 接近零。如果预测不准，ξ 持续增长。

**ξ 增长到一定程度（> 0.3），就触发结构生长（sprout）**——系统认为"我的结构不够好，无法预测这个信号"，于是**长出一条新的连接**来更好地编码它。

---

## 结构生长：从信号到形态

### Sprout（发芽）

当某个 bundle 的 |ξ| > 0.3 且距离上次检查已过 10 秒：

系统在那个 bundle 的附近**创建一条新的连接**。新 bundle 的权重从 0 开始，要通过 STDP 自己学习。

这就是你说的"录音"——信号（时间性的）被转录为结构（空间性的）。

### Prune（修剪）

如果一条连接的权重低于 0.001（太弱了），就被删除。

这是自然选择：有用的连接变强存活，无用的连接衰退消亡。

### Mitosis（分裂）

如果一个 Motor 神经元持续过载（放电太频繁 → 疲劳电压持续高），它就**分裂**成母+子。

子继承了母的所有配置和连接。这就是**克隆体**——它和母亲做完全一样的事。

**现在的约束**：每轴最多 20 个 motor（P2），且只有在 motor 确实有效（efference copy 验证 body 在响应）时才允许分裂（P0）。

### Fruit（果实）

每个 bundle 有一个生命周期状态：

```
dormant（休眠）→ growing（生长）→ ripe（成熟）→ decayed（衰退）
```

dormant：权重太低，基本不传信号。新 sprout 出生时就是 dormant。

growing：权重在通过 STDP 增长。信号在流过，结构在"录音"。

ripe：权重稳定了。这条连接已经学会了它要编码的因果关系。

decayed：权重在下降。说明这条连接编码的关系不再出现，正在被遗忘。

**目前大多数 bundle 卡在 dormant**——因为信号太弱，STDP 学到的权重增长不足以触发 growing。

### Maturation（成熟）

每个 Neuron 有一个成熟阶段：

```
Stage 0 (新兵/spine)：高可塑性，快速学习
Stage 1 (成熟/column)：BCM 精调，选择性学习
Stage 2 (老兵/area)：冻结，不再学习
```

转变条件：累积激活量 Φ 超过阈值。活得越久、激活得越多，就越成熟。

**目前大多数神经元停在 Stage 0**——因为 Φ 的积累速度很慢。

## 影子层：沉默的观察者

影子层是主层的一个**结构副本**：它有自己的 24 个神经元、35 条连接、3 层细胞外基质、1 套血管冷却系统。

它接收的不是原始信号，而是**主层每条 bundle 的 Xin**——预测误差的绝对值。

也就是说：**影子层观测的不是"世界发生了什么"，而是"主层对世界的预测有多少错误"。**

影子层在这些 Xin 上做自己的 STDP 学习，计算出三个量：

**κ（收缩度）**：跨轴的相关性 / 自身的方差。如果不同轴的 Xin 彼此相关（同升同降），κ > 1，说明结构在"扩张"——信号模式丰富。如果各轴 Xin 独立甚至反相关，κ < 1，说明结构在"收缩"——信号模式单调。

**ν（运动势）**：自由能 K 的变化率。K = Σ ξ²。如果 K 在增大（Xin 在积累），ν > 0——系统在远离平衡，需要生长来吸收。如果 K 在减小，ν < 0——系统在趋向平衡。

**ds²（时空间隔）**：两个 Col 之间的"距离"。如果它们的 Xin 同方向变化（正相关），ds² > 0（类空——它们在空间上有关联）。如果反方向（反相关），ds² < 0（类时——它们在时间上有因果关系）。

**但这三个量目前都被浪费了——没有任何机制读取它们。** 影子层是一个完成度很高但未接线的子系统。

## 回荡的本质

2M 实验中，Oto Xin 在 19-147 之间永久振荡。这就是"回荡"。

它的产生过程：

1. Motor 放电 → 身体移动 → 加速度产生 → Otolith 感受到
2. Otolith 信号进入前庭链 → STDP 更新权重 → 改变了下一步的预测 P
3. 但 body 一直在动（环形世界不会卡住）→ R 也在变
4. P 和 R 的差 = ξ → ξ 积累 → 触发 sprout
5. 但 sprout 后被 prune → 净增长为零 → ξ 无法被新结构吸收
6. ξ 只能留在权重中 → 权重变化 → 改变 Motor 输出 → body 运动模式变了
7. body 运动变了 → otolith 信号变了 → ξ 又变了 → 循环

**回荡 = Xin 在闭环中无法衰减到零，因为结构容量已满。** 这就是你说的"信息在被限定的结构中反弹"。

回荡的存在证明了**过程的存在**——系统不是静态的，有持续的信息流在其中循环。而这个过程本身就是时间的内部表现形式：权重在变、spike 模式在变、Xin 在振荡——一切都在流动。

| 运动状态判别 | ✅ 完好 | Met→HC→Aff→Enc→Col |
