## 1. 当前系统的热模型审计

### 1.1 现有热耗散机制

```python
# neuron.py L352-354 — 唯一的热源
self.heat_output = clamped_current ** 2 * self._power.r_internal
self.energy = max(0.0, self.energy - self.heat_output)
```

| 机制 | 实现 | 问题 |
|------|------|------|
| 热产生 | I²R (内阻焦耳热) | ✓ 物理正确 |
| 热耗散 | **无** — 热被减掉后就消失了 | ✗ 不现实 |
| 热传导 | **无** — 邻居间无热交换 | ✗ 不现实 |
| 热转移 (血流) | **无** — 没有循环系统 | ✗ 不现实 |
| 基质缓冲 | **无** — 没有胞外空间 | ✗ 不现实 |

**结论: 当前系统的热只是一个数字被减掉，没有任何物理去向。这等价于假设完美散热器（即时散热到无穷大热沉）。**

### 1.2 为什么这是个问题

1. **没有热梯度**: 高活动区和低活动区温度相同
2. **没有热反馈**: 温度不影响离子通道动力学
3. **没有热转移**: 不能模拟 "靠血流带走热" 的机制
4. **能量守恒被破坏**: I²R 产生的热凭空消失

## 2. 真实神经系统的热耗散 — 三层架构

### 2.1 层 1: 胞外基质 (ECM) — 局部缓冲

**胞外基质不只是结构支撑，它是离子缓冲器和信号调制器。**

| ECM 功能 | 机制 | 文献 |
|---------|------|------|
| **离子缓冲** | CSPG 负电荷吸附 Na⁺/K⁺/Ca²⁺ → 维持微环境稳定 | Härtig 1999 |
| **信号增益** | PNN 优化 PV+ 中间神经元的放电精度 | Tewari 2018 |
| **可塑性门控** | PNN 阻止新突触形成 → 关键期后稳定 | Pizzorusso 2002 |
| **扩散屏障** | 限制神经递质扩散范围 → 空间精度 | Syková 2008 |
| **热容** | 含水基质高比热 → 吸收局部温度波动 | 间接 |

**nexus_v1 中缺失的对应物**: 
当前 Neuron 之间的空间是"真空"的 — 没有胞外介质。
r_leak 部分模拟了膜泄漏，但不是胞外缓冲。

### 2.2 层 2: 星形胶质细胞 (Astrocyte) — 代谢中继

**星形胶质细胞是脑能量系统的核心角色。**

```
神经元活动 → 释放谷氨酸 → 星形胶质细胞吸收
    ↓
星形胶质细胞: 有氧糖酵解 → 产生乳酸
    ↓
乳酸穿梭 → 经 MCT 运回神经元 → 氧化磷酸化
    ↓
ATP 再生 (能量恢复)
    
同时:
神经元活动 → Ca²⁺ 信号 → 星形胶质细胞 → 释放血管活性物质
    ↓
血管舒张/收缩 → 局部血流调控 → 热交换
```

| 功能 | nexus_v1 对应 | 状态 |
|------|-------------|------|
| 乳酸穿梭 (能量回收) | VoltageRegulator.recovery | **极度简化** — 固定速率 |
| 谷氨酸清除 | 无 | **缺失** |
| 血管调控 | 无 | **缺失** |
| K⁺ 缓冲 | r_leak 泄漏 | **间接** |

### 2.3 层 3: 脑血管系统 (CBF) — 热转移

**真正的热耗散主力不是传导，是对流 — 血流带走热。**

```
大脑代谢功率: ~20W (占体重 2%, 耗能 20%)
每分钟血流量: ~750 mL
水的比热: 4.18 J/(g·K)
温度梯度: 动脉 36.6°C → 脑组织 37.2°C → 静脉 37.0°C

热转移率 ≈ 750 × 4.18 × 0.4 / 60 ≈ 20.9 W ← 匹配代谢产热
```

**关键洞察**: 高活动区域（如运动皮层在运动时）有更强的血流 → 更强的散热。
这就是用户问的 "某些区域确实热耗散更强，靠的是更强的热源转移方式" — **答案是对的，靠的就是局部血流增加（神经血管耦合）**。

## 3. nexus_v1 的实现差距与提案

### 3.1 差距清单

| # | 缺失机制 | 生物实现 | 后果 |
|---|---------|---------|------|
| H1 | **热传导** | 组织/ECM 热扩散 | 无法模拟热梯度 |
| H2 | **热对流** | 脑血流带走热 | 高活动区无法散热 |
| H3 | **温度反馈** | T 影响 τ_gate, gm | 缺少温度-性能耦合 |
| H4 | **基质缓冲** | PNN/ECM 离子缓冲 | 无微环境稳定 |
| H5 | **代谢中继** | 乳酸穿梭 | VR 过于简化 |
| H6 | **活动-血流耦合** | 神经血管耦合 | 活跃区无优先供能 |

### 3.2 提案: 两个新元件

#### A. ExtracellularMatrix (ECM) — 胞外基质

```python
class ExtracellularMatrix:
    """胞外基质: 离子缓冲 + 热容 + 扩散介质。
    
    功能:
    1. 离子缓冲: 吸收神经元释放的 K⁺ → 维持 V_rest
    2. 热容: 吸收局部热量 → 平滑温度波动
    3. 扩散: 限制信号扩散范围 → 空间精度
    4. 可塑性门控: PNN 控制突触稳定性
    
    EE: 散热器 (heat sink) + 去耦电容 (bypass cap)
    BIO: PNN/CSPG/hyaluronan matrix
    """
    thermal_capacity: float     # J/K — 热容
    thermal_conductance: float  # W/K — 与邻居的热传导系数
    ion_buffer_capacity: float  # 离子缓冲量
    temperature: float          # 当前温度
```

#### B. VascularCooling (血管冷却)

```python
class VascularCooling:
    """脑血管散热系统。
    
    功能:
    1. 基础血流: 持续带走代谢产热
    2. 活动耦合: 高活动 → 血流增加 → 散热增加
    3. 能量递送: 血流也带来 O₂/葡萄糖 → 恢复 ATP
    
    EE: 液冷系统 (liquid cooling loop)
    BIO: 神经血管耦合 (neurovascular coupling)
    
    数学模型:
      Q_remove = flow_rate × c_blood × (T_tissue - T_artery)
      flow_rate = base_flow × (1 + NVC_gain × local_activity)
      energy_delivery = flow_rate × O2_concentration
    """
    base_flow: float            # 基础血流
    nvc_gain: float             # 神经血管耦合增益
    c_blood: float              # 血液比热
    t_artery: float             # 动脉温度 (参考)
```

## 4. 对 nexus_v1 项目的影响

### 4.1 热耗散影响分析

当前系统中 heat_output 对行为的唯一影响是减少 energy。
如果 energy → 0，VR 无法恢复，但实际上 VR 总能追上（base_rate=0.1 >> heat）。

**所以当前的热模型是"装饰性"的 — 它不真正影响动力学。**

如果我们加入真正的温度场：
- 温度 ↑ → τ_gate ↓ (通道动力学加速) → 放电频率 ↑
- 温度 ↑↑ → 蛋白质变性 → 功能失效
- 局部过热 → 血流增加 → 降温 (负反馈)

### 4.2 基质对 CV 的可能影响

PNN/ECM 离子缓冲的效果等价于:
- 稳定 V_rest 波动 → 减少阈值交叉的随机性 → CV ↓
- 增加膜时间常数 (通过增加有效电容) → 平滑 ISI → CV ↓

**这可能是解决 CV 的另一条路径 — 不是靠振荡器，而是靠增加膜的有效电容。**
